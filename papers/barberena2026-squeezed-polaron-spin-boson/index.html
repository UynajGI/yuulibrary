    自旋玻色子模型的“压缩极化子”变分波函数#

    Diego Barberena ∗ 和 Nigel Cooper

    T.C.M. Group, Cavendish Laboratory, University of Cambridge, J.J. Thomson Avenue, Cambridge CB3 0US, UK (日期：2026年7月9日)

    亚欧姆自旋玻色子模型的局域化相变会产生玻色子浴关联，这些关联在低频下对于标准相干态极化子变分尝试函数而言是解析不可达的。本文我们引入了一个“压缩极化子”波函数，该函数包含了由自旋杂质诱导的通用高斯型玻色子-玻色子关联。这给出了浴观测量在局域化相变附近正确的临界幂律标度，同时系统地改进了基态能量，并更精确地确定了临界耦合强度。我们的波函数在深亚欧姆区域捕捉到了相变正确的平均场性质。在浅亚欧姆区域，它还展示了非平均场临界指数。以压缩极化子为起点，我们推导了玻色子的散射相移，并展示了它们如何编码在局域化相变处消失的涌现能量尺度。

    引言：自旋玻色子模型[1, 2]是量子耗散的标志性模型，描述了单个两能级系统与玻色子模式环境浴之间的相互作用。因此，它为研究广泛参数范围内（包括以浴诱导记忆效应[3, 4]和强系统-环境耦合[5, 6]为特征的区域）的退相干和弛豫等现象提供了一个最小设置。该模型也具有实验相关性，既是例如多种系统中耗散量子隧穿[7–10]的有效描述，也是量子模拟工作的目标[11, 12]。

    尽管其组成简单，自旋玻色子模型却展现出丰富的现象学。在其基态中，它可以表现出二级量子相变[13, 14]。通过增加两能级系统与其环境之间的耦合（由谱函数 $J ( \omega ) \propto \alpha \omega^{s}$ 参数化）可以到达这个相变。除了整体相互作用强度 $\alpha$ 外，$J ( \omega )$ 还依赖于指数 s，它控制最低频模式与两能级系统相互作用的强弱。当 $s \leq 1$（亚欧姆区域）且 α 较小时，基态是唯一的，两能级系统磁化强度为零，产生“退局域化”相。在较大的 α 下，系统进入一个具有两个简并基态的“局域化”相，其中两能级系统具有相反的磁化强度，低频玻色子具有无限占据。由于模型简单，该相变已使用多种数值技术得到了广泛研究[15–19]。

    该模型也使用解析方法进行了研究[17, 20]。特别是，变分原理在捕捉自旋玻色子基态的关键物理方面非常成功。极化子尝试函数[21–23]，其中两能级系统的自旋向上和向下态各自被有限数量的叠加玻色子相干态（即“极化子”）所修饰，有效地包含了自旋-浴关联。此外，它们还作为旨在捕捉基态其他预期特征的改进的起点[24–27]，是更详尽数值方法的基础[28]，并且对于研究散射[29–31]、直接驱动[32]、量子态制备[33]等动力学过程也很有用。同时，有限数量的相干态极化子无法[18]准确捕捉在局域化相变附近最低频玻色子之间发展起来的玻色子-玻色子关联。

    在本文中，我们用一种允许通用高斯玻色子-玻色子关联的ansatz替代了相干态极化子ansatz。此前，这类关联仅在欧姆极限（s = 1）下通过数值方法被考虑过[34]。我们找到了两个解分支，分别对应局域化相和退局域化相。我们解析地证明了退局域化分支在玻色子可观测量上展现出正确的低能标度律[18]。此外，由我们的波函数推导出的临界耦合值接近于通过数值精确方法获得的值，并且在$s \lesssim 0.4$时，与自旋周围玻色子云的激发变为无带隙时的耦合值一致。该模型还为物理可观测量提供了更好的精度，特别是在浅亚欧姆区间$(0.5 < s < 1)$。有趣的是，在退局域化分支中，当s从深亚欧姆区域$(0 < s < 0.5)$移动到浅亚欧姆区域时，我们的“压缩”极化子也反映了从平均场行为到非平均场行为的变化，尽管我们在后者中发现的临界指数数值上并不精确，并且它们出现在局域化分支能量更低的参数区域。在论文结尾，我们预测了玻色子在压缩极化子背景下的散射特性。通过在宽频率范围内分析这些特征，我们展示了如何利用环境探针来提取关于系统的更深层信息。

    浴对TLS的影响由谱函数$\begin{array} {r} {J ( \omega ) = \pi \sum_{\beta} g_{\beta}^{2} \delta ( \omega - \omega_{\beta} )} \end{array}$决定[1]。对于我们的亚欧姆浴，$J ( \omega ) = 2 \pi \alpha \omega_{c}^{1 - s} \omega^{s} \theta ( \omega_{c} - \omega )$由截止频率$\omega_{c} \gg \Delta$、耦合强度α和浴指数s指定。当$s \leq 1$时，系统在临界α值处经历从低α、具有$\langle \hat{s}_{z} \rangle = 0$的退局域化相，到更高α、具有$\langle \hat{s}_{z} \rangle \neq 0$的局域化相的局域化转变。当$0 < s \le 0.5$时，该转变属于平均场普适类，但$0.5 < s < 1$时则不是平均场行为。

    对于方程(1)的一大类变分波函数，可以通过将基态|ψ⟩表示为$\left| \psi \right. = {\hat{U}} \left| \phi \right.$来写出，其中|ϕ⟩是一个简单态，并且

    $$ \hat{U} = \hat{\mathcal{D}}_{c} \hat{\mathcal{D}}_{u} \hat{R} ( \theta )\tag{2} $$

    是一个由三个操作组成的幺正算符。第一个操作 $\hat{R} ( \theta ) = \exp ( - i \theta \hat{s}_{y} )$ 绕 y 方向旋转自旋。第二个操作 $\begin{array} {r} {\hat{\mathcal{D}}_{u} = \exp \left[ - i \sqrt{2} \sum_{\beta} \alpha_{\beta} \hat{p}_{\beta} \right]} \end{array}$ 在相空间中沿 $x_{\beta}$ 方向平移热库模式；最后一个操作 $\begin{array} {r} {\hat{\mathcal{D}}_{c} = \exp \left[ i \sqrt{2} \hat{s}_{z} \sum_{\beta} q_{\beta} \hat{p}_{\beta} \right]} \end{array}$ 也沿 $x_{\beta}$ 平移热库模式，但此时平移方向与自旋态的 z 投影相关[见图1(b)]。从物理上讲，$\hat{\mathcal{D}}_{c}$ 通过为自旋上、下分量配备不同的玻色子云来引入自旋-热库纠缠和关联。

    在由 $\hat{U}_{;}$ 定义的框架内，著名的 Silbey-Harris [21] 波函数通过设置 $\alpha_{\beta} = \theta = 0$，选取 $\vert \dot{\phi}_{0} \rangle = \vert - x \rangle \otimes \vert 0 \rangle$（即沿 −x 方向极化的自旋与玻色子真空的乘积），并对 $q_{\beta}$ 进行极小化得到。Chin [22] 发展的推广使用相同的 $\left| \phi_{0} \right.$，但同时对 $\alpha_{\beta} , \theta , q_{\beta}$ 进行极小化，以捕捉系统的两个相。

    压缩极化子：我们的改进在于允许 |ϕ⟩ 为玻色子热库的通用高斯态，这意味着它完全由 $\hat{x}_{\beta} , \hat{p}_{\beta}$ 的两点关联函数通过维克定理表征。极小化过程将选出由于 TLS 存在而引起的最优热库-热库关联。利用 |ϕ⟩ 的高斯性，我们得到以下基态能量

    $$ \begin{array} {l} {E = - \displaystyle \frac{\cos \theta} {2} \overbrace{\Delta{e}^{- \sum_{\beta \beta^{\prime}} q_{\beta} q_{\beta^{\prime}} \langle \hat{p}_{\beta} \hat{p}_{\beta^{\prime}} \rangle} {}_{\beta}}^{D} + \sum_{\beta} \omega_{\beta} \langle \hat{a}_{\beta}^{\dagger} \hat{a}_{\beta} \rangle_{\phi}} \\ {+ \sum_{\beta} \omega_{\beta} \alpha_{\beta}^{2} + \sum_{\beta} ( g_{\beta} - \omega_{\beta} q_{\beta} ) \alpha_{\beta} \sin \theta} \\ {+ \sum_{\beta} \frac{\big ( g_{\beta} - \omega_{\beta} q_{\beta} \big )^{2}} {4 \omega_{\beta}} - \sum_{\beta} \frac{g_{\beta}^{2}} {4 \omega_{\beta}} ,} \end{array}\tag{3} $$

    其中平均 $\langle \rangle_{\phi}$ 是相对于高斯态 $| \phi \rangle$ 进行的，并且我们引入了如上括号所示的重整化相干性 D。另请注意，E 仅依赖于 |ϕ⟩ 的两点关联函数。一般的极小化过程详见补充材料 [35]。最终结果是，在固定角度 $\theta$ 下，位移变为缩放模式频率 $y_{\beta} = \omega_{\beta} \cos \theta / D$ 的函数，并由下式给出

    $$ q_{\beta} = \frac{g_{\beta} \cos \theta / D} {y_{\beta} + m ( y_{\beta} )} , \qquad \quad \alpha_{\beta} = - \frac{q_{\beta} m ( y_{\beta} ) \sin \theta} {2 y_{\beta}} ,\tag{4} $$

    其中 $m ( y )$ 是满足积分方程的函数

    $$ m ( y ) = \int_{0}^{\infty} \frac{2 y \mathrm{d} z / \pi} {y^{2} + z^{2}} \frac{1} {1 + 2 b \int_{0}^{y_{c}} \frac{x^{s + 1} \mathrm{d} x} {\left( x^{2} + z^{2} \right) \left[ x + m ( x ) \right]^{2}}} ,\tag{5} $$

    其中 $y_{c} ~ = ~ \omega_{c} \cos \theta / D$。方程(5)中的参数 b 是缩放耦合常数 $\alpha_{\Delta} = \alpha ( \omega_{c} / \Delta )^{1 - s}$ 的函数，通过以下关系隐式定义

    $$ \alpha_{\Delta} = \frac{b} {( \cos \theta )^{3 - s}} \exp \left( - \frac{b ( 1 - s )} {( \cos \theta )^{2}} \int_{0}^{y_{c}} \frac{m ( y ) y^{s} \mathrm{d} y} {\left[ y + m ( y ) \right]^{2}} \right) ,\tag{6} $$

    退定域支：在这个分支中 $\theta = \alpha_{\beta} = 0$ 且 $y_{\beta} = \omega_{\beta} / D$。此外，参数 b 在数值上等于重整化耦合常数 $\alpha_{\Delta} ( \Delta / D )^{1 - s}$。

    我们波函数的所有物理性质都源于 $m ( y )$ 的结构，该结构由方程(5)决定。特别地，$m ( y )$ 在低频下的行为主导了方程(4)中关联位移 $q_{\beta}$ 的低频特性，因此与定域化转变相关。对于固定的 $s = 0.8$ 和不同的 b 值，$m ( y )$ 作为 $y = \omega / D$ 函数的曲线图如图3(b)所示。在所有情况下，当 $y \gg 1$ 时，我们有 $m ( y ) \approx 1$。在相反的 $y \ll 1$ 区域，$m ( y )$ 趋于常数 $m_{0}$。因此，在小 $\omega$ 下的位移 $q_{\beta}$ 是有限的，并由 $g_{\beta} / ( D m_{0} )$ 给出。

    $$ \langle{\hat{s}}_{z} ( {\hat{a}}_{\beta} + {\hat{a}}_{\beta}^{\dagger} ) \rangle = {\frac{q_{\beta}} {2}} = {\frac{g_{\beta}} {2 \pi \alpha \omega_{c}^{1 - s} \cot ( \pi s / 2 ) \omega_{\beta}^{s}}} $$$$ \langle ( \hat{a}_{\beta} + \hat{a}_{\beta}^{\dagger} )^{2} \rangle = 1 + \frac{g_{\beta}^{2} ( 1 + s )} {2 \pi \alpha \omega_{c}^{1 - s} \cot ( \pi s / 2 ) \omega_{\beta}^{1 + s}}\tag{9} $$

    这些结果正确捕捉了这些可观测量的频率和 s 依赖性，早期通过基于图解化的解析论证[17, 18]已得到。此外，这种标度律通常与涉及有限数量相干态极化子[18]的变分波函数不相容。

    当 $m_{0} \to 0$ 时，激发自旋周围玻色子云所需的能量趋近于 0。这个涌现的尺度 $\omega_{*}$ 在参数上远小于 D，并控制着系统的低频性质。这反映在函数 $m ( y )$ 上，对于频率 $\omega \ll D$，它具有标度形式[35]

    $$ m ( y ) \approx m_{0} r \left( \omega / \omega_{*} \right) ,\tag{10} $$

    其中 $r ( 0 ) = 1$，$\omega_{*} = D ( m_{0} / b )^{1 / s} \ll D$。当 $\omega \ll D$ 时，这一标度函数决定了系统的物理性质，并在“临界”区域 $\omega \gg \omega_{*}$ 和“非临界”区域 $\omega \ll \omega_{*}$ 之间插值。

    我们认为，图2中定域分支与离域分支之间的虚假一级跳变是由对临界点附近涨落的处理不当造成的。定域相中真实的波函数更准确地应取为两个定域解的猫态叠加[25, 43]。因此，在离域相中但靠近临界点时，也应该存在类似猫态的涨落，但猫态波瓣之间的相空间分离在挤压极化子的量子噪声中无法完全分辨。

    总结与结论：在本文中，我们分析了在自旋-玻色子模型的标准极化子变分波函数中添加高斯涨落所带来的后果。通过包含玻色子-玻色子关联，我们得到了自旋-玻色子基态大部分性质正确的定性（在某些情况下是定量）描述。这些性质包括：当趋近于定域化相变时，玻色子扇区中临界涨落的出现及其本质；随着浴指数s的增加，从平均场行为到非平均场行为的转变；以及自旋杂质散射玻色子的效应。

    这里我们简要概述积分方程的数值求解方法

    $$ m ( y ) = \int_{0}^{\infty} \frac{dz} {\pi} \frac{y} {y^{2} + z^{2}} \frac{1} {1 + 2 b \int_{0}^{y_{c}} \frac{x^{s + 1} d x} {\left( x^{2} + z^{2} \right) \left[ x + m ( x ) \right]^{2}}} .\tag{S1} $$

    我们首先在固定的 $y_{c} = \omega_{c}$ cos $\theta / D = \omega_{c} / \Delta \times ( \Delta \cos \theta / D )$ 下求解该方程，然后系统地改进其值。现在我们按照以下关系定义函数序列 $m_{n} ( x )$

    $$ m_{n + 1} ( y ) = \int_{0}^{\infty} \frac{2 d z} {\pi} \frac{y} {y^{2} + z^{2}} \frac{1} {1 + 2 b \int_{0}^{\infty} \frac{x^{s + 1} d x} {( x^{2} + z^{2} ) \left[ x + m_{n} ( x ) \right]^{2}}} .\tag{S2} $$

    如果我们选择 $m_{0}^{I} ( y ) = 1$，那么 $n_{1}^{I} ( y ) < m_{0}^{I} ( y )$。此外，我们还有：如果 $m_{n + 1} ( y ) \leq m_{n} ( y )$，则逐点地有 $m_{n + 2} ( y ) \leq m_{n + 1} ( y )$。因此，在我们选择 $m_{0}^{I} ( y )$ 的情况下，序列中每一次迭代都是逐点递减的。相反，如果选择 $\dot{m}_{0}^{II} ( y ) = 0$，则 $m_{1}^{II} ( y ) > m_{0}^{II} ( \bar{y} )$，并且如果 $m_{n + 1} ( y ) \geq m_{n} ( y )$，则 $m_{n + 2} ( y ) \geq m_{n + 1} ( y )$。因此序列 $m_{n}^{II} ( y )$ 是逐点递增的。数值计算表明，两个序列都收敛到同一个函数。在远离临界点的地方，通常需要 3 到 10 次迭代即可收敛，迭代之间的逐点相对误差为 $10^{- 6}$。在更接近临界点时，迭代次数会增多。

    核心论证链#

    1. 构造更一般的变分流形 — 将波函数写为 $|\psi\rangle=\mathcal{D}_c\mathcal{D}_u\hat{R}_y(\theta)|{-x}\rangle|\phi\rangle$，其中 $\mathcal{D}_c$ 为含自旋‑玻色子纠缠的关联位移，$\mathcal{D}_u$ 为非关联位移，$\hat{R}_y(\theta)$ 为自旋旋转，$|\phi\rangle$ 为零均值高斯态。为什么这么做：传统相干态极化子只包含位移自由度（线性变换），无法描述浴模的压缩（squeezing），而这种压缩在高频区对基态能量有显著贡献。结论：变分流形从 $2N$ 维（$N$ 个位移参数）扩展为 $N+N(N+1)/2$ 维（位移 + 高斯协方差），更充分地捕捉了浴模的量子涨落。

    2. 用关联位移化简哈密顿量 — 施加 $\mathcal{D}_c^\dagger\hat{H}\mathcal{D}_c$ 变换将关联位移吸收进哈密顿量的显式形式（式 S1），得到含参数 $q_\beta$（关联位移强度）的重整化哈密顿量。为什么这么做：变换后的哈密顿量中自旋与玻色子的纠缠被显式解耦，使后续可在无自发纠缠的参考系中对 $|\phi\rangle$ 做高斯变分。结论：变换后的哈密顿量中自旋‑玻色子耦合项变为 $\propto (g_\beta-\omega_\beta q_\beta)\hat{s}_z(\hat{a}_\beta+\hat{a}_\beta^\dagger)$，可通过选择 $q_\beta$ 消除至一阶。

    3. 对高斯态 $|\phi\rangle$ 做变分 — 在约束 $\langle\phi|\phi\rangle=1$ 下对 $|\phi\rangle$ 变分，得到关于 $\hat{X}$、$\hat{P}$ 的高斯本征值方程（式 S4），并利用预解式技巧写出关联函数 $\langle\hat{P}\hat{P}^T\rangle$ 的紧凑形式（式 S8–S9）。为什么这么做：高斯变分等价于在 Wigner 函数为高斯形的流形中寻找能量极值，$\langle\hat{P}\hat{P}^T\rangle$ 编码了压缩的信息。结论：$\langle\hat{P}\hat{P}^T\rangle$ 可写为对角矩阵加秩‑1 扰动的预解式，这为后续推导 $m(y)$ 的积分方程铺平了道路。

    4. 推导 $q_\beta$ 和 $m(y)$ 的自洽方程 — 分别对 $q_\beta$ 和 $\alpha_\beta$（非关联位移参数）做变分得到式 (S5)，消去 $\alpha_\beta$ 后得到 $q_\beta=\frac{g_\beta\cos\theta}{\omega_\beta\cos\theta+D\,m(\omega_\beta\cos\theta/D)}$（式 4），其中 $m(y)$ 满足非线性积分方程（式 5）。为什么这么做：$m(y)$ 是压缩效应的核心量，它的自洽方程封闭了整个变分问题。结论：$m(y)$ 在 $y$ 小的区域由 $b$ 和 $s$ 唯一确定，其低频行为直接决定是否发生局域化相变。

    5. 对自旋旋转角 $\theta$ 做变分并建立相变判据 — 对 $\theta$ 变分得到 $(\cos\theta)^2=2b\int_0^{y_c}y^{s-1}m(y)^2/[y+m(y)]^2\,dy$（式 6），并证明 $\sin^2\theta=m(0)=m_0$（附录 J）。为什么这么做：$\theta$ 控制着自旋从 $x$ 方向偏转的程度，$\theta\neq0$ 标志着局域化的出现。结论：$m_0\to0^+$ 对应于连续相变的临界点，$m_0>0$ 对应局域相。

    6. 推导重正化隧穿分裂 $D$ 的自洽方程 — $D=\Delta\exp\{-b/(\cos\theta)^2\int_0^{y_c}m(v)v^s/[v+m(v)]^2dv\}$（式 7）。为什么这么做：浴模的零涨落通过指数因子压制裸隧穿，$D$ 是唯一携带能量量纲的标度。结论：$D$、$m(y)$、$\theta$ 三者通过式 (5)‑(7) 构成一个完整的封闭方程组，任何物理量都可通过这组方程计算。

    7. 解析抽取临界指数 — 在 $y\ll1$ 下假设 $m(y)\sim B y^p$ 并代入积分方程，得到 $p=s$ 和 $B=\pi b\cot(\pi s/2)$（附录 E）；在 $b\to b_c$ 附近引入标度函数 $r(u)=m(y)/m_0$，通过微扰展开得到 $m_0\sim|b-b_c|^{\max(1,\,s/(1-s))}$ 和 $\nu=1/(1-s)$（附录 G）。为什么这么做：临界指数刻画了相变的普适类，数值上可直接验证。结论：指数在 $s=0.8$ 时数值结果为 $3.986\pm0.001$，与理论值 $s/(1-s)=4$ 吻合。

    8. 计算散射相移并与 CFT 比较 — 在离域相构造单激发格林函数，得到 $S$ 矩阵仅含一个能量依赖的相移（附录 H），在低能临界区 $\omega_*\ll\omega\ll D$ 内可化简为 $e^{i\pi s}$。为什么这么做：散射实验是探测量子临界动力学的直接手段。结论：该相移与边界 CFT 分析的结果（利用 $\hat{s}_z$ 算符的标度维度 $\Delta=(1-s)/2$ 固定）一致，验证了压缩极化子波函数在低频动力学上的正确性。

    批判性思考#

    1. 高斯约束的上限能力 — 压缩极化子将变分流形限制在零均值高斯态，这意味着所有奇数阶关联函数为零、偶数阶关联完全由协方差矩阵决定。对于亚欧姆浴，当 $s<0.5$ 时红外奇异性增强，浴模之间可能产生三阶以上的非高斯关联（例如 $\langle\hat{x}_\beta\hat{x}_\gamma\hat{x}_\delta\rangle\neq0$），高斯流形无法捕捉这些结构。相比之下，变分多极化子（VPA）虽计算更昂贵，但可描述非高斯的多极化子纠缠。本文未给出任何定量判据（如累积量展开的截断误差）来说明高斯近似在哪个 $s$ 区间失效。

    2. 积分方程迭代收敛性的数学风险 — 附录 C 采用逐次迭代 $m_{n+1}=F[m_n]$ 求解式 (5)，收敛性仅依赖于从上下两个初始猜想的单调逼近（$m_0^I=1$ 和 $m_0^{II}=0$ 分别给出递减和递增序列），但未证明映射 $F$ 是压缩映射（Lipschitz 常数 $<1$）。在临界点附近迭代次数增多，且文中未给出收敛速率的标度关系（如 $|m_{n+1}-m_n|\propto \lambda^n$ 中的 $\lambda$ 是否随 $b\to b_c$ 趋近于 1）。若存在多个不动点，单调逼近只能得到其中一个，无法保证是全局最小值。

    3. 临界指数推导中的微扰展开截断 — 附录 G 将 $r(u)$ 在大 $u$ 区写为 $r(u)=Bu^s(1+C/u^t)$ 并代入积分方程，仅保留到 $O(1/u^t)$ 一阶。但 $r(u)$ 的积分方程（式 S1）在被积函数中同时出现 $r$ 的平方和 $1/r^2$，高阶修正可能以 $O(1/u^{2t})$、$O(1/u^{s+t})$ 等形式耦合进来，文中未证明这些项的贡献可忽略。特别当 $s$ 接近 0 或 1 时，$t=\min(s,1-s)$ 很小，高阶项与一阶项可能同量级。

    4. 有限 $\omega_c/\Delta$ 与标度极限之间的映射误差 — 附录 D 将标度极限（$\omega_c/\Delta\to\infty$）下求解的 $\alpha_\Delta^c$ 通过因子 $(0.1)^{1-s}$ 转化为 $\omega_c/\Delta=10$ 下的 $\alpha_c$ 以与 QMC 等数值方法对比。但这种映射假定有限 $\omega_c$ 的修正仅是一个乘性标度因子，忽略了 $y_c=\omega_c\cos\theta/D$ 的自洽性——$D$ 本身依赖于 $y_c$，而 $y_c$ 又是 $\omega_c/\Delta$ 的函数。这种非线性反馈可能产生不可忽略的系统偏差，这或许是 $s=0.8$ 时压缩极化子预测值（0.3313）与 QMC 值（0.360）偏离 8% 的来源。

    5. $\sin^2\theta=m_0$ 的等价关系仅在 $y_c\to\infty$ 时严格成立 — 附录 J 的证明利用了 $y_c\to\infty$ 将积分化为闭合形式。对于有限 $y_c$（数值计算中的实际情况），$J(0)$ 的积分上限截断为 $y_c$，$m_0$ 的定义式也使用相同 $y_c$，理论上 $\sin^2\theta=m_0$ 应近似成立，但推导中某些积分仅在 $y_c\to\infty$ 时与截断无关。对于较小的 $s$（如 $s=0.1$），低频行为主导，有限 $y_c$ 的截断效应可能使 $\sin^2\theta$ 与 $m_0$ 产生数值偏差，文中未对此进行定量估计。

    6. 散射计算中单激发截断的合理性 — 附录 H 中投影到单激发扇区，假定 $\hat{p}_k\hat{p}_{k'}$ 项的符号翻转不产生多粒子激发。然而，自旋翻转后玻色子模式的有效哈密顿量改变，对变分真空 $|{-x}\rangle|\phi\rangle$ 作用可能会产生双粒子甚至多粒子成分。文中仅通过"投影到粒子数守恒扇区"规避此问题，未提供定量论证（例如计算双粒子产生振幅与单粒子振幅的比值作为能量函数）。对于亚欧姆浴，低能时多玻色子过程可能被红外奇异性增强。

    7. 与变分多极化子（VPA）的复杂度对比缺失 — 附录 D 仅列出两种方法的临界耦合数值，未对比计算复杂度。VPA 需要优化多个极化子的位置和权重，自由度随极化子数 $N$ 线性增长（典型 $N=10$–$50$）；压缩极化子的自由度由一个连续函数 $m(y)$ 描述，离散化为 $10^3$ 个网格点。但 VPA 可以描述非高斯结构，压缩极化子则严格限制在高斯流形内。对于 $s$ 接近 1 的欧姆极限，两者预测的 $\alpha_c$ 差异如何？文中未讨论。

    局限性#

    • 高斯约束排除非高斯关联：变分波函数中 $|\phi\rangle$ 为高斯态，所有奇次关联为零、偶次关联完全由协方差矩阵决定。对于 $s<0.5$ 的强亚欧姆浴，浴模之间可能产生不可忽略的三阶及更高阶累积量，高斯流形无法捕捉这些非高斯结构，且本文未给出任何定量判据（如高阶累积量与二阶的比值）来界定高斯近似的有效区间。

    • 有限 $\omega_c/\Delta$ 的外推缺少系统扫描：临界耦合的数值对比（附录 D）全部在 $\omega_c/\Delta=10$ 下进行，但压缩极化子的自洽方程组求解实际在标度极限（$\omega_c/\Delta\to\infty$）下完成后再通过乘性因子外推。文中未系统扫描 $\omega_c/\Delta=5,\,20,\,50$ 来验证外推的收敛性，也未给出有限 $\omega_c$ 修正的不确定度带。

    • 临界点附近的迭代收敛性退化：数值迭代在 $b\to b_c$ 时收敛速度显著下降（附录 C 已提及），且未给出收敛速率与 $|b-b_c|$ 的标度关系。对于 $s=0.8$ 时 $m_0\sim|b-b_c|^4$ 的高指数行为，$b_c$ 定位需极高精度；文中扫描精度为 2–4 轮精细化，但未说明最终 $b_c$ 的不确定度对临界指数拟合的影响。

    • 单激发散射截断缺乏定量验证：附录 H 的散射计算限定在单激发扇区，忽略了多粒子产生通道。双粒子振幅与单粒子振幅的比值作为能量的函数未被计算，因此无法确认在 $\omega\ll D$ 的低能区单激发近似是否一致有效。对于亚欧姆浴，红外行为可能增强多玻色子过程的相对权重。

    • 局域相（$\theta>0$）的分析不完整：论文主要关注离域相（$\theta=0$）的临界行为，对局域相仅给出能量表达式（式 S26）和 $\sin^2\theta=m_0$ 关系，未讨论局域相的磁化率、自旋‑自旋关联长度、或局域磁矩的有限大小标度。这些量是实验可观测的，且与量子比特退相干直接相关。

    • $s=0.5$ 的对数修正缺失：当 $s=0.5$ 时，两个临界指数 $1$ 和 $s/(1-s)=1$ 简并，文中仅提到"预期会有对数修正"（附录 G 末），但未推导对数修正的具体形式（如 $m_0\sim|b-b_c|^2/\log|b-b_c|$ 或 $m_0\sim|b-b_c|^2(\log|b-b_c|)^{\pm1}$），也未在数值上验证此简并点的标度行为。

    • 零温限制：所有计算限于 $T=0$ 的基态。有限温度下高斯态 $|\phi\rangle$ 将替换为热态（混合高斯态），但 $\sin^2\theta=m_0$ 的关系、临界耦合的温度漂移、以及有限温相变是否存在等问题均未讨论。要将压缩极化子推广到 $T>0$ 需要将变分原理替换为自由能最小化。

    • 与多通道或多杂质的不可推广性：论文处理的是单杂质耦合至单一连续玻色子浴。对于多杂质（如自旋链耦合至公共浴）或多通道（如两自旋分别耦合至不同浴），$m(y)$ 将推广为矩阵值函数 $M_{\alpha\beta}(y)$，自洽方程的维度随通道数平方增长。文中未说明这种推广是否仍能保持可解性。

    • $\sin^2\theta=m_0$ 的证明依赖于 $y_c\to\infty$ 前提：附录 J 的推导在将 $y_c$ 积分上限延拓至无穷后方可闭合，对于有限 $y_c$ 截断（数值计算真实情况），$m_0$ 与 $\sin^2\theta$ 之间可能存在系统偏差，文中未对此截断效应做误差分析。

    • 高频截止 $\omega_c$ 的物理含义未深入讨论：浴谱函数 $J(\omega)=2\pi\alpha\omega_c^{1-s}\omega^s$ 在 $\omega>\omega_c$ 处突然截断。这种硬截断与实际物理系统（如光子晶体或声子浴）中的平滑衰减不同。文中未讨论截断形状（硬截断 vs. 指数截断 vs. Lorentzian 截断）对临界耦合和临界指数的敏感性。
