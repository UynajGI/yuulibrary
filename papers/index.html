    表面声波声子作为量子信息的传播载体——声学量子光学

    超导量子比特的纳米机械测量 LaHaye 2009 NEMS 谐振器与 Cooper-pair box 色散耦合，机械版「单原子折射率效应」。 › 边带冷却微机械运动至量子基态 Teufel 2011 铝膜冷却到 nm = 0.34，强耦合 + 近量子极限测量，开启量子声学。 › 混合电路腔量子电动力学与微机械谐振器 Pirkkalainen 2013 transmon 同时耦合微波腔与机械膜，声子 Stark 频移 + 边带 Rabi 振荡。 › 机械振子在热退相干率处的测量与控制 Wilson 2015 近场光机械测量不精确度低于 SQL 40 dB，反馈冷却到 nm = 5.3。 › 低于标准量子极限的连续力与位移测量 Mason 2019 ponderomotive 挤压测量低于 SQL 1.5 dB，机械量子传感的里程碑。 ›
    在固态量子比特中观测到 Berry 相位 Leek 2007 Cooper pair box 中首次可控积累几何相位，固态几何量子计算奠基。 › 行进单个微波光子的实验态层析 Eichler 2011 线性放大器测四阶矩，重建行进单光子 Wigner 函数，负值证明量子性。 › 耗散驱动的 Rabi 模型中的量子相变信号 De Filippis 2023 耗散 Rabi 模型在深强耦合区发生 BKT 量子相变，序参量与磁化率挂钩。 ›
    传播声子耦合到人工原子 Gustafsson 2014 表面声波声子耦合 transmon，声子版量子光学：非线性反射、单声子饱和。 ›
