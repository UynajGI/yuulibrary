# 理解策略风险

## 15.1 动机

正如我们在[第 3 章](ch03.md)和[第 13 章](ch13.md)中看到的，投资策略通常以持仓直到满足两个条件之一来实施：(1) 以盈利退出头寸的条件（止盈），或 (2) 以亏损退出头寸的条件（止损）。即使策略没有明确声明止损，也总有一个隐含的止损限制，在此投资者无法再为其头寸融资（追加保证金通知）或承受不断增加的未实现亏损带来的痛苦。由于大多数策略都（隐含或明确地）有这两个退出条件，通过二项过程建模结果分布是有意义的。这反过来将帮助我们理解什么组合的下注频率、赔率和收益是不经济的。本章的目标是帮助你评估策略何时容易受到这些变量中任何一个的微小变化的影响。

## 15.2 对称收益

考虑一个每年产生 n 个 IID 下注的策略，其中下注 i ∈ [1, n] 的结果 X~i~ 是概率为 P[X~i~ = π] = p 的利润 π > 0，以及概率为 P[X~i~ = −π] = 1 − p 的损失 −π。你可以将 p 视为二元分类器的精确率，其中正例意味着对机会下注，负例意味着放过机会：真阳性被奖励，假阳性被惩罚，而负例（无论真假）没有收益。由于下注结果 {X~i~}~i=1,...,n~ 是独立的，我们将计算每次下注的期望矩。一次下注的期望利润为 E[X~i~] = πp + (−π)(1 − p) = π(2p − 1)。方差为 V[X~i~] = E[X~i~²] − E[X~i~]²，其中 E[X~i~²] = π²p + (−π)²(1 − p) = π²，因此 V[X~i~] = π² − π²(2p − 1)² = π²[1 − (2p − 1)²] = 4π²p(1 − p)。对于每年 n 个 IID 下注，年化夏普比率（θ）为

注意 π 如何从上面的方程中抵消，因为收益是对称的。就像在高斯情况下一样，θ[p, n] 可以理解为重新缩放的 t 值。这说明了即使在 ![](images/Image00048.webp) 很小的情况下，对于足够大的 n，夏普比率也可以很高。这是高频交易的经济基础，其中 p 可以略高于 0.5，成功业务的关键是增加 n。夏普比率是精确率而非准确率的函数，因为放过机会（负例）不直接被奖励或惩罚（尽管太多负例可能导致小的 n，这将使夏普比率向零压低）。

## 15.3 非对称收益

考虑一个每年产生 n 个 IID 下注的策略，其中下注 i ∈ [1, n] 的结果 X~i~ 是概率为 P[X~i~ = π~+~] = p 的 π~+~，以及概率为 P[X~i~ = π~−~] = 1 − p 的 π~−~（π~−~ < π~+~）。一次下注的期望利润为 E[X~i~] = pπ~+~ + (1 − p)π~−~ = (π~+~ − π~−~)p + π~−~。方差为 V[X~i~] = E[X~i~²] − E[X~i~]²，其中 E[X~i~²] = pπ~+~² + (1 − p)π~−~² = (π~+~² − π~−~²)p + π~−~²，因此 V[X~i~] = (π~+~ − π~−~)²p(1 − p)。对于每年 n 个 IID 下注，年化夏普比率（θ）为

## 15.4 策略失败的概率

在上面的例子中，参数 π~−~ = −.01、π~+~ = .005 由投资组合经理设定，随执行订单传给交易员。参数 n = 260 也由投资组合经理设定，因为她决定什么构成值得下注的机会。不受投资组合经理控制的两个参数是 p（由市场决定）和 θ*（投资者设定的目标）。因为 p 未知，我们可以将其建模为随机变量，期望值为 E[p]。让我们定义 ![](images/Image00703.webp) 为 p 的值，低于该值策略将表现不如目标夏普比率 θ*，即 ![](images/Image00111.webp)。我们可以使用上面的方程（或 `binHR` 函数）得出，对于 ![](images/Image00030.webp)，![](images/Image00122.webp)。这突显了该策略涉及的风险，因为 p 的相对较小下降（从 p = .7 到 p = .67）将抹去所有利润。该策略本质上是有风险的，即使持仓不是。这就是我们希望在本章中确立的关键区别：*策略风险*不应与*投资组合风险*混淆。

大多数公司和投资者计算、监控和报告投资组合风险，却没有意识到这不能告诉我们策略本身的风险。策略风险不是首席风险官计算的底层投资组合的风险。策略风险是投资策略随着时间推移将无法成功的风险，这个问题与首席投资官更为相关。回答「这个策略失败的概率是多少？」等价于计算 ![](images/Image00212.webp)。以下算法将帮助我们计算策略风险。

    1. 该策略可行吗？
    2. *Ceteris paribus*，使策略有利可图所需的精确率是多少？
