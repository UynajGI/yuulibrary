如今，随机过程（Stochastics）是金融学中最重要的数学与数值学科之一。在现代金融学早期，主要是20世纪70年代和80年代，金融研究的主要目标是为特定金融模型下的期权定价等问题寻找闭式解。近年来，需求已发生巨大变化——金融市场参与者不仅需要正确估值单个金融工具，还需要对整本衍生品账簿等进行一致估值。同样，要在整个金融机构范围内得出一致的风险度量指标（如风险价值（Value-at-Risk, VaR）和信用估值调整（Credit Valuation Adjustment, CVA）），必须考虑机构的所有账簿及其所有交易对手。这些艰巨的任务只能依靠灵活高效的数值方法来解决。因此，随机过程一般方法以及蒙特卡洛模拟（Monte Carlo Simulation）这一特定方法在金融领域日益重要。

## 模拟

蒙特卡洛模拟（Monte Carlo Simulation, MCS）是金融中最重要的数值技术之一，甚至可以说是最重要且最广泛使用的。这主要源于它是在评估数学表达式（例如积分）方面最灵活的数值方法，尤其是对金融衍生品的估值。然而，灵活性是以相对较高的计算负担为代价的，因为往往需要进行数十万甚至数百万次复杂计算才能得出一个估值。

## 随机变量

例如，考虑 Black-Scholes-Merton 期权定价框架。在该框架下，给定当前水平 $S_{0}$，未来日期 T 的股票指数水平 $S_{T}$ 由公式12-1给出。

{{< caption >}}公式12-1 Black-Scholes-Merton 框架中模拟未来指数水平{{< /caption >}}

$$ $$
S_{T} = S_{0} \exp \left(\left(r - \frac{1}{2} \sigma^{2}\right) T + \sigma \sqrt{T} z\right)


变量和参数的含义如下：

$S_{T}$ 日期 $T$ 的指数水平 $r$ 恒定无风险短期利率
$\sigma$ S 的恒定波动率（即收益率的标准差）
$z$ 标准正态分布随机变量

该金融模型参数化并进行如下模拟。模拟代码的输出如{{< caption >}}图12-3{{< /caption >}}所示：

```python
In [12]: S0 = 100  # 初始指数水平
    r = 0.05  # 恒定无风险短期利率
    sigma = 0.25  # 恒定波动率因子
    T = 2.0  # 以年为单位的时间跨度
    I = 10000  # 模拟次数
    ST1 = S0 * np.exp((r - 0.5 * sigma ** 2) * T + sigma * math.sqrt(T) * npr.standard_normal(I))
In [13]: plt.figure(figsize=(10, 6))
plt.hist(ST1, bins=50)
plt.xlabel('index level')
plt.ylabel('frequency');
```

## 随机过程

粗略地说，随机过程（Stochastic Process）是一个随机变量序列。在这个意义上，模拟过程时应该期望类似于对随机变量进行重复模拟的结果。这基本上是成立的，除了抽取通常不是独立的，而是依赖于前一次抽取的结果。不过，金融中使用的随机过程通常具有马尔可夫性（Markov Property），即过程的明天值仅取决于过程的今天状态，而不取决于任何其他更”历史”的状态甚至整个路径历史。这种过程也称为无记忆过程。

## 几何布朗运动

现在考虑 Black-Scholes-Merton 模型的动态形式，如公式12-2中的随机微分方程（Stochastic Differential Equation, SDE）所述。其中 $Z_{t}$ 是标准布朗运动（Standard Brownian Motion）。该 SDE 被称为几何布朗运动（Geometric Brownian Motion, GBM）。$S_{t}$ 的值呈对数正态分布，（边际）收益率 $\frac{d S_{t}} {S_{t}}$ 呈正态分布。

{{< caption >}}公式12-2 Black-Scholes-Merton 框架中的随机微分方程{{< /caption >}}

$$ $$
d S_{t} = r S_{t} d t + \sigma S_{t} d Z_{t}


公式12-2中的 SDE 可以通过 Euler 方案精确离散化。该方案如公式12-3所示，其中 Δt 是固定的离散化区间，$z_{t}$ 是标准正态分布随机变量。

{{< caption >}}公式12-3 Black-Scholes-Merton 框架中动态模拟指数水平{{< /caption >}}

$$ $$
S_{t} = S_{t - \Delta t} \exp \left(\left(r - \frac{1}{2} \sigma^{2}\right) \Delta t + \sigma \sqrt{\Delta t} z_{t}\right)

## 平方根扩散

另一类重要的金融过程是均值回复过程（Mean-Reverting Process），用于建模短期利率或波动率过程等。一个流行且广泛使用的模型是 Cox、Ingersoll 和 Ross（1985）提出的平方根扩散（Square-Root Diffusion）。公式12-4给出了相应的 SDE。

{{< caption >}}公式12-4 平方根扩散的随机微分方程{{< /caption >}}

## 随机波动率

Black-Scholes-Merton 模型的主要简化假设之一是恒定波动率。然而，波动率通常既不是恒定的也不是确定性的——它是随机的。因此，20世纪90年代初引入的所谓随机波动率模型（Stochastic Volatility Model）是金融建模的重大进展。该类中最流行的模型之一是 Heston（1993）模型，如公式12-7所示。

{{< caption >}}公式12-7 Heston 随机波动率模型的随机微分方程{{< /caption >}}

$$ $$
\begin{array}{r l r} {d S_{t}} & = & {r S_{t} d t + \sqrt{\nu_{t}} S_{t} d Z_{t} ^{1}} \\ {d \nu_{t}} & = & {\kappa_{\nu} (\theta_{\nu} - \nu_{t}) d t + \sigma_{\nu} \sqrt{\nu_{t}} d Z_{t} ^{2}} \\ {d Z_{t} ^{1} d Z_{t} ^{2}} & = & {\rho} \end{array}


变量和参数的含义可以从几何布朗运动和平方根扩散的讨论中容易推断出来。参数 $\rho$ 表示两个标准布朗运动 $Z_{t} ^{1} , Z_{t} ^{2}$ 之间的瞬时相关性。这使我们能够解释一个称为杠杆效应（Leverage Effect）的风格化事实，其本质是在压力时期（市场下跌）波动率上升，而在牛市时期（市场上涨）波动率下降。

## 估值

蒙特卡洛模拟最重要的应用之一是或有债权（期权、衍生品、混合工具等）的估值。简单来说，在风险中性世界中，或有债权的价值就是风险中性（鞅）测度下的贴现期望收益。该概率测度使所有风险因子（股票、指数等）以无风险短期利率漂移，从而使贴现过程成为鞅。根据资产定价基本定理（Fundamental Theorem of Asset Pricing），这种概率测度的存在等价于无套利条件。

金融期权体现了在给定到期日（欧式期权）或给定时间段内（美式期权）以给定价格（行权价）买入（看涨期权）或卖出（看跌期权）指定金融工具的权利。让我们首先考虑较简单的欧式期权估值情况。

## European Options

欧式看涨期权在到期日的收益由 $h ( S_{T} ) \equiv$ max $( S_{T} - K , 0 )$ 给出，其中 $S_{T}$ 是到期日 T 的指数水平，K 是行权价（Strike Price）。给定相关随机过程（如几何布朗运动）的风险中性测度，此类期权的价格由公式12-10给出。

{{< caption >}}公式12-10 风险中性期望定价{{< /caption >}}

$$ $$
C_{0} = e^{- r T} \mathbf{E} _{0} ^{Q} (h (S_{T})) = e^{- r T} \int_{0} ^{\infty} h (s) q (s) d s


[第11章](ch11.md)介绍了如何通过蒙特卡洛模拟数值评估积分。该方法将在以下内容中使用并应用于公式12-10。公式12-11提供了欧式期权的相应蒙特卡洛估计量，其中 $\tilde{S} _{T} ^{i}$ 是到期时第 i 个模拟的指数水平。

{{< caption >}}公式12-11 风险中性蒙特卡洛估计量{{< /caption >}}

$$ $$

## 美式期权

与欧式期权相比，美式期权的估值更为复杂。在这种情况下，需要解决一个最优停时问题（Optimal Stopping Problem）才能得出期权的公允价值。公式12-12将美式期权的估值表述为这样一个问题。问题表述已经基于用于数值模拟的离散时间网格。因此从某种意义上说，更准确的表述是在百慕大式行权下给出期权价值。当时间区间趋近于零长度时，百慕大式期权的价值收敛于美式期权的价值。

{{< caption >}}公式12-12 美式期权价格作为最优停时问题{{< /caption >}}

$$ $$
V_{0} = \sup_{\tau \in \{0, \Delta t, 2 \Delta t, \dots , T \}} e^{- r T} \mathbf{E} _{0} ^{Q} (h_{\tau} (S_{\tau}))


下面描述的算法称为最小二乘蒙特卡洛（Least-Squares Monte Carlo, LSM），来自 Longstaff 和 Schwartz（2001）的论文。可以证明，任何给定日期 t 的美式（百慕大式）期权价值为 $V_{t} ( s ) = \operatorname* {m a x} \left( h_{t} ( s ) , C_{t} ( s ) \right)$，其中 $C_{t} ( s ) = \mathbf{E} _{t} ^{Q} \big ( e^{- r \Delta t} V_{_{t + \Delta t}} ( S_{_{t + \Delta t}} ) \big | ~ S_{t} = s \big )$ 是指数水平为 $S_{t} {=} s$ 时所谓的期权持有价值（Continuation Value）。

现在假设我们在 M 个等长 Δt 的时间间隔上模拟了 I 条指数水平路径。定义 $\begin{array} {r} {Y_{t , i} \equiv e^{- r \Delta t} V_{t + \Delta t , i}} \end{array}$ 为路径 i 在时间 t 的模拟持有价值。我们不能直接使用这个数值，因为这将意味着完美预见。然而，我们可以使用所有此类模拟持有价值的横截面数据，通过最小二乘回归来估计（期望）持有价值。

给定一组基函数（Basis Function）$b_{d} , d = 1 , \cdots , D$，持有价值则由回归估计 $\begin{array} {r} {\hat{C} _{t , i} ~ = ~ \sum_{d = 1} ^{D} \alpha_{d , t} ^{*} \cdot b_{d} ( S_{t , i} )} \end{array}$ 给出，其中最优回归参数 α<sup>\*</sup> 是公式12-13中所述最小二乘问题的解。

{{< caption >}}公式12-13 美式期权估值的最小二乘回归{{< /caption >}}

$\min_{\alpha_{1,t},\ldots,\alpha_{D,t}}\frac{1}{I}\sum_{i=1}^{I}\left(Y_{t,i}-\sum_{d=1}^{D}\alpha_{d,t}\cdot b_{d}(S_{t,i})\right)^{2}$

函数 gbm\_mcs\_amer() 同时为美式看涨和看跌期权实现了 LSM 算法：<sup>4</sup>

```python

    参数
    ========
    K: float

    option: string
    要估值的期权类型（'call' 看涨, 'put' 看跌）

    返回
    ========
    C0: float
    美式看涨期权的估计现值
    '''

    df = math.exp(-r * dt)
    # 模拟指数水平

```

```python
sn = gen_sn(M, I)
```

## 风险度量

除估值外，风险管理是随机方法和模拟的另一个重要应用领域。本节说明当今金融行业中两个最常用风险度量的计算/估计。

## 风险价值

风险价值（Value-at-Risk, VaR）是使用最广泛、也最具争议的风险度量之一。从业者喜爱其直观性，但许多人在理论层面上对其广泛讨论和批评，主要涉及其捕捉尾端风险（Tail Risk）的能力有限（稍后将详述）。简而言之，VaR 是一个以货币单位（如美元、欧元、日元）表示的数字，指示在给定时间段内、以某个置信水平（概率）不会超过的（组合、单个头寸等的）损失。

考虑一个当前价值100万美元的股票头寸，在30天（一个月）的时间段内、99%置信水平下的 VaR 为50,000美元。这个 VaR 数字表示，在99%的概率下（即100例中99例），30天内的预期损失不会超过50,000美元。然而，它并未说明一旦损失超过50,000美元时损失的大小——即最大损失是100,000美元还是500,000美元，以及这种特定"高于 VaR 的损失"的概率是多少。它只说明有1%的概率会发生至少50,000美元或更高的损失。

假设 Black-Scholes-Merton 框架，考虑以下参数化和对未来日期 T = 30/365（30天期间）的指数水平模拟。VaR 的估计需要将模拟的绝对盈亏相对于当前头寸价值进行排序，即从最大损失到最大利润。{{< caption >}}图12-18{{< /caption >}}显示了模拟绝对表现值的直方图：

## Python 脚本

以下展示了与 Black-Scholes-Merton 模型相关的核心函数的实现，用于欧式（看涨）期权的解析定价。关于模型的详细信息，请参见 Black 和 Scholes（1973）以及 Merton（1973）。参见附录B中基于 Python 类的替代实现。

```python
def bsm_call_value(S0, K, T, r, sigma):
    ''' BSM 模型中欧式看涨期权的估值。
    解析公式。

    参数

    S0: float
    初始股票/指数水平
    K: float
```

```python

恒定无风险短期利率
sigma: float

返回
=====
value: float
欧式看涨期权的现值
',
from math import log, sqrt, exp
from scipy import stats

S0 = float(S0)
d1 = (log(S0 / K) + (r + 0.5 * sigma ** 2) * T) / (sigma * sqrt(T))
d2 = (log(S0 / K) + (r - 0.5 * sigma ** 2) * T) / (sigma * sqrt(T))
# stats.norm.cdf --> 正态分布的累积分布函数
value = (S0 * stats.norm.cdf(d1, 0.0, 1.0) -
K * exp(-r * T) * stats.norm.cdf(d2, 0.0, 1.0))
return value
```

## 结论

本章涉及蒙特卡洛模拟在金融中应用的重要方法和技术。具体来说，首先展示了如何基于不同分布律生成伪随机数，然后讨论了随机变量和随机过程的模拟——这在许多金融领域都很重要。本章深入讨论了两个应用领域：欧式与美式行权期权的估值，以及风险价值（VaR）和信用估值调整（CVA）等风险度量的估计。

本章说明，Python 结合 NumPy 非常适合实现即使是计算密集型的任务，如通过蒙特卡洛模拟对美式期权进行估值。这主要是因为 NumPy 的大多数函数和类都是用 C 实现的，通常比纯 Python 代码具有显著的速度优势。另一个好处是由于向量化操作，生成的代码紧凑且可读性强。
