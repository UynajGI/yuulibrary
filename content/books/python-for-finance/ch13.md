# 统计学

{{< callout type="quote" >}}
除了真相，我可以用统计数据证明任何事情。
—George Canning
{{< /callout >}}

统计学（Statistics）是一个广阔的领域，但它提供的工具和结果已成为金融领域不可或缺的部分。这解释了为什么 R 等特定领域语言在金融行业中如此流行。统计模型越精细复杂，拥有易用且高性能的计算解决方案就越重要。

一本像这样的书中的单一章节无法全面展现统计学的丰富性和深度。因此，与许多其他章节一样，本章的方法是聚焦于精选的主题，这些主题看似重要，或者为使用 Python 处理特定任务提供了良好的起点。本章有四个重点：

"正态性检验" 见第398页

大量重要的金融模型，如现代投资组合理论（Modern Portfolio Theory, MPT，即均值-方差投资组合理论）和资本资产定价模型（Capital Asset Pricing Model, CAPM），都依赖于证券收益率呈正态分布的假设。因此，本章介绍了检验给定时间序列收益率正态性的方法。

"投资组合优化" 见第415页

MPT 可被视为统计学在金融领域最成功的应用之一。从20世纪50年代初先驱 Harry Markowitz 的工作开始，这一理论开始用严谨的数学和统计方法取代人们在金融市场投资中对判断和经验的依赖。在这个意义上，它或许是金融领域第一个真正的量化模型和方法。

## "贝叶斯统计学" 见第429页

在概念层面上，贝叶斯统计学（Bayesian Statistics）引入了代理信念（Beliefs）和信念更新的概念。在线性回归中，这可能表现为回归参数具有统计分布而非单一点估计（例如回归线的截距和斜率）。如今，贝叶斯方法在金融领域被广泛使用，因此本节通过一些示例来说明贝叶斯方法。

## "机器学习" 见第444页

机器学习（Machine Learning，或称统计学习）基于先进的统计方法，被认为是人工智能（AI）的子学科。与统计学本身一样，机器学习提供了丰富的方法和模型来从数据集中学习，并根据所学内容进行预测。区分了不同的学习算法，如监督学习（Supervised Learning）或无监督学习（Unsupervised Learning）。算法解决的问题类型也有所不同，如估计或分类。本章中的示例属于分类的监督学习范畴。

本章的许多方面涉及日期和/或时间信息。请参阅附录A，了解如何使用 Python、NumPy 和 pandas 处理此类数据。

## 正态性检验

正态分布（Normal Distribution）可以被认为是金融中最重要的分布，也是金融理论的主要统计基石之一。以下金融理论的基石在很大程度上依赖于金融工具收益率呈正态分布的假设：<sup>1</sup>

## 投资组合理论

当股票收益率呈正态分布时，最优投资组合选择可以转化为一个框架，其中只有（期望）平均收益率、收益率方差（或波动率）以及不同股票之间的协方差与投资决策（即最优投资组合构成）相关。

## 资本资产定价模型

同样，当股票收益率呈正态分布时，单只股票的价格可以优雅地表示为与广泛市场指数的线性关系；这种关系通常通过衡量单只股票与市场指数共同运动的指标——贝塔（Beta 或 β）来表示。

## 有效市场假说

有效市场（Efficient Market）是指价格反映所有可用信息的市场，其中"所有"可以狭义或广义地定义（例如，"所有公开可用"信息 vs. 也包括"仅私下可用"信息）。如果该假说成立，那么股票价格随机波动，收益率呈正态分布。

## 期权定价理论

布朗运动是建模金融工具随机价格运动的基准模型；著名的 Black-Scholes-Merton 期权定价公式使用几何布朗运动作为股票随时间随机价格波动的模型，导致价格呈对数正态分布，收益率呈正态分布。

这个远非详尽无遗的列表凸显了正态性假设在金融中的重要性。

## 正态对数收益率

在时间 0 < s < t 之间的对数收益率 log $\begin{array} {r} {\frac{\boldsymbol{S} _{t}} {\boldsymbol{S} _{s}} = \log \boldsymbol{S} _{t} - \log \boldsymbol{S} _{s}} \end{array}$ 呈正态分布。

以下使用函数 gen\_paths() 生成几何布朗运动的样本蒙特卡洛路径（另见[第12章](ch12.md)）：

```python
In [3]: def gen_paths(S0, r, sigma, T, M, I):
    ''' 生成几何布朗运动的蒙特卡洛路径。

参数
====================
S0: float
    初始股票/指数价值
r: float
    恒定短期利率
sigma: float
    恒定波动率
T: float
    最终时间水平
M: int
    时间步长/间隔数
I: int
    要模拟的路径数

返回
====================
paths: ndarray, shape (M + 1, I)
    给定参数下的模拟路径
,...
dt = T / M
paths = np.zeros((M + 1, I))
paths[0] = S0
for t in range(1, M + 1):
    rand = np.random.standard_normal(I)
    rand = (rand - rand.mean()) / rand.std() ①  # 匹配一阶和二阶矩
    paths[t] = paths[t - 1] * np.exp((r - 0.5 * sigma ** 2) * dt + sigma * math.sqrt(dt) * rand)
return paths
```

尽管图形方法很有吸引力，但它们通常无法替代更严格的检验程序。下一个示例中使用的函数 normality\_tests() 结合了三种不同的统计检验：

## 偏度检验（skewtest()）

检验样本数据的偏度是否"正常"（即值足够接近零）。

## 峰度检验（kurtosistest()）

## 投资组合优化

现代投资组合理论（即均值-方差投资组合理论）是金融理论的主要基石之一。基于这一理论突破，其发明者 Harry Markowitz 于1990年获得诺贝尔经济学奖。尽管该理论形成于20世纪50年代，但至今仍在教授给金融专业学生，并在实践中应用（通常带有一些次要或主要修改）。<sup>3</sup> 本节阐述了该理论的基本原则。

Copeland、Weston 和 Shastri（2005）一书的[第5章](ch05.md)介绍了与 MPT 相关的正式主题。如前所述，收益率呈正态分布的假设是该理论的基础：

{{< callout type="quote" >}}
仅关注均值和方差，我们必然假设没有其他统计量是描述期末财富分布所必需的。除非投资者具有特殊类型的效用函数（二次效用函数），否则必须假设收益率具有正态分布，这可以通过均值和方差完全描述。
{{< /callout >}}

## 随机投资组合权重……

## ……归一化为1或100%。

正如验证所示，权重确实加起来等于1；即 $\Sigma_{I} w_{i} = 1$，其中 I 是金融工具的数量，$w_{i} > 0$ 是金融工具 i 的权重。公式13-1给出了给定单个工具权重下的期望投资组合收益率公式。这是一种期望投资组合收益率，因为历史平均表现被认为是未来（期望）表现的最佳估计量。其中 $r_{i}$ 是依赖于状态的未来收益率（假设呈正态分布的收益率值向量），$\mu_{i}$ 是工具 i 的期望收益率。最后，$\boldsymbol{w} ^{T}$ 是权重向量的转置，μ 是期望证券收益率向量。

{{< caption >}}公式13-1 期望投资组合收益率的一般公式{{< /caption >}}

$$
\begin{array}{r c l} \mu_{p} & = & \mathbf{E} \big (\sum_{I} w_{i} r_{i} \big) \\ & = & \sum_{I} w_{i} \mathbf{E} (r_{i}) \\ & = & \sum_{I} w_{i} \mu_{i} \\ & = & w^{T} \mu \end{array}
$$

翻译成 Python 后，这简化为包含年化在内的单行代码：

```javascript
In [50]: np.sum(rets.mean() * weights) * 252
Out[50]: 0.09179459482057793
```

给定投资组合权重下的年化投资组合收益率。

MPT 中第二个重要的对象是期望投资组合方差。两个证券之间的协方差定义为 $\sigma_{i j} = \sigma_{j i} = \mathbf{E} ( r_{i} - \mu_{i} ) ( r_{j} - \mu_{j} )$。证券的方差是与自身协方差的特例：$\dot{\sigma_{i} ^{2}} = \mathbf{E} \big ( ( r_{i} - \mu_{i} ) ^{2} \big )$。{{< caption >}}图13-12{{< /caption >}}提供了证券投资组合的协方差矩阵（假设每只证券的等权重为1）。

$$
\Sigma = \left[ \begin{array}{c c c c} \sigma_{1} ^{2} & \sigma_{12} & \dots & \sigma_{1 I} \\ \sigma_{21} & \sigma_{2} ^{2} & \dots & \sigma_{2 I} \\ \vdots & \vdots & \ddots & \vdots \\ \sigma_{I 1} & \sigma_{I 2} & \dots & \sigma_{I} ^{2} \end{array} \right]
$$

{{< caption >}}图13-12 投资组合协方差矩阵{{< /caption >}}

有了投资组合协方差矩阵，公式13-2给出了期望投资组合方差的公式。

{{< caption >}}公式13-2 期望投资组合方差的一般公式{{< /caption >}}

$$
\begin{array}{r l r} \sigma_{p} ^{2} & = & \mathbf{E} \big ((r - \mu) ^{2} \big) \\ & = & \sum_{i \in I} \sum_{j \in I} w_{i} w_{j} \sigma_{i j} \\ & = & w^{T} \Sigma w \end{array}
$$

In Python, this all again boils down to a single line of code, making heavy use of NumPy vectorization capabilities. The np.dot() function gives the dot product of two vectors/matrices. The T attribute or transpose() method gives the transpose of a vector or matrix. Given the portfolio variance, the (expected) portfolio standard deviation or volatility $\sigma_{{p}} = \sqrt{\sigma_{{p}} ^{2}}$ is then only one square root away:

```txt
In [51]: np.dot(weights.T, np.dot(rets.cov() * 252, weights)) ①
Out[51]: 0.014763288666485574

In [52]: math.sqrt(np.dot(weights.T, np.dot(rets.cov() * 252, weights))) ②
Out[52]: 0.12150427427249452
```

## 资本市场线（Capital Market Line）

除了股票或商品（如黄金）等风险金融工具外，通常还存在一种普遍的无风险投资机会：现金或现金账户。在理想化的世界中，存放在大型银行现金账户中的资金可以被视为无风险（例如，通过公共存款保险计划）。其缺点是，这种无风险投资的收益通常很低，有时接近于零。

然而，考虑到这种无风险资产，投资者的有效投资机会集会显著扩大。基本思想是，投资者首先确定一个风险资产的有效投资组合，然后将无风险资产加入组合。通过调整投资于无风险资产的财富比例，可以在风险-收益空间中实现位于无风险资产与有效投资组合之间直线上的任何风险-收益特征。

在众多选择中，哪一个有效投资组合是最优投资对象？是有效前沿的切线恰好穿过无风险投资组合风险-收益点的那个投资组合。例如，考虑一个无风险利率 $r_{f} = 0.01$。需要在有效前沿上找到这样一个投资组合，使得其切线在风险-收益空间中穿过点 $\left( \sigma_{f} , r_{f} \right) = \left( 0 , 0.01 \right)$。

在接下来的计算中，需要使用有效前沿的函数近似及其一阶导数。三次样条插值提供了这种可微的函数近似（见[第11章](ch11.md)）。对于样条插值，只使用有效前沿上的那些投资组合。通过这种数值方法，可以为有效前沿定义一个连续可微函数 f(x) 及其相应的一阶导数函数 df(x)：

## 贝叶斯统计学（Bayesian Statistics）

贝叶斯统计学（Bayesian Statistics）如今在实证金融中广为流行。本章当然无法为该领域的所有概念奠定基础。因此，如有需要，读者应参考教科书，如 Geweke（2005）的综合导论，或 Rachev（2008）的金融导向版本。

## 贝叶斯公式（Bayes' Formula）

贝叶斯公式在金融中最常见的解释是历时解释（Diachronic Interpretation）。这主要是指随着时间的推移，人们会学习到关于某些感兴趣的变量或参数的新信息，例如时间序列的均值收益。方程13-4正式表述了该定理。

方程13-4 贝叶斯公式

$$
p (H \mid D) = \frac{p (H) \cdot p (D \mid H)}{p (D)}
$$

这里，H 代表事件，即假设（Hypothesis），D 代表实验或现实世界可能呈现的数据（Data）。<sup>5</sup> 基于这些基本概念，我们有：

$p(H)$ 先验概率（Prior Probability）

p(D) 在任何假设下的数据概率，称为归一化常数（Normalizing Constant）

p(D | H) 在假设 H 下的数据似然（Likelihood），即概率

p(H | D)

后验概率（Posterior Probability），即在观察到数据之后的概率

考虑一个简单的例子。有两个盒子，$B_{1}$ 和 $B_{2}$。盒子 $B_{1}$ 包含 30 个黑球和 60 个红球，而盒子 $B_{2}$ 包含 60 个黑球和 30 个红球。从两个盒子中随机抽取一个球。假设抽到的球是黑色的。那么假设 $H_{1}$：球来自盒子 $B_{1}$，和假设 $H_{2}$：球来自盒子 $B_{2}$ 的概率分别是多少？

在随机抽取球之前，两个假设同样可能。当确定球是黑色之后，需要根据贝叶斯公式更新两个假设的概率。考虑假设 $H_{1}$：

• 先验概率：$\begin{array} {r} {p ( H_{1} ) = \frac{1} {2}} \end{array}$

• 归一化常数：$\begin{array} {r} {p ( D ) = {\frac{1} {2}} \cdot{\frac{1} {3}} + {\frac{1} {2}} \cdot{\frac{2} {3}} = {\frac{1} {2}}} \end{array}$

• 似然：$\begin{array} {r} {p ( D \mid H_{1} ) = \frac{1} {3}} \end{array}$

由此得到 $H_{1}$ 的更新后概率为 $\begin{array} {r} {p ( H_{1} \mid D ) = \frac{\frac 12 \cdot \frac 13} {\frac 12} = \frac 1 {3} .} \end{array}$

这个结果在直观上也是有道理的。从盒子 $B_{2}$ 中抽到黑球的概率是从盒子 $B_{1}$ 中抽到黑球的两倍。因此，在抽到黑球后，假设 $H_{2}$ 的更新后概率为 $\begin{array} {r} {p ( H_{2} \mid D ) = \frac{2} {3}} \end{array}$，是假设 $H_{1}$ 更新后概率的两倍高。

贝叶斯回归的一个主要元素是马尔可夫链蒙特卡洛（Markov Chain Monte Carlo，MCMC）采样。<sup>7</sup> 原则上，这与上一节简单示例中多次从盒子中抽球相同——只是以更系统、更自动化的方式进行。

在技术采样方面，有三个不同的函数需要调用：

• find\_MAP() 通过推导局部最大后验（Maximum A Posteriori）点来寻找采样算法的起点。

• sample() 根据 find\_MAP() 的起始值和 NUTS 算法的最优步长，抽取一定数量的样本。

## 随时间更新估计值（Updating Estimates over Time）

如前所述，金融中的贝叶斯方法通常在被视为历时性（Diachronic）时最为有用——即随着时间的推移，新数据的出现使得通过更新或学习能够获得更好的回归和估计。

为了在当前示例中融入这一概念，假设回归参数不仅是随机的并以某种方式分布，而且它们随时间遵循某种随机游走（Random Walk）。这与金融理论中从随机变量过渡到随机过程（本质上是随机变量的有序序列）时所使用的推广相同。

为此，定义一个新的 PyMC3 模型，这次将参数值指定为随机游走。在指定了随机游走参数的分布之后，接着指定 alpha 和 beta 的随机游走。为了使整个过程更高效，每 50 个数据点共享共同的系数：

```python
In [28]: from pymc3.distributions.timeseries import GaussianRandomWalk

In [29]: subsample_alpha = 50
    subsample_beta = 50

In [30]: model_randomwalk = pm.Model()
    with model_randomwalk:
    sigma_alpha = pm.Exponential('sig_alpha', 1. / .02, testval=.1)
    sigma_beta = pm.Exponential('sig_beta', 1. / .02, testval=.1)
    alpha = GaussianRandomWalk('alpha', sigma_alpha ** -2,
    shape=int(len(data) / subsample_alpha))
    beta = GaussianRandomWalk('beta', sigma_beta ** -2,
    shape=int(len(data) / subsample_beta))
    alpha_r = np.repeat(alpha, subsample_alpha)
    beta_r = np.repeat(beta, subsample_beta)
    regression = alpha_r + beta_r * data['GDX'].values[:2100]
    sd = pm.Uniform('sd', 0, 20)
    likelihood = pm.Normal('GLD', mu=regression, sd=sd,
    observed=data['GLD'].values[:2100])
```

## 监督学习（Supervised Learning）

监督学习是在已知结果或观测数据形式的指导下进行的机器学习。这意味着原始数据已经包含了 ML 算法应当学习的内容。接下来的重点将放在分类问题上，而非估计问题。估计问题通常是关于实值数量的估计，而分类问题的特点是将特定的特征组合分配给来自相对较小的类别（整数值）集合中的某个类别（整数值）。

上一小节的示例表明，在无监督学习中，算法会为识别出的簇自行生成分类标签。对于四个簇，标签是 0、1、2 和 3。在监督学习中，这样的分类标签已经给定，因此算法可以学习特征与类别之间的关系。换句话说，在拟合步骤中，算法知道给定的特征值组合对应的正确类别。

本小节说明了以下分类算法的应用：高斯朴素贝叶斯（Gaussian Naive Bayes）、逻辑回归（Logistic Regression）、决策树（Decision Trees）、深度神经网络（Deep Neural Networks）和支持向量机（Support Vector Machines）。<sup>10</sup>

## 深度神经网络（Deep Neural Networks）

深度神经网络（DNN）被认为是用于估计和分类的最强大——但也是计算需求最高的——算法之一。Google 开源的 TensorFlow 包以及相关的成功故事在一定程度上推动了它们的流行。DNN 能够学习和建模复杂的非线性关系。虽然它们的起源可以追溯到 20 世纪 70 年代，但直到最近由于硬件（CPU、GPU、TPU）、数值算法和相关软件实现的进步，才在大规模上变得可行。

虽然其他 ML 算法，如 LR 类型的线性模型，可以基于标准优化问题高效拟合，但 DNN 依赖于深度学习，这通常需要大量重复步骤来调整某些参数（权重）并将结果与数据进行比较。在这个意义上，深度学习可以比作数学金融中的蒙特卡洛模拟，例如，欧式看涨期权的价格可以基于标的资产的 100,000 条模拟路径来估计。另一方面，Black-Scholes-Merton 期权定价公式具有闭式形式，可以解析计算。

虽然蒙特卡洛模拟是数学金融中最灵活和最强大的数值技术之一，但需要付出高计算负担和大内存占用的代价。深度学习也是如此，它通常比许多其他 ML 算法更灵活，但需要更大的计算能力。

## 结论（Conclusion）

统计学本身不仅是一门重要的学科，还为许多其他学科（如金融和社会科学）提供了不可或缺的工具。在一个章节中不可能全面概述如此庞大的主题。因此，本章聚焦于四个重要主题，在真实示例的基础上说明了 Python 及若干统计库的使用：

## 正态性

关于金融市场收益的正态性假设对许多金融理论和应用至关重要；因此，能够检验特定时间序列数据是否符合这一假设非常重要。如第398页的"正态性检验"所见——通过图形和统计手段——现实世界的收益数据通常不是正态分布的。

## 投资组合优化

现代投资组合理论（MPT）关注收益的均值和方差/波动率，不仅可以被视为统计学在金融领域的早期成就之一，也是其主要的概念性成功之一；投资多样化的重概念在这一背景下得到了精美的阐述。

## 贝叶斯统计学

贝叶斯统计学（特别是贝叶斯回归）已成为金融中流行的工具，因为这种方法克服了其他方法的一些缺点，例如在[第11章](ch11.md)中介绍的方法；即使数学和形式化更为复杂，其基本思想——如随时间更新概率/分布信念——也很容易掌握（至少在直觉上是这样）。

## 机器学习

如今，机器学习已在金融领域与传统统计方法和技术并存。本章介绍了无监督学习（如 k-means 聚类）和有监督学习（如 DNN 分类器）的 ML 算法，并说明了选定的相关主题，如特征变换和训练-测试集划分。

## 算法交易（Algorithmic Trading）

本书的这一部分是关于使用 Python 进行算法交易。越来越多的交易平台和经纪商允许其客户使用 REST API 以编程方式检索历史数据或流数据，或下达买卖订单。长期以来属于大型金融机构的领域，现在即使是零售算法交易者也可以触及。在这个领域，Python 作为编程语言和技术平台已经确立了顶级地位。除其他因素外，这得益于许多交易平台（如 FXCM  Forex Capital Markets）为其 REST API 提供了易于使用的 Python 包装包。

本书的这一部分包括三章：

• [第14章](ch14.md)介绍 FXCM 交易平台、其 REST API 和 fxcmpy 包装包。

• [第15章](ch15.md)侧重于使用统计学和机器学习的方法来推导算法交易策略；本章还展示了如何使用向量化回测。

• 第16章探讨自动化算法交易策略的部署；涉及资金管理、绩效与风险回测、在线算法以及部署。
