# 交易策略

{{< callout type="quote" >}}
[T]他们愚蠢到认为可以通过观察过去来预测未来。
—The Economist<sup>1</sup>
{{< /callout >}}

本章介绍算法交易策略的向量化回测。术语"算法交易策略"（algorithmic trading strategy）用于描述任何基于算法的金融交易策略，该算法旨在自主地（无需人工干预）对金融工具建立多头、空头或中性头寸。一个简单的算法，如"每五分钟在苹果公司股票的多头和中性头寸之间切换"，就符合这一定义。从更技术的角度来说，本章中，算法交易策略由一段Python代码表示，该代码在新数据可用时，决定买入或卖出某种金融工具，以对其建立多头、空头或中性头寸。

本章不提供算法交易策略的全面概述（关于算法交易策略的详细参考资料，请参见第519页的"延伸资源"）。而是重点介绍几种精选策略的向量化回测（vectorized backtesting）技术方面。在这种方法中，被测试策略所基于的金融数据通常整体上进行操作，对存储金融数据的NumPy ndarray和pandas DataFrame对象应用向量化操作。<sup>2</sup>

本章的另一个重点是应用机器学习和深度学习算法来制定算法交易策略。为此，在历史数据上训练分类算法，以预测未来的方向性市场运动。这通常需要将金融数据从实际值转换为相对较少的分类值。<sup>3</sup> 这使我们能够利用此类算法的模式识别能力。

本章分为以下几个部分：

## 向量化回测

现在可以按如下方式实施向量化回测。首先，计算对数收益率。然后将以+1或-1表示的头寸乘以其对应的对数收益率。这种简单的计算是可行的，因为多头头寸赚取苹果股票的收益率，而空头头寸赚取苹果股票的负收益率。最后，需要将苹果股票和基于SMA的算法交易策略的对数收益率分别累加，并应用指数函数以获得业绩数值：

## 简化假设

本小节介绍的向量化回测方法基于若干简化假设。其中包括未考虑交易成本（固定费用、买卖价差、借券成本等）。对于一个在多年间仅产生少量交易的交易策略来说，这可能情有可原。此外，还假设所有交易均按苹果股票的日终收盘价执行。更真实的回测方法需要将这些和其他（市场微观结构）因素考虑在内。

## 优化

自然会有一个问题：所选参数SMA1=42和SMA2=252是否"正确"。通常，在其他条件相同的情况下，投资者偏好更高的收益而非更低的收益。因此，人们可能会倾向于寻找能在相关时期内最大化收益的参数。为此，可以使用暴力搜索方法，简单地重复不同参数组合下的整个向量化回测过程，记录结果，然后进行排序。以下代码实现了这一过程：

In [21]: from itertools import product

```python
In [22]: sma1 = range(20, 61, 4) ①
    sma2 = range(180, 281, 10) ②

In [23]: results = pd.DataFrame()
    for SMA1, SMA2 in product(sma1, sma2): ③
    data = pd.DataFrame(raw[symbol])
    data.dropna(inplace=True)
    data['Returns'] = np.log(data[symbol] / data[symbol].shift(1))
    data['SMA1'] = data[symbol].rolling(SMA1).mean()
    data['SMA2'] = data[symbol].rolling(SMA2).mean()
    data.dropna(inplace=True)
    data['Position'] = np.where(data['SMA1'] > data['SMA2'], 1, -1)
    data['Strategy'] = data['Position'].shift(1) * data['Returns']
    data.dropna(inplace=True)
    perf = np.exp(data[['Returns', 'Strategy']].sum())
    results = results.append(pd.DataFrame(
    {'SMA1': SMA1, 'SMA2': SMA2,
    'MARKET': perf['Returns'],
    'STRATEGY': perf['Strategy'],
    'OUT': perf['Strategy'] - perf['Returns']},
    index=[0]), ignore_index=True) ④
```

① 指定SMA1的参数值。

② 指定SMA2的参数值。

③ 将SMA1的所有值与SMA2的所有值组合。

④ 将向量化回测结果记录在DataFrame对象中。

## 过拟合

通常，算法交易策略中的任何类型的优化、拟合或训练都容易出现过拟合（overfitting）。这意味着所选参数可能在所使用的数据集上表现（异常）良好，但在其他数据集或实际应用中可能表现（异常）糟糕。

## 随机游走假说

上一节介绍了向量化回测作为回测算法交易策略的高效工具。基于单个金融时间序列（即苹果股票的历史日终价格）回测的单一策略，在同一时期内优于简单地长期持有苹果股票的基准投资。

尽管这些结果具有特定的性质，但它们与随机游走假说（Random Walk Hypothesis, RWH）的预测相悖——即此类预测方法不应产生任何超额收益。RWH假设金融市场中的价格遵循随机游走，或者在连续时间中，服从无漂移的算术布朗运动。无漂移的算术布朗运动在未来任何时间点的预期值都等于其当前值。<sup>6</sup> 因此，如果RWH成立，从最小二乘意义上讲，对明日价格的最佳预测就是今日价格。

其后果总结如下：

{{< callout type="quote" >}}
多年来，经济学家、统计学家和金融学教师一直致力于开发和检验股票价格行为模型。从这项研究中演变出的一个重要模型是随机游走理论。该理论对许多其他描述和预测股票价格行为的方法——这些方法在学术界以外相当流行——提出了严重质疑。例如，我们稍后将看到，如果随机游走理论是对现实的准确描述，那么各种预测股票价格的"技术"或"图表"方法就完全没有价值。
—Eugene F. Fama (1965)
{{< /callout >}}

RWH与有效市场假说（Efficient Markets Hypothesis, EMH）是一致的，通俗地说，该假说认为市场价格反映了"所有可用信息"。通常区分不同程度的效率，如弱式、半强式和强式，更具体地定义了"所有可用信息"的含义。从形式上讲，这样的定义可以基于理论上的信息集概念和编程上的数据集概念，如下面的引文所示：

{{< callout type="quote" >}}
如果基于信息集S进行交易不可能获得经济利润，那么市场相对于信息集S是有效的。
—Michael Jensen (1978)
{{< /callout >}}

使用Python，可以针对特定情况测试RWH。使用一个历史市场价格金融时间序列，为其创建多个滞后版本——例如五个。然后使用OLS回归基于之前创建的滞后市场价格来预测市场价格。其基本思想是，昨天和之前四天的市场价格可以用来预测今天的市场价格。

## 线性OLS回归

本节应用线性OLS回归基于历史对数收益率来预测市场运动的方向。为简单起见，仅使用两个特征。第一个特征（lag\_1）表示滞后一天的对数收益率。第二个特征（lag\_2）表示滞后两天的对数收益率。对数收益率——与价格不同——通常是平稳的，这往往是应用统计和机器学习算法的必要条件。

使用滞后对数收益率作为特征的基本思想是，它们可能对预测未来收益率具有信息价值。例如，可以假设在两次下跌之后，上涨的可能性更大（"均值回复"），或者相反，再次下跌的可能性更大（"动量"或"趋势"）。回归技术的应用使得这种非正式推理形式化成为可能。

## 特征类型

本章专门使用滞后收益率数据作为特征数据，且主要以二值化或数字化形式呈现。这主要是为了方便，因为此类特征数据可以直接从金融时间序列本身推导得出。然而，在实际应用中，特征数据可以来自多种不同的数据源，可能包括其他金融时间序列及其衍生统计量、宏观经济数据、公司财务指标或新闻文章。关于此主题的深入讨论，请参考López de Prado (2018)。此外，还有一些Python包可用于自动化时间序列特征提取，例如tsfresh。

## 随机训练-测试分割

分类算法在二值化或数字化特征数据上进行训练和测试。其思想是，特征值模式能够以高于50%的命中率预测未来的市场运动。这里隐含的假设是，模式的预测能力随时间持续存在。从这个意义上说，算法在数据的哪一部分进行训练、在哪一部分进行测试，应该不会（太）有影响——这意味着我们可以打破数据的时间顺序进行训练和测试。

一种典型的方法是使用随机训练-测试分割来测试分类算法在样本外的性能——再次尝试模拟现实情况，即算法在交易中持续面对新数据。使用的方法与第12章"训练-测试分割：支持向量机"中应用于样本数据的方法相同。基于此方法，SVM算法再次显示出最好的样本外性能（参见{{< caption >}}图15.16{{< /caption >}}）：

## 深度神经网络

深度神经网络（Deep Neural Network, DNN）试图模拟人脑的功能。它们通常由一个输入层（特征）、一个输出层（标签）和若干隐藏层组成。隐藏层的存在使得神经网络具有深度，使其能够学习更复杂的关系，并在多种问题类型上表现得更好。当应用DNN时，通常称之为深度学习（deep learning）而非机器学习。有关此领域的介绍，请参考Géron (2017)或Gibson和Patterson (2017)。

## 业绩结果

截至目前所示的不同算法交易策略在向量化回测中的所有业绩结果仅用于说明目的。除了无交易成本的简化假设外，结果还依赖于许多其他（大多任意选择的）参数。它们还依赖于整个过程中使用的相对较小的EUR/USD汇率日终价格数据集。重点在于说明不同方法和机器学习算法在金融数据上的应用，而非推导出一个可在实践中部署的稳健算法交易策略。下一章将讨论其中的一些问题。

## 结论

本章介绍了算法交易策略，并基于向量化回测评估其性能。首先从一个基于两条简单移动平均线相当简单的算法交易策略开始，这种策略在实践中已为人所知并使用了数十年。该策略用于说明向量化回测，充分利用了NumPy和pandas的数据分析向量化能力。

通过OLS回归，本章还基于真实金融时间序列说明了随机游走假说。这是衡量任何算法交易策略是否有价值的基准。

本章的核心是将第12章中介绍的机器学习算法应用于实践。使用了多种算法（其中大多数属于分类类型），并基本遵循相同的"节奏"。在特征方面，使用了滞后对数收益率数据的多种变体——尽管这一限制显然不是必须的，主要是出于方便和简单的考虑。此外，分析基于若干简化假设，因为重点主要在于将机器学习算法应用于金融时间序列数据以预测金融市场运动方向的技术方面。
