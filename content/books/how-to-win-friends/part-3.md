## 本部分章节

<a class="part-chapter" href="ch10.html">
  <span class="part-chapter-num">第 10 章</span>
  <span class="part-chapter-title">第一大忌：伤敌一万，自损八千</span>
  <span class="part-chapter-desc">避免争论——你赢不了争论，即使辩赢了也失去了人心和合作的可能。</span>
</a>

<a class="part-chapter" href="ch11.html">
  <span class="part-chapter-num">第 11 章</span>
  <span class="part-chapter-title">第二大忌：一句惹怒全世界的话</span>
  <span class="part-chapter-desc">尊重他人意见，永远不要说"你错了"——这是避免冲突的黄金法则。</span>
</a>

<a class="part-chapter" href="ch12.html">
  <span class="part-chapter-num">第 12 章</span>
  <span class="part-chapter-title">拆招卸力</span>
  <span class="part-chapter-desc">如果错了，立即坦率地承认——这能化解敌意，让对手无话可说。</span>
</a>

<a class="part-chapter" href="ch13.html">
  <span class="part-chapter-num">第 13 章</span>
  <span class="part-chapter-title">开口前结局已定</span>
  <span class="part-chapter-desc">以友善的方式开始——友好的态度是一切说服工作的前提和基础。</span>
</a>

<a class="part-chapter" href="ch14.html">
  <span class="part-chapter-num">第 14 章</span>
  <span class="part-chapter-title">如何引导对方的思路</span>
  <span class="part-chapter-desc">让对方立刻说"是"——苏格拉底式提问，引导对方进入肯定的思维模式。</span>
</a>

<a class="part-chapter" href="ch15.html">
  <span class="part-chapter-num">第 15 章</span>
  <span class="part-chapter-title">贵人语迟</span>
  <span class="part-chapter-desc">让对方多说话——倾听是最好的说服，让对方充分表达自己的观点。</span>
</a>

<a class="part-chapter" href="ch16.html">
  <span class="part-chapter-num">第 16 章</span>
  <span class="part-chapter-title">洗脑术：如何让人深信不疑</span>
  <span class="part-chapter-desc">让对方觉得这是他的主意——人们更相信自己的结论，而非被告知的答案。</span>
</a>

<a class="part-chapter" href="ch17.html">
  <span class="part-chapter-num">第 17 章</span>
  <span class="part-chapter-title">如何让别人关注你</span>
  <span class="part-chapter-desc">试着从对方的角度看问题——换位思考是理解他人、化解冲突的关键。</span>
</a>

<a class="part-chapter" href="ch18.html">
  <span class="part-chapter-num">第 18 章</span>
  <span class="part-chapter-title">提高你的人格魅力</span>
  <span class="part-chapter-desc">同情对方的处境和想法——同理心是最有力的说服工具和人格魅力来源。</span>
</a>

<a class="part-chapter" href="ch19.html">
  <span class="part-chapter-num">第 19 章</span>
  <span class="part-chapter-title">如何让人乐意服你</span>
  <span class="part-chapter-desc">诉诸更高尚的动机——人们愿意相信自己的行为出于高尚的理由。</span>
</a>

<a class="part-chapter" href="ch20.html">

  <span class="part-chapter-desc">把你的想法戏剧化——用生动的故事和行动打动对方，而非枯燥的说教。</span>
</a>

<a class="part-chapter" href="ch21.html">
