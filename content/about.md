# 关于

这个数字图书馆归档我阅读过的书籍和论文笔记。PDF 通过 MinerU VLM 提取排版，AI 辅助校对翻译，Hugo 构建发布。

右下角的 AI 助手可以从全部馆藏中检索并回答问题——自带 API Key，浏览器直连，不经过第三方服务器。

## 项目

- [Yuunagi Library](https://github.com/uynajgi/yuulibrary) — 本站源码
- [scout skill](https://github.com/uynajgi/yuulibrary/tree/main/.claude/skills/scout) — AI 会话开局顾问
