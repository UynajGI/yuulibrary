    数据可视化#

    一图胜千言。 —Arthur Brisbane (1911)

    本章介绍 matplotlib 和 plotly 包的基本可视化功能。

    虽然有更多的可视化包可用，但 matplotlib 已确立了自己作为基准和——在许多情况下——一种稳健可靠的可视化工具的地位。它既易于用于标准绘图，在处理更复杂的绘图和自定义时也非常灵活。此外，它与 NumPy 和 pandas 及其提供的数据结构紧密集成。

    matplotlib 只能生成位图形式的图表（例如 PNG 或 JPG 格式）。另一方面，现代 Web 技术——例如基于数据驱动文档（Data-Driven Documents, D3.js）标准的技术——允许创建漂亮的交互式且可嵌入的图表（交互式的意思是，例如，可以放大查看特定区域的更多细节）。plotly 是一个可以方便地用 Python 创建此类 D3.js 图表的包。一个名为 Cufflinks 的小型附加库将 plotly 与 pandas DataFrame 对象紧密集成，并允许创建流行的金融图表（如蜡烛图（candlestick chart））。

    本章主要涵盖以下主题：

    "静态二维绘图" on page 168

    本节介绍 matplotlib，并展示一系列典型的二维绘图，从最简单的到一些更高级的——带有双刻度或不同子图的绘图。

    "静态三维绘图" on page 191

    基于 matplotlib，本节展示了一些对某些金融应用有用的三维绘图。

    "交互式二维绘图" on page 195

    本节介绍 plotly 和 Cufflinks 以创建交互式二维绘图。利用 Cufflinks 的 QuantFigure 功能，本节还介绍了技术分析中使用的典型金融图表。

    本章无法全面介绍 Python、matplotlib 或 plotly 的数据可视化功能，但提供了一些示例，展示了这些包在金融领域的基本和重要功能。其他示例也出现在后续章节中。例如，第8章更深入地展示了如何使用 pandas 库可视化金融时间序列数据。

    NumPy 数组与 matplotlib#

    你可以简单地将 NumPy ndarray 对象传递给 matplotlib 函数。matplotlib 能够解释这些数据结构以简化绘图。但注意不要传递过大和/或过于复杂的数组。

    由于大多数 ndarray 方法返回一个 ndarray 对象，也可以传递带有方法（在某些情况下甚至多个方法）的对象。通过调用包含示例数据的 ndarray 对象的 cumsum() 方法，可以得到这些数据的累积和，并得到不同的输出（见图7-3）：

    In [12]: plt.plot(y.cumsum());

    图7-3 带方法调用的 ndarray 对象绘图

    一般来说，默认的绘图样式不能满足报告、出版物等的典型要求。例如，可能需要自定义使用的字体（例如，为了与 LaTeX 字体兼容）、在轴上添加标签，或绘制网格以提高可读性。这就是绘图样式发挥作用的地方。此外，matplotlib 提供了大量函数来自定义绘图样式。有些函数易于使用；其他一些则需要更深入的了解。例如，易于使用的是那些操作坐标轴以及与网格和标签相关的函数（见图7-4）：

    In [13]: plt.plot(y.cumsum())
plt.grid(False)
plt.axis('equal');

    ② 使两个轴等比例缩放。

    图7-4 无网格的绘图

    二维数据集#

    绘制一维数据可以被视为一个特例。通常，数据集会包含多个独立的数据子集。使用 matplotlib 处理此类数据集的规则与处理一维数据相同。然而，在这种情况下可能会出现一些额外的问题。例如，两个数据集的缩放尺度可能差异很大，以至于无法使用相同的 y 轴和/或 x 轴缩放来绘制。另一个问题可能是希望以不同的方式可视化两个不同的数据集，例如一个用线图，另一个用条形图。

    以下代码生成一个二维样本数据集，以形状为 20 × 2 的 NumPy ndarray 对象形式，包含标准正态分布的伪随机数。在该数组上，调用 cumsum() 方法沿轴0（即第一个维度）计算样本数据的累积和：

    In [16]: y = np.random.standard_normal((20, 2)).cumsum(axis=0)

    一般来说，也可以将此类二维数组传递给 plt.plot()。它会自动将包含的数据解释为独立的数据集（沿轴1，即第二个维度）。相应的绘图如图7-7所示：

    In [17]: plt.figure(figsize=(10, 6))
    plt.plot(y, lw=1.5)
    plt.plot(y, 'ro')
    plt.xlabel('index')
    plt.ylabel('value')
    plt.title('A Simple Plot');

    图7-7 两个数据集的绘图

    在这种情况下，额外的注释可能有助于更好地解读图表。可以为每个数据集添加单独的标签，并使其显示在图例中。函数 plt.legend() 接受不同的位置参数。0 代表最佳位置，即尽可能少的数据被图例遮挡。

    图7-8 展示了两个数据集的绘图，这次带有图例。在生成代码中，ndarray 对象不是整体传递的，而是分别访问两个数据子集（y[:, 0] 和 y[:, 1]），这允许为它们附加单独的标签：

    In [18]: plt.figure(figsize=(10, 6))
plt.plot(y[:, 0], lw=1.5, label='1st')
plt.plot(y[:, 1], lw=1.5, label='2nd')

    检查图7-9 可以发现，第一个数据集仍然"视觉可读"，而第二个数据集在新的 y 轴缩放下现在看起来像一条直线。从某种意义上说，关于第二个数据集的信息现在"视觉上丢失了"。有两种基本方法可以通过绘图手段解决这个问题，而不是调整数据（例如通过重新缩放）：

    • 使用两个 y 轴（左/右）
    • 使用两个子图（上/下、左/右）

    以下示例为绘图引入了第二个 y 轴。图7-10 现在有两个不同的 y 轴。左侧 y 轴用于第一个数据集，而右侧 y 轴用于第二个数据集。因此，还有两个图例：

    在 matplotlib 图形对象中放置子图是通过使用特殊的坐标系完成的。plt.subplot() 接受三个整数作为参数：numrows（行数）、numcols（列数）和 fignum（子图编号），可以逗号分隔或不分隔。numrows 指定行数，numcols 指定列数，fignum 指定子图的编号，从1开始，到 numrows * numcols 结束。例如，一个有九个等大小子图的图形将具有 numrows=3、numcols=3 和 fignum=1,2,...,9。右下角的子图将具有以下"坐标"：plt.subplot(3, 3, 9)。

    有时，可能需要或希望选择两种不同的绘图类型来可视化此类数据。使用子图方法，可以自由组合 matplotlib 提供的任意类型的绘图。¹

    图7-12 结合了线/点图和条形图：

    plt.subplot(121)
    plt.plot(y[:, 0], lw=1.5, label='1st')
    plt.plot(y[:, 0], 'ro')
    plt.legend(loc=0)
    plt.xlabel('index')
    plt.ylabel('value')
    plt.title('1st Data Set')
    plt.subplot(122)
    plt.bar(np.arange(len(y)), y[:, 1], width=0.5, color='g', label='2nd')
    plt.legend(loc=0)
    plt.xlabel('index')
    plt.title('2nd Data Set');

    其他绘图样式#

    在二维绘图方面，线图和点图可能是金融中最重要的绘图类型；这是因为许多数据集包含时间序列数据（time series data），这些数据通常通过此类图表可视化。第8章将详细讨论金融时间序列数据。然而，本节继续使用二维随机数数据集，并展示一些替代的、对金融应用有用的可视化方法。

    首先是散点图（scatter plot），其中一个数据集的值作为另一个数据集的 x 值。图7-13 展示了这样的绘图。这种绘图类型可用于，例如，将一个金融时间序列的收益率（returns）与另一个金融时间序列的收益率作图。这个示例使用了一个新的包含更多数据的二维数据集：

    In [24]: y = np.random.standard_normal((1000, 2)) ①
In [25]: plt.figure(figsize=(10, 6))
    plt.plot(y[:, 0], y[:, 1], 'ro') ②
    plt.xlabel('1st')
    plt.ylabel('2nd')
    plt.title('Scatter Plot');

    ① 创建一个更大的随机数数据集。

    ② 通过 plt.plot() 函数生成散点图。

    图7-13 通过 plt.plot() 函数绘制的散点图

    matplotlib 也提供了一个特定的函数来生成散点图。它的工作方式基本相同，但提供了一些额外的功能。图7-14 显示了与图7-13 对应的散点图，这次是使用 plt.scatter() 函数生成的：

    In [26]: plt.figure(figsize=(10, 6))

    plt.ylabel('2nd')

    ① 通过 plt.scatter() 函数生成的散点图。

    图7-14 通过 plt.scatter() 函数绘制的散点图

    plt.scatter() 绘图函数还允许添加第三维度，可以通过不同颜色进行可视化，并使用颜色条（color bar）进行描述。图7-15 显示了一个散点图，其中第三维度通过各个点的不同颜色和作为颜色图例的颜色条来说明。为此，以下代码生成第三个随机数据集，这次由 0 到 10 之间的整数组成：

    c=c,

    plt.ylabel('2nd')
    plt.title('Scatter Plot');

    ③ 将标记定义为粗点。

    另一种常见的绘图类型——直方图（histogram），也常用于金融收益率分析。图7-16 将两个数据集的频率值并排显示在同一个图中：

    In [29]: plt.figure(figsize=(10, 6))

    Iy = func(Ix)

    交互式二维绘图#

    matplotlib 允许创建静态位图对象或 PDF 格式的图表。如今，有许多库可以基于 D3.js 标准创建交互式图表。此类图表支持放大缩小、悬停效果以检查数据等。它们通常也可以轻松嵌入到网页中。

    plotly 是一个流行的平台和绘图库。它专注于数据科学的可视化，在数据科学社区中广泛使用。plotly 的主要优点包括与 Python 生态系统的紧密集成以及易用性——特别是与 pandas DataFrame 对象和封装包 Cufflinks 结合使用时。

    某些功能需要免费账户。获得凭据后，应将其存储在本地以供永久使用。有关详细信息，请参阅"Plotly for Python 入门"指南。

    结论#

    matplotlib 可以被视为 Python 数据可视化的基准和全能选手。它与 NumPy 和 pandas 紧密集成，基本功能可以方便地访问。然而，matplotlib 是一个功能强大的库，其 API 有些复杂。这使得本章无法全面介绍 matplotlib 的所有功能。

    本章介绍了 matplotlib 在二维和三维绘图方面的基本功能，这些功能在许多金融环境中都很有用。其他章节提供了更多关于如何使用该包进行可视化的示例。

    此外，本章还介绍了 plotly 与 Cufflinks 的组合。这种组合使得创建交互式 D3.js 图表变得非常方便，因为通常只需要在 DataFrame 对象上调用一个方法。所有技术细节都在后端处理。此外，Cufflinks 通过 QuantFig 对象提供了一种创建带有流行金融指标的典型金融图表的简便方法。
