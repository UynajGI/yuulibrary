    第5章 · 量子蒙特卡洛入门#

    本章我们将扩展上一章讨论的思想和技术，介绍几种用于量子多体系统蒙特卡洛模拟的基本方法。出于教学目的，我们仅考虑非零温度下的简单量子自旋-1/2模型。我们讨论的一些技术，例如带有局部更新和虚时离散化的路径积分蒙特卡洛方法，只是为介绍关键思想和概念，为后续讨论最先进的连续时间团簇算法和虫洞算法（worm algorithm）做准备。在后续章节中，我们将讨论更高级的量子自旋算法以及费米子和玻色子系统的零温和有限温算法。

    5.1 经典表示#

    目前，所有蒙特卡洛模拟都是在经典计算机上进行的。使用这些计算机，我们只能直接操作经典数，其乘积彼此对易。然而，量子力学处理的是算符，大多数算符的乘积彼此不对易。因此，我们需要为量子系统找到一种经典表示，将量子问题重新表述为经典数的形式。我们从一个简单直观的方法开始。

    我们都熟悉量子力学的矩阵表述，特别是将哈密顿算符表示为一个厄米矩阵。这种表示是量子问题最基本的一种"经典化"。虽然一般来说它们的乘积不对易，但矩阵只是经典数的有序排列，经典计算机可以轻松处理其代数运算。特别是，经典计算机可以对这样的矩阵进行对角化，并给出所有特征值和特征向量。这些信息就"解决"了量子问题。

    在感兴趣的量子多体问题中，相空间与经典情况一样，是许多局部相空间的直积。如果局部空间的大小为n，且我们有N个格点，则相空间随系统物理尺寸呈指数增长，为$n^{N}$。哈密顿矩阵的阶数等于相空间的大小，因此也呈现指数增长。由于计算机内存限制，我们只能获得、事实上也只能期望获得小尺寸格点的所有特征值和特征向量（第10.1节）。然而，小尺寸格点的解通常会受到边界效应的影响，往往不能反映热力学极限下的物理性质。因此，我们必须利用蒙特卡洛方法对非常大的相空间中的重要部分进行采样的能力。

    还有其他两种经典化的方法：高温级数展开（Handscomb, 1962a）和费曼路径积分表述（Suzuki, 1976a,b）。费曼路径积分将d维量子问题转化为(d+1)维经典问题。d维中的原始哈密顿量作用于沿着新维度堆叠的层上，并有一组新的相互作用将各层耦合起来。在量子统计力学中，这个额外维度是虚时轴。

    路径积分算法有离散时间和连续时间两种版本，这些算法通常用于模拟量子多体系统的有限温平衡性质。第一代路径积分算法使用离散化的虚时。为了消除离散化误差（该误差在虚时步长上呈多项式增长），我们必须外推到连续虚时极限。通常，我们可以在算法层面做到这一点，因此现代算法的模拟结果没有离散化误差。1

    高温展开导致一种经典表示，其在形式上类似于路径积分方法的离散虚时表示。一个重要的区别是，高温级数方法的离散化误差是指数小的，在大多数实际情况下可以忽略。

    在第6章中，我们将详细讨论量子高温级数展开和路径积分表述。在作为量子蒙特卡洛算法初步发展的本章中，我们将简要介绍路径积分方法。

    5.2 量子自旋#

    脚注

    1 然而，在处理自旋-1/2系统时，通过外推到零虚时步长来完全消除离散化误差可能很棘手，尤其是对于较大自旋S的情形。在S=1/2的特殊情况下，使用环回算法可以完全消除离散化误差；而对于较大的S，即使使用环回算法也可能存在残留的离散化误差。

    现在我们着手建立第一个量子蒙特卡洛算法。我们从自旋-1/2量子伊辛模型和XY模型的离散虚时路径积分表示开始。通过取连续时间极限，我们最终用环回/团簇算法和虫洞算法取代了传统的离散时间(世界线)蒙特卡洛方法。通过这样做，我们实现了类似于我们在第4章对经典多体问题所展示的算法演变。我们的目标是揭示量子蒙特卡洛算法中使用的几个基本特征和技术。虽然我们的讨论是在特定的简单有限温度实例背景下进行的，但其中一些特征和技术对于零温问题也很有用。

    5.2.1 纵向场伊辛模型#

    经典一维伊辛模型 (1.3) 的量子版本由哈密顿量给出

    $$ H = - \sum_{i = 1}^{N} \left( J^{z} S_{i}^{z} S_{i + 1}^{z} + H^{z} S_{i}^{z} \right) .\tag{5.1} $$

    这里，$S_{i}^{z}$ 是格点 $i$ 处 $S = \frac{1}{2}$ 角动量算符的 z 分量，因此其特征值为 $\frac{1}{2}$ 或 $-\frac{1}{2}$。如果 $H^{z} = 0$，该哈密顿量在每一个格点 i 处执行变换 $S_{i}^{z} \rightarrow - S_{i}^{z}$ 时保持不变。如果 $H^{z} \neq 0$，只要 $H^{z}$ 的符号反转，它在该变换下也保持不变。因此，这个量子模型与经典模型具有相同的对称性。如下文将看到的，这个“量子模型”实际上等效于经典伊辛模型。

    在量子统计力学中，配分函数为

    $$ Z = \mathrm{Tr} \left[ \mathrm{e}^{- \beta H} \right] ,\tag{5.2} $$

    其中 $\beta = 1 / k T$ 表示逆温度。(下文中，我们设玻尔兹曼常数 $k = 1$。) 迹可以用一组完备的正交基态来计算，我们将其记为 $\{| C \rangle \}$，

    $$ \langle C | C^{\prime} \rangle = \delta ( C , C^{\prime} ) , \quad \sum_{C} | C \rangle \langle C | = I . $$

    利用这样的基，(5.2) 变为

    $$ Z = \sum_{C} \langle C | e^{- \beta H} | C \rangle . $$

    如果 $| C \rangle$ 同时也是 $H$ 的本征态，即 $H | C \rangle = E ( C ) | C \rangle$，那么可得 $\langle C | e^{- \beta H} | C \rangle = e^{- \beta E ( C ) }$，因此

    $$ Z = \sum_{c} e^{- \beta E ( C )} .\tag{5.3} $$

    类似地，某个物理可观测量 X 的热期望值变为

    $$ \langle X \rangle = {\frac{1} {Z}} \mathrm{Tr} \Big [ X e^{- \beta H} \Big ] = {\frac{1} {Z}} \sum_{C} \sum_{C^{\prime}} \langle C | X | C^{\prime} \rangle \langle C^{\prime} | e^{- \beta H} | C \rangle .\tag{5.4} $$

    特别地，当 $\{| C \rangle\}$ 是哈密顿量的正交本征态时，我们得到

    $$ \langle X \rangle = {\frac{1} {Z}} \sum_{C} X ( C ) e^{- \beta E ( C )} , $$

    其中 $X ( C ) = \langle C | X | C \rangle$，这个表达式让人想起经典统计力学中的期望值。

    因为知道 H 的本征能量和本征态等同于知道问题的解，所以将 (5.2) 简化为 (5.3) 通常只能是形式上的。然而，对于模型 (5.1)，本征态是

    $$ | C \rangle = | s_{1} \rangle \otimes | s_{2} \rangle \otimes \dots \otimes | s_{N} \rangle = | s_{1} s_{2} \cdot \cdot \cdot s_{N} \rangle ,\tag{5.5} $$

    其中 $s_{i} = \pm \frac{1}{2}$ 且 $| s_{i} \rangle$ 是 $S_{i}^{z}$ 的特征对。那么，特征值 $E ( C )$ 就是经典模型的能量，只是经典自旋 $s_{i}$ 的取值是 $\pm \frac{1}{2}$ 而不是 $\pm 1$，2 而对配分函数 (5.3) 进行采样的问题与经典伊辛问题相同。

    5.2.2 横向场伊辛模型#

    现在我们考虑一个不那么平凡的量子自旋模型——处于横向磁场中的自旋-1/2链，并通过离散化虚时区间显式推导费曼路径积分。横向场伊辛模型的哈密顿量为

    $$ H = - \sum_{i = 1}^{N} \left( J^{z} S_{i}^{z} S_{i + 1}^{z} + H^{x} S_{i}^{x} \right) ,\tag{5.6} $$

    我们将其分解为两个非对易部分

    $$ H = H_{0} + H_{1}\tag{5.7} $$

    定义为

    $$ H_{0} = - J^{z} \sum_{i = 1}^{N} S_{i}^{z} S_{i + 1}^{z} , \quad H_{1} = - H^{x} \sum_{i = 1}^{N} S_{i}^{x} .\tag{5.8} $$

    2 在经典模型(1.3)中，自旋变量是 $\sigma_i$，取其值为 $\pm 1$。

    构建有限温度量子蒙特卡洛算法的难点在于对哈密顿量取指数。由于当前模型存在精确解 (Sachdev, 1999)，原则上我们可以精确地进行取指数运算，因此不需要蒙特卡洛模拟。然而，我们的目的是教学，所以暂时假设我们无法进行该取指数运算。

    我们首先将配分函数 $Z = \mathrm{Tr} \ e^{- \beta \cal H}$ 改写为具有经典自由度的形式。注意到任何算符都与自身对易，我们写出

    $$ e^{- \beta H} = \underbrace{e^{- \Delta \tau H} e^{- \Delta \tau H} \cdot \cdot \cdot e^{- \Delta \tau H}}_{M \mathrm{个因子}} ,\tag{5.9} $$

    其中 $\Delta \tau = \beta / M$，因此

    $$ {\cal Z} = \sum_{C_{1} , C_{2} , \ldots , C_{M}} \langle C_{1} | e^{- \Delta \tau H} | C_{M} \rangle \langle C_{M} | e^{- \Delta \tau H} | C_{M - 1} \rangle \cdot \cdot \cdot \langle C_{2} | e^{- \Delta \tau H} | C_{1} \rangle .\tag{5.10} $$

    该方程是量子力学配分函数的费曼路径积分（Feynman path-integral）表达式。哈密顿量指数的矩阵乘积遍历一个有序的基矢序列。每个基态对应相空间中的一个构型，因此我们可以将 (5.10) 中的求和理解为对虚时（imaginary-time）有序构型序列的求和。构型序列 $C_{1} \ \ C_{2} \ \ \cdot \cdot \cdot C_{M} \ \ C_{1}$ 定义了相空间中的一条路径。因此，配分函数是对所有可能（闭合）路径的求和。在采样构型时，蒙特卡洛算法会选择出这些路径中最重要的一些。

    目前来看，(5.10) 是一个平凡的结果。如果我们知道 $\exp ( - \Delta \tau H )$ 在所选基矢下的矩阵元，就可以轻松计算它。如果知道这些矩阵元，路径积分表述反而是一种退步，因为我们必须计算所有矩阵乘积，这比直接求迹 $\exp ( - \beta H )$ 的矩阵元工作量更大。表述 (5.10) 适用于我们无法精确表达 $\exp ( - \Delta \tau H )$ 矩阵元的情况。通过将 $\exp ( - \beta H )$ 写成多个 $\exp ( - \Delta \tau H )$ 因子的乘积，我们将一个小参数 $\Delta \tau$ 引入了问题，这使得我们可以为 $\exp ( - \Delta \tau H )$ 的每个矩阵元获得一个好的近似。

    对 $H$ 取指数的困难源于 $S_{i}^{x}$ 和 $S_{j}^{z}$ 在同格点 $( i = j )$ 上不具有对易性（noncommutativity）。因此，$H$ 的两个部分不能同时对角化。相应地，(5.5) 中定义的态不是哈密顿量 (5.6) 的本征态；它们仅是 $H_{0}$ 的本征态。然而，在这个基矢下（或者在 $H_{1}$ 的本征基下），$\langle C^{\prime} | e^{- \Delta \tau H_{0}} | C \rangle$ 和 $\langle C^{\prime} | e^{- \Delta \tau H_{1}} | C \rangle$ 都是已知的。不幸的是，如果 $[ H_{0} , H_{1} ] ~ \neq ~ 0$，则 $e^{- \Delta \tau ( H_{0} + H_{1} )} \ \ne$ $e^{- \Delta \tau H_{0}} e^{- \Delta \tau H_{1}}$ 。为了继续，我们需要制定一种策略来处理 $[ A , B ] \neq 0$ 情况下形如 $e^{- \Delta \tau ( A + B )}$ 的指数。

    让我们考虑如果我们采用近似

    $$ e^{- \Delta \tau ( A + B )} \approx e^{- \Delta \tau A} e^{- \Delta \tau B} \approx e^{- \Delta \tau B} e^{- \Delta \tau A} .\tag{5.11} $$

    所引入的误差。对于算符（和矩阵），$\exp ( A ) = I + A + {\textstyle{\frac{1} {2 !}}} A^{2} + {\textstyle{\frac{1} {3 !}}} A^{3} + \cdot \cdot \cdot$ 由其幂级数定义。因此，

    $$ e^{- \Delta \tau ( A + B )} = e^{- \Delta \tau A} e^{- \Delta \tau B} + \mathcal{O} ( ( \Delta \tau )^{2} ) ,\tag{5.12} $$

    这意味着误差量级为 $( \Delta \tau )^{2}$ 。一个更高阶（对称）的近似是

    $$ e^{- \Delta \tau ( A + B )} = e^{- \Delta \tau B / 2} e^{- \Delta \tau A} e^{- \Delta \tau B / 2} + \mathcal{O} ( ( \Delta \tau )^{3} ) .\tag{5.13} $$

    像(5.12)和(5.13)这样的近似称为铃木-Trotter近似（Suzuki-Trotter approximations, Hatano and Suzuki, 2005），或简称为Trotter近似。

    自然地，我们选择一组基，在其中可以方便地求出$H_{0}$和$H_{1}$指数算符的矩阵元。基矢(5.5)使得对$H_{0}$求指数算符变得简单，同时也便于计算$\exp ( - \Delta \tau H_{1} )$的矩阵元。利用低阶Trotter近似(5.12)，我们写出$e^{- \Delta \tau H} \approx e^{- \Delta \tau H_{0}} e^{- \Delta \tau H_{1}}$，然后有

    $$ \langle{\cal C}_{k + 1} | e^{- \Delta \tau H} | C_{k} \rangle \approx e^{- \Delta \tau E_{0} \left( C_{k + 1} \right)} \langle C_{k + 1} | e^{- \Delta \tau H_{1}} | C_{k} \rangle , $$

    其中$E_{0} ( C_{k} )$是零磁场下对应态$| C_{k} \rangle$的经典伊辛能量（同样，自旋$s_{i}$取值为$\pm \frac{1} {2}$而非±1）。

    我们必须为每个时间步$\tau_{k} = k \Delta \tau$计算能量$E_{0} ( C_{k} )$，并通过为自旋本征值附加额外下标来记录每个自旋本征值对应哪个时间步，从而写出

    $$ E_{0} ( C_{k} ) = - J^{z} \sum_{i} s_{i , k} s_{i + 1 , k} . $$

    因此，

    $$ e^{- \Delta \tau \sum_{k} E_{0} ( C_{k} )} = e^{\Delta \tau J^{z} \sum_{i k} s_{i , k} s_{i + 1 , k}} , $$

    这对应于M条解耦的经典伊辛类链的玻尔兹曼权重。

    剩下的任务是确定哈密顿量中横场部分的指数算符的矩阵元。首先，我们注意到

    $$ \left. C | e^{- \Delta \tau H_{1}} | C^{\prime} \right. = \prod_{i = 1}^{N} \left. s_{i} | e^{\Delta \tau H^{x} S_{i}^{x}} | s_{i}^{\prime} \right. .\tag{5.14} $$

    接下来，

    $$ \begin{array} {r l} {e^{\Delta \tau H^{x} S_{i}^{x}}} & {= I + \Delta \tau S_{i}^{x} H^{x} + \frac{1} {2 !} \big ( \Delta \tau H^{x} S_{i}^{x} \big )^{2} + \frac{1} {3 !} \big ( \Delta \tau H^{x} S_{i}^{x} \big )^{3} + \cdots} \\ & {= I + ( \Delta \tau H^{x} ) S_{i}^{x} + \frac{1} {4 \cdot 2 !} ( \Delta \tau H^{x} )^{2} I + \frac{1} {4 \cdot 3 !} ( \Delta \tau H^{x} )^{3} S_{i}^{x} + \cdots ,} \end{array} $$

    这里我们利用了$\begin{array} {r} {( S^{x} )^{2} = \frac{1} {4}} \end{array}$这一事实。累加幂级数得到

    $$ \begin{array} {r} {\langle s | e^{\Delta \tau S^{x} H^{x}} | s^{\prime} \rangle = \langle s | s^{\prime} \rangle \cosh{\frac{1} {2}} \Delta \tau H^{x} + 2 \langle s | S^{x} | s^{\prime} \rangle \sinh{\frac{1} {2}} \Delta \tau H^{x} .} \end{array} $$

    注意到第二项中唯一的非零矩阵元为

    $$ \begin{array} {r} {\langle \downarrow | S^{x} | \uparrow \rangle = \frac{1} {2} , \quad \langle \uparrow | S^{x} | \downarrow \rangle = \frac{1} {2} ,} \end{array} $$

    我们将指数算符的矩阵元重新表示为

    $$ \langle s \vert e^{\Delta \tau S^{x} H^{x}} \vert s^{\prime} \rangle = Q e^{K^{\tau} s s^{\prime}} ,\tag{5.15} $$

    其中$s , s^{\prime} = \pm \frac{1} {2}$，且

    $$ \begin{array} {r} {Q = \sqrt{\sinh \frac{1} {2} \Delta \tau H^{x} \cosh \frac{1} {2} \Delta \tau H^{x}} , \quad K^{\tau} = 2 \ln \coth \frac{1} {2} \Delta \tau H^{x} .} \end{array}\tag{5.16} $$

    最终，

    $$ Z \propto \sum_{C_{1} , C_{2} , \ldots , C_{M}} \exp \left( \Delta \tau J^{z} \sum_{i , k} s_{i , k} s_{i + 1 , k} + K^{\tau} \sum_{i , k} s_{i , k} s_{i , k + 1} \right) .\tag{5.17} $$

    这正是一个零场、各向异性、二维经典伊辛类模型的配分函数（除了因子$\frac{1} {2}$在场中的差异）。如果我们将这个经典模型的逆温度$\beta_{\mathrm{classical}}$设为1，则x方向上的耦合为$K^{x} = \Delta \tau J^{z}$，τ方向上的场依赖交换为$K^{\tau}$：

    $$ H_{\mathrm{classical}} = - K^{x} \sum_{i , k} s_{i , k} s_{i + 1 , k} - K^{\tau} \sum_{i , k} s_{i , k} s_{i , k + 1} .\tag{5.18} $$

    该格点在空间维度上的大小为$L$，在时间维度上的大小为$M = \beta / \Delta \tau$。在$\Delta \tau$的主阶上，空间和时间方向的耦合常数为

    $$ \begin{array} {r} {K^{x} = \Delta \tau J^{z} , \qquad K^{\tau} = - 2 \ln \frac{1} {2} \Delta \tau H^{x} .} \end{array}\tag{5.19} $$

    5.2 量子自旋#

    让我们总结一下刚刚发生的事情。通过将 $\exp ( - \beta H )$ 的矩阵元表示为 $\exp ( - \Delta \tau H )$ 的矩阵乘积，利用 Trotter 近似，并以 $H_{0}$ 的本征态为基，我们得到了一个由伊辛类变量 $s_{i , k}$ 构成的二维格点。表示 $\exp ( - \Delta \tau H_{0} )$ 的矩阵产生了无外场、独立伊辛类链的类玻尔兹曼因子。这些链中的自旋变量仅在 x 方向相互作用。表示 $\exp ( - \Delta \tau H_{1} )$ 的矩阵则在 τ 方向产生了这些链之间的相互作用。由路径积分表示可知，这个 τ 方向实际上是一个虚时方向。将 $\exp ( - \beta H )$ 表示为因子 $\exp ( - \Delta \tau H )$ 的乘积，得到了路径积分的离散虚时表示。根据费曼的说法（Feynman and Hibbs, 1965），将场论转化为量子统计力学是通过对路径积分的动力学因子 $\exp ( - i t H )$ 进行解析延拓，令 $i t \to \tau$，并要求在虚时方向上具有周期性来实现的。迹表达了这种周期性。在将 d 维量子系统映射到 (d+1) 维经典模型时，哈密顿量并非简单地提升到额外维度，而是沿着这个额外维度引入了新的相互作用，以描述虚时动力学。

    在我们基于 Trotter 分解的推导中，策略是将哈密顿量分解成若干部分，使其指数形式能在某个基下（通常是哈密顿量某一部分的本征基）简便地求得。最直接的后果是引入了离散化误差，然而，如下文所述，我们可以通过重新表述算法来完全避免这一误差。另一个更棘手的问题（即使在连续时间表述中依然存在）是可能出现符号问题。由于哈密顿量是厄密的，其指数总是一个正定算子，这意味着它的所有特征值都大于零。算子的这一性质与我们所选择的用以表示其矩阵元的基无关。例如，在纵场伊辛模型中，表示哈密顿量指数的矩阵就明显地体现了这种正定性。它在所选的基下是对角化的，并且因为其对角元是某个实数的指数，所以它们都是正的。

    正定矩阵不同于正矩阵（即所有元素都大于零的矩阵），也不同于非负矩阵（即没有负元素的矩阵）。正定性和非负性取决于我们选择用来表示算子的基。对于蒙特卡洛模拟，我们希望得到正矩阵或非负矩阵。然而，量子虚时传播子仅能保证是正定的。只有在对角化表示中（而我们几乎从不使用这种表示），才能确保表示 $\exp ( - \beta H )$ 的矩阵是非负的。如果我们无法找到一种可行的表示来消除负的矩阵元，就可能出现符号问题。这个问题会影响我们验证结果的能力，并且可能严重到使量子蒙特卡洛算法无法为计算平均值相关的统计误差提供可靠的估计。我们将在后续章节中进一步讨论这个问题。

    对于横场模型，可能出现了符号问题，但先前未加评论地跳过了。在 (5.15) 中，如果 $H^{x}$ 是负的，非对角矩阵元就是负的。如果是这样，某些构型会对配分函数做出负贡献。针对当前情况，我们可以通过要求（正如我们隐含假设的那样）$H^{x} > 0$ 来避免符号问题。对于 $H^{x} < 0$ 的情况，我们可以通过对哈密顿量应用一个幺正变换 $( S_{i}^{x} , S_{i}^{y} , S_{i}^{z} ) \to ( - S_{i}^{x} , S_{i}^{y} , - S_{i}^{z} )$ 来恢复物理，这会导致磁场 $H^{x} \to - H^{x}$ 的反转。这种不变性体现在：对于每个自旋，在虚时方向上由 $S_{i}^{x}$ 引起的自旋翻转次数总是偶数，这使我们能够简单地忽略与单个自旋翻转事件相关的符号。

    5.2.3 连续时间极限#

    将一维量子横向场伊辛模型转化为（1+1）维经典伊辛模型，意味着我们可以将任何用于经典伊辛模型的蒙特卡洛算法作为量子蒙特卡洛算法，用于横向场模型。然而，除非我们取极限 $M \to \infty ( \Delta \tau \to 0 )$ ，否则 (5.17) 并不精确，因此我们的模拟结果会因时间离散化而存在系统误差。我们需要消除这种误差。一个直接的解决方案是使用不同的 $M$ 值进行多次蒙特卡洛模拟，然后外推至 $M \to \infty$ 。尽管此方法行之有效，并在实践中得到应用，但对于大多数问题来说，它已不再是必需的。在本节和第5.2.5节中，我们将第4.4节讨论的团簇算法的基本思想扩展到两个简单的量子自旋模型。然后，在第6章中，我们将讨论其在各式其他系统中的应用。

    现在我们解释如何对横向场伊辛模型 (5.6) 取连续时间极限。前一小节的讨论表明，该模型的离散虚时表示不过是具有各向异性相互作用的二维经典伊辛模型，相互作用强度为 $\begin{array} {r l} {K^{x}} & {{} =} \end{array}$ $\Delta \tau J^{x}$ 和 $K^{\tau} = - 2 \ln ( \Delta \tau H^{x} / 2 )$ ，且“温度”为 $1 / \beta_{\mathrm{classical}} ~ = ~ 1$ 。我们观察到，随着 $\Delta \tau$ 的减小，时间方向的有效耦合常数会增加。这种增加是量子到经典映射的一个普遍特征，它确保了构型（在此情形下，是自旋向上和向下时间间隔的集合）具有明确定义的极限 $\Delta \tau \to 0$ 。我们可以通过注意到离散时间表述中的自旋构型仅仅是“真实”连续费曼路径的“低分辨率”图像来理解这一点。假设在连续极限下，自旋翻转（扭结）之间的平均距离为 $\tau_{\mathrm{kink}}$ 。那么，在离散时间中，我们平均每 $\tau_{\mathrm{kink}} / \Delta \tau$ 个时间步长就有一个扭结。为了在时间方向获得如此长的特征长度（以 $\Delta \tau$ 为单位），该方向的有效耦合必须很大。

    我们已经引入了Swendsen-Wang算法作为模拟伊辛模型的高效算法（第4.4节）。为了将此方法应用于横向场伊辛模型的离散时间表示 (5.17)，我们切换到伊辛变量 $\tilde{s}_{i} = \pm 1$ ，并采用耦合常数 $\begin{array} {r} {\tilde{K}^{x} = \frac{1} {4} K^{x} = \frac{1} {4} \Delta \tau J^{z}} \end{array}$ 和 $\begin{array} {r} {\tilde{K}^{\tau} = \frac{1} {4} K^{\tau} = - \frac{1} {2} \ln ( \frac{1} {2} \Delta \tau H^{x} )} \end{array}$ 。在离散时间团簇模拟中，我们随后

    1. 以概率 $\begin{array} {r} {1 - e^{- 2 \tilde{K}^{x}} \approx \frac{1} {2} \Delta \tau J^{z}} \end{array}$ 放置一个水平键，该键绑定空间方向上的两个相邻平行自旋。

    2. 以概率 $\begin{array} {r} {1 - e^{- 2 \tilde{K}^{\tau}} \approx 1 - \frac{1} {2} \Delta \tau H^{x}} \end{array}$ 连接时间方向上的相邻平行自旋。

    现在讨论如何在算法 11 中生成均匀且随机分布的时间。需注意这些时间及其数量 n 是随机变量，并且这些时间最终要按升序排列，即 $0 < \tau_{1} < \tau_{2} < \cdots < \tau_{n} < \beta$。一种生成具有均匀概率密度 $\lambda$ 的离散时间有序事件的随机过程称为齐次泊松过程（Karlin and Taylor，1975）。毫不意外地，我们将需要用到泊松分布 (2.11)，

    $$ P ( N ( \beta ) = n ) = \frac{( \lambda \beta )^{n}} {n !} e^{- \lambda \beta} ,\tag{5.20} $$

    其中 $N ( \beta )$ 是在区间 $[ 0 , \beta )$ 内发生的事件数量。

    我们可以通过以下方式说明该分布的适用性：如第 2.3.1 节所述，区间 $[ 0 , \beta )$ 内发生的平均事件数为 $\lambda \beta$，因此它们以均匀密度 $\lambda$ 发生。从其函数形式很容易看出，泊松分布具有以下短时行为：

    1. 当区间长度 $\Delta \tau$ 很小时，在时间段 $[ \tau , \tau + \Delta \tau )$ 内恰好发生一个事件的概率为 $\lambda \Delta \tau$，即它与区间长度成正比。
    2. 在此区间内没有事件发生的概率为 $1 - \lambda \Delta \tau$。
    3. 两个事件发生的概率为零，直到 $\mathcal{O} ( ( \Delta \tau )^{2} )$ 量级。

    重要的观察结果是，这种短时行为与我们基于极限情况的概率一致。关于在时间 $\Delta \tau$ 内可发生事件数量的条件与我们隐含假设的相同。因此，为了开发对时间进行采样的算法，我们无需从短时性质逐步构建到有限区间，而可以直接在目标有限时间区间 $[ 0 , \beta )$ 上使用泊松分布。

    泊松分布的函数形式引出了许多有趣的性质，为采样算法提供了严格基础。根据定义可直接得到 $P ( N ( \beta ) = 1 ) = \lambda \beta \exp ( - \lambda \beta )$（指数分布）和 $P ( N ( \beta ) = 0 ) = \exp ( - \lambda \beta )$。以下重要关系式（不加证明给出，Karlin and Taylor，1975）：

    $$ P ( N ( \tau_{1} + \tau_{2} ) - N ( \tau_{2} ) = n ) = P ( N ( \tau_{1} ) = n ) = \frac{( \lambda \tau_{1} )^{n}} {n !} e^{- \lambda \tau_{1}} , \quad \tau_{1} , \tau_{2} > 0 . $$

    这一关系表明，我们可以将一个泊松过程分割成多个区间，并且每个区间内的分布也是一个泊松分布。该结果隐含的事实是，不重叠的区间在统计上是独立的。

    5.2.5 零场 XY 模型#

    5.2.5 环回模拟#

    我们现在将团簇方案（最初在 4.4 节引入，并在 5.2.3 节针对横场伊辛模型重新表述）推广到量子系统配分函数的路径积分表示中。这里我们以 XY 模型为例。5.2.3 节讨论的横场伊辛模型的推广，通过将离散虚时的路径积分映射到 $( d + 1 )$ 维经典伊辛模型，是相当直接的。而本节对 XY 模型的推广则涉及更多内容，即根据相互作用项的性质引入各种类型的图元。本节我们将侧重于“如何操作”而非“为何如此”，并将所提议团簇类型算法的详细推导和证明推迟到第 6 章。

    5.2.6 使用虫洞进行模拟#

    我们不必将整个晶格分解为环，然后通过随机翻转这些环来计算新的世界线构型，而是可以通过创建虫洞并让其移动来更新世界线构型，就像第4.5节中讨论的伊辛模型的虫洞更新一样。这里的基本思想相同：通过引入不连续点来扩展构型空间，相应地推广权重，并构造满足与广义权重的细致平衡的马尔可夫转移矩阵。在一些简单的例子中，包括本节展示的 $S ~ = ~ {\textstyle{\frac{1} {2}}} ~ X Y$ 模型，得到的算法与前一小节讨论的环回算法非常接近，主要区别仅在于虫洞是一个接一个地顺序创建和翻转的，而不像环回算法那样同时构建所有环。因此，该算法也被称为有向环回算法。4 然而，一般而言，虫洞算法和环回算法之间没有直接的对应关系。我们将在第6章详细研究虫洞和有向环回算法。下面的简单例子是一个“预览”，没有严格推导该过程。

    该算法的第一步是定义潜在扭结位置的位置。但不是在特定位置上插入特定的图元，而是插入“顶点”。顶点类似于环回算法中的图元，但顶点上的图元类型是预先不确定的。它更像是一个散射中心，虫洞头部会在该处以非确定性的方式改变其轨迹。

    具体而言，在移除上一周期的顶点后，我们在每个已有的扭结上放置一个顶点，并以密度 ${\frac{1} {4}} J^{x y}$ 在每对相邻格点之间放置额外的顶点。然后，我们将虫洞的头部和尾部放置在时空中的一个随机选择点，并随机选择头部运动的初始方向（时间上的向前或向后）。假设选择的方向是“向前”。在这种情况下，头部在时间上向前移动，直到碰到下一个顶点。5 如果该顶点位于扭结上，则我们以概率 $\frac{1} {2}$ 为该顶点分配一个图元（对角图元或水平图元）。如果顶点不在扭结上，则图元由世界线构型决定：如果顶点位于两个平行自旋之间，则图元为对角图元；否则，为水平图元。无论哪种情况，虫洞头部都会在顶点处发生散射。在对角图元的情况下，散射意味着头部在相邻格点上继续向前移动；而水平图元则意味着头部反转方向，在相邻格点上向后移动。遵循这些规则，虫洞头部最终会回到其原始位置，并与尾部湮灭。此时，世界线构型通过一个环得到了更新，然后我们随机放置另一对虫洞头部和尾部，并重复该过程。这个循环如图5.5所示。

    一个值得注意的微妙之处在于，分配给顶点的图元可能在虫洞穿越时空的过程中发生变化，并且虫洞头部可能以不同方式在同一顶点处散射多次。例如，在图5.5中，只有一个顶点被头部实际访问，并且它被访问了两次。第一次相遇时，头部被对角散射；而第二次，头部则被水平散射。分配给这个顶点的图元不是固定的，而是在每次头部访问该顶点时随机确定的。从这个意义上说，该算法并不完全等同于环回算法的顺序执行。虽然我们在前面的讨论中使用了“向顶点分配图元”这一表述，以强调与环回算法的联系，但“头部在顶点处的散射”是对该过程更恰当的描述。

    5.2.7 遍历性与绕数#

    严格来说，使用局部更新的世界线方法不是遍历的。它守恒绕数。如我们之前所讨论的，环回或虫洞算法用全局采样取代了局部面采样，这种变化使得绕数可以波动，并使不等时关联函数的测量以及巨正则系综中的模拟更易进行。虽然算法的效率可能不强烈依赖于系综之间的差异，但通过使用巨正则系综，我们可以直接访问一些有趣的物理量。

    如果使用空间周期性边界条件（虚时方向总是周期性的），时空晶格位于一个 $( d + 1 )$ 维环面上。绕数是路径在回到其起点之前绕环面的圈数。空间方向上的绕数与玻色系统中的超流密度和自旋系统中的螺旋模量（helicity modulus）密切相关（Pollock 和 Ceperley, 1987）。具体来说，我们通过 $\mu \left( = x , y , \ldots \right)$ 方向的绕数 $W_{\mu}$ 得到超流密度 $\rho_{s}$ 为：

    $$ \rho_{s} = \frac{L_{\mu}^{2}} {\Omega} \langle W_{\mu}^{2} \rangle ,\tag{5.26} $$

    其中 $\Omega \equiv L_{\tau} L_{x} L_{y} \cdot \cdot \cdot$，在玻色子模型中跳跃常数 $t$ 时，$L_{\tau} \equiv 2 t \beta$。在 XY 模型中，$L_{\tau} = J^{x y} {\beta}$ 是时间方向上的无量纲长度。类似地，我们得到无量纲压缩率 $\tilde{\kappa} \equiv 2 t \partial \langle n \rangle / \partial \mu$ 为

    $$ \tilde{\kappa} = \frac{L_{\tau}^{2}} {\Omega} \langle W_{\tau}^{2} \rangle ,\tag{5.27} $$

    其中 $W_{\tau}$ 是时间绕数。这些量在正常态（$\rho_{s}$）和不可压缩态（$\tilde{\kappa}$）中呈指数级小。因此，对于大晶格，在这些态中非零绕数构型的概率极低，所以看似灾难性的局部更新非遍历性实际上并不构成严重问题。即使在超流态或可压缩态中，热力学量通常对正则系综和巨正则系综之间的差异也不敏感。然而，使用巨正则系综通常是有利的，因为 $\rho_{s}$ 和 $\kappa$ 通常是表征量子系统态的相关响应函数。

    5.3 玻色子和费米子#

    量子蒙特卡罗方法不局限于有限温度下量子自旋系统的模拟。玻色子和费米子系统，以及量子多体系统的基态性质，也具有相当大的研究意义。并非所有有限温度算法都能通过环回/团簇或虫算法方便地以连续时间形式表述。即使可以，这些算法的效率也无法保证。大多数量子蒙特卡洛算法，无论是针对量子自旋、玻色子还是费米子，都可能面临符号问题。

    费米子系统的算法具有不同于量子自旋和玻色子算法的特征。然而，有些玻色子系统是用与量子自旋算法非常相似的算法来模拟的。具有不同量子统计的一维系统可以相互映射。在某些情况下，这种等价性可能允许直接采用现有算法，而非重新创建算法。在以下小节中，我们将简要讨论这些主题。

    5.3.1 玻色子#

    如今，环回和虫算法不仅是模拟一维，也是模拟二维和三维量子自旋模型的标准方法。它们与这些模型典型的短程交换相互作用很好地契合。环回和虫算法同样适用于硬核玻色子模型的模拟。7 针对这种量子统计的 world-line 算法的推导过程与量子自旋情形类似，基态是玻色子数算符的本征态。

    在一维情况下存在一条捷径。我们无需重新推导玻色子 world-line 算法的转移概率，而是使用松原-松田变换（Matsubara-Matsuda 变换，1956）将硬核玻色子模型映射到自旋- 1模型，然后采用自旋算法。

    具体来说，松原-松田变换

    $$ S_{i}^{+} a_{i}^{\dagger} , S_{i}^{-} a_{i} , S_{i}^{z} + \textstyle{\frac{1} {2}} a_{i}^{\dagger} a_{i} = n_{i} $$

    将硬核玻色子映射到自旋 $-\frac{1}{2}$，反之亦然。例如，如果我们将此变换应用于纵向场 XXZ 模型

    $$ H = - \sum_{i = 1}^{N} \left[ J^{x y} \left( S_{i}^{x} S_{i + 1}^{x} + S_{i}^{y} S_{i + 1}^{y} \right) + J^{z} S_{i}^{z} S_{i + 1}^{z} + H^{z} S_{i}^{z} \right] $$

    我们得到

    $$ H = - t \sum_{i = 1}^{N} \left( a_{i}^{\dagger} a_{i + 1} + a_{i + 1}^{\dagger} a_{i} \right) - \mu \sum_{i = 1}^{N} n_{i} + V \sum_{i = 1}^{N} n_{i} n_{i + 1} + \mathrm{c o n s t .} , $$

    其中 ${t} = {\textstyle{\frac{1} {2}}} J^{x y} , \ V = J^{z}$，且 $\mu = H^{z} - J^{z}$。由此得到的模型称为扩展硬核玻色-哈伯德模型（extended hard-core Bose-Hubbard model）。“扩展”一词源于模型中存在最近邻耦合，而“硬核”则表示在位排斥无穷大。在第6.6节中，我们将讨论标准的玻色-哈伯德模型，其中仅包含有限大小的在位排斥。

    由此可见，这种映射非常直接。毫不意外，用于消除量子自旋算法中潜在符号问题的技术在此同样适用。对于这两类量子统计，哈密顿量矩阵元的符号通常由模型参数的符号决定。这使得自旋和玻色子算法的构造与费米子算法所面临的挑战有所不同。

    5.3.2 费米子#

    费米子世界线算法的推导过程与量子自旋类似。在对哈密顿量各部分进行指数化时，为了体现不同的量子统计，会出现细微差别。基态通常是费米子数算符的本征态。问题不在于无法构造算法，而在于很难构造出能高效工作的算法。

    为了更好地理解困难的根源，我们考虑一个特例，其中约旦-维格纳变换（Jordan-Wigner transformation）（Jordan and Wigner, 1928）将一维费米子模型映射到量子自旋-$\frac{1}{2}$模型。我们应用该变换，定义如下：

    $$ c_{i} = K_{i} S_{i}^{+} , \quad c_{i}^{\dagger} = S_{i}^{-} K_{i}^{\dagger} , \quad K_{i} \equiv e^{i \pi \sum_{j = 1}^{i - 1} S_{j}^{+} S_{j}^{-}} , $$

    将其应用于一维无自旋无相互作用费米子模型

    $$ H = - t \sum_{i = 1}^{N} \Big ( c_{i}^{\dagger} c_{i + 1} + c_{i + 1}^{\dagger} c_{i} \Big ) . $$

    算符 $K_{i}$ 称为纽结算符（kink operator），它统计了位置 $i$ 左侧自旋从下到上翻转的次数。这自动计入了由两个粒子交换产生的费米子符号。纽结算符是酉且自共轭的，

    $$ K_{i}^{\dagger} = K_{i} , \quad K_{i}^{2} = I , $$

    并且与 $j \geq i$ 时的 $S_{j}^{\pm}$ 对易。利用

    $$ c_{i}^{\dagger} c_{i + 1} = S_{i}^{+} S_{i + 1}^{-} , \quad c_{i + 1}^{\dagger} c_{i} = S_{i + 1}^{+} S_{i}^{-} , \quad c_{i}^{\dagger} c_{i} = S_{i}^{+} S_{i}^{-} ,\tag{5.28} $$

    我们得到

    $$ H = - 2 t \sum_{i = 1}^{N} \big ( S_{i}^{x} S_{i + 1}^{x} + S_{i}^{y} S_{i + 1}^{y} \big ) , $$

    即 $J^{x y} = 2 t$ 的 XY 模型。因此，我们可以利用针对该自旋模型讨论的算法来模拟无自旋费米子。8 当我们在问题中加入费米子自旋时，就必须为自旋向上和自旋向下的粒子分别引入世界线。然后，模拟还需要额外的蒙特卡罗步骤，根据自旋向上和自旋向下之间的相互作用来接受或拒绝世界线的移动（Kawashima et al., 1994）。

    让我们强调一下自旋升降算符与产生湮灭算符之间映射中的纽结算符。在松原-松田变换中并没有类似的算符。回顾一下，我们的基态是直接乘积态，以特定顺序列出格点。只要格点按顺序编号，约旦-维格纳变换就适用于任意维度。之所以难以将上述一维解推广到更高维度，是因为不可能以这样一种方式对所有格点进行编号，使得所有相互作用的格点对在列表中彼此相邻。(如果可能，则意味着该晶格是一维的。) 更具体地说，(5.28) 的一般形式是

    $$ c_{i}^{\dagger} c_{j} = e^{i \pi \sum_{k = i + 1}^{j - 1} S_{k}^{+} S_{k}^{-}} S_{i}^{+} S_{j}^{-} , $$

    8 但需注意，边界条件可能会改变：周期边界的费米子模型会映射到反铁磁的自旋模型，反之亦然。

    带着一个丑陋的相位因子，当 $| j - i | > 1$ 时该因子不为零，并使问题在一般情况下不可解。这个符号问题实际上限制了世界线方法只能用于一维系统。

    简言之，模拟费米子问题的困难源于费米子间排列产生的符号，该符号由相位因子 $e^{i \pi \sum_{k = i + 1}^{j - 1} n_{k}} = \pm 1$ 表示。如果我们暂时忽略这个相位因子，费米子哈密顿量与硬核玻色子或 $\begin{array} {r} {S = {\frac{1} {2}}} \end{array}$ 自旋模型完全相同。因此，我们可以将针对自旋或玻色子的环回/团簇（或任何其他）算法也应用于费米子。然而，要以一种不会导致不同蒙特卡洛构型之间严重抵消的方式来考虑相位因子

    $$ S_{i}^{+} = c_{i} K_{i} , \quad S_{i}^{-} = c_{i}^{\dagger} K_{i} , $$

    是非常困难的，这被称为"费米子符号问题"或"负符号问题"。迄今为止，这个问题没有通用的解决方案。

    5.4 负符号问题#

    负符号问题是量子模型数值模拟中最大的瓶颈。对于我们这里考虑的算法，它源于哈密顿量的负矩阵元。当局部哈密顿量 H 的某些非对角矩阵元为负时，对于某些构型 C，权重 $w ( C )$ 通常为负。在这种情况下，我们执行蒙特卡洛模拟，其目标分布是 $| w ( C ) |$，而不是 $w ( C )$。然后，我们利用恒等式计算可观测量 Q 的期望值

    $$ \begin{array} {r l} & {\langle Q \rangle = \frac{\sum_{C} \left| w ( C ) \right| \mathrm{sign} ( C ) Q ( C )} {\sum_{C} \left| w ( C ) \right| \mathrm{sign} ( C )}} \\ & {\quad =frac{\sum_{C} \left| w ( C ) \right| \mathrm{sign} ( C ) Q ( C ) / \sum_{C} \left| w ( C ) \right|} {\sum_{C} \left| w ( C ) \right| \mathrm{sign} ( C ) / \sum_{C} \left| w ( C ) \right|}} \\ & {\quad =frac{\langle \mathrm{sign} ( C ) Q ( C ) \rangle_{\mathrm{MC}}} {\langle \mathrm{sign} ( C ) \rangle_{\mathrm{MC}}} ,} \end{array}\tag{5.29} $$

    其中 sign $( C ) = \pm 1$ 表示 $w ( C )$ 的符号，$\langle \cdots \rangle_{\mathrm{MC}}$ 是以 $| w ( C ) |$ 为权重的蒙特卡洛平均。

    配分函数的负贡献 $Z_{-}$ 抵消了正贡献 $Z_{+}$ 的一部分，而总和 ${Z = Z_{+} - Z_{-}}$ 必须始终为正。事实上，在许多情况下，负贡献几乎完全抵消了正贡献。我们可以通过考虑一个虚构的哈密顿量 $H^{\prime}$ 来理解这一点（Loh 等人，1990；Hatano 和 Suzuki，1992），该哈密顿量的矩阵元是原始哈密顿量 H 相应矩阵元的绝对值：$\langle C^{\prime} | H^{\prime} | C \rangle \equiv$ $| \langle C^{\prime} | H | C \rangle |$。当两个哈密顿量之间的差异是广延量时，每个格点的自由能差 $\Delta f \equiv ( F^{\prime} - F ) / N$ 为 $\mathcal{O} ( 1 )$。由于 $\beta{\cal F} = \ln Z$ 且 $\beta{\cal F}^{\prime} = \ln Z^{\prime}$，其中 $Z^{\prime}$ 是虚构系统的配分函数，因此（5.29）中的分母可以表示为

    $$ \frac{Z_{+} - Z_{-}} {Z_{+} + Z_{-}} = \frac{Z} {Z^{\prime}} = e^{- \beta N \Delta f} . $$

    从这个表达式可以清楚地看出，在低温和系统尺寸较大的情况下，$Z$ 的正部和负部几乎完全抵消，使得从（5.29）计算期望值 $\langle Q \rangle$ 实际上变得不可能，因为分子和分母中的统计误差都将远大于平均值。

    5.5 动力学#

    平衡量子蒙特卡洛结果拥有物理上有意义的动力学信息，而经典蒙特卡洛结果则没有。能够获取真实的动力学信息是对我们在模拟中必须处理额外虚时维度的回报。要获取这些信息，必须进行特定类型的测量。根据定义，量子力学算符 X 的期望值为

    $$ \langle X \rangle = {\frac{\operatorname{Tr} e^{- \beta H} X} {\operatorname{Tr} e^{- \beta H}}} . $$

    虚时 $\tau$ 的算符为 $X ( \tau ) = e^{\tau H} X e^{- \tau H}$，我们有

    $$ \langle X ( \tau ) \rangle = \frac{\mathrm{Tr} e^{- ( \beta - \tau ) H} X e^{- \tau H}} {\mathrm{Tr} e^{- \beta H}} . $$

    由于平衡态的时间平移不变性，有 $\begin{array} {r l} {\langle X \rangle} & {{} =} \end{array}$ $\begin{array} {r} {{\frac{1} {\beta}} \int_{0}^{\beta} d \tau \langle X ( \tau ) \rangle} \end{array}$ 。这类期望值称为等时期望值（equal-time expectation values）。

    动力学探针涉及两个（或多个）不同时刻算符的乘积

    $$ \left. A ( \tau ) B ( 0 ) \right. = \frac{\mathrm{Tr} e^{- ( \beta - \tau ) H} A e^{- \tau H} B} {\mathrm{Tr} e^{- \beta H}} . $$

    这些期望值同样具有时间平移不变性：$\left. A ( \tau ) B ( 0 ) \right. =$ $\langle A ( \tau + \tau^{\prime} ) B ( \tau^{\prime} ) \rangle$

    非等时测量的重要性在于它们与谱密度（spectral density）的关系，

    $$ \langle A ( \tau ) B ( 0 ) \rangle = \int_{- \infty}^{\infty} d \omega{\frac{e^{- \tau \omega} D ( \omega )} {1 \pm e^{- \beta \omega}}} , $$

    其中 $\pm$ 取决于 $A$ 和 $B$ 是反对易还是对易。谱密度 $D ( \omega )$ 将虚时动力学与真实动力学信息联系起来。更具体地说，谱密度包含了关于系统元激发（elementary excitations）的可测量信息。例如，测量 $\langle S_{i}^{z} S_{j}^{z} \rangle$ 的空间傅里叶变换的虚时依赖性，并对积分方程求逆，得到 $D ( k , \omega )$ （Kawashima et al., 1996），这通常通过非弹性中子散射实验测量。该函数中的峰是 $z$ 分量自旋激发的标志，峰的位置作为 $k$ 的函数给出激发的色散关系 $\omega ( k )$ 。即使没有蒙特卡洛估计伴随的统计误差，对积分方程求逆也是一项困难的任务。我们将在第12章讨论这种求逆技术。更易于处理的是静态磁化率（static susceptibilities），通过对虚时积分得到。例如，静态纵向和横向自旋磁化率为

    $$ \begin{array} {l} {{\displaystyle \chi_{i j}^{\parallel} = \int_{0}^{\beta} d \tau \left. S_{i}^{z} ( \tau ) S_{j}^{z} ( 0 ) \right.} ,} \\ {{\displaystyle \chi_{i j}^{\perp} = \int_{0}^{\beta} d \tau \left. S_{i}^{x} ( \tau ) S_{j}^{y} ( 0 ) + S_{i}^{y} ( \tau ) S_{j}^{x} ( 0 ) \right.} .} \end{array} $$

    对于对角关联函数的测量，我们使用恒等式

    $$ \langle S_{i}^{z} ( \tau ) S_{j}^{z} ( 0 ) \rangle \equiv \langle s_{i} ( \tau ) s_{j} ( 0 ) \rangle_{\mathrm{MC}} , $$

    其中 $\langle \cdots \rangle_{\mathrm{MC}}$ 表示对蒙特卡洛模拟生成的环状位形（world-line configurations）的简单平均。虽然非对角关联函数的测量不像这样直接，但当我们使用环/团簇更新（loop/cluster update）或蠕虫更新（worm update）时，公式几乎同样简单。例如，我们可以使用以下方法估计 $\operatorname{spin} {-} \frac{1} {2} X Y$ 模型中 $S^{x}$ 算符之间的关联函数

    $$ \langle S_{i}^{x} ( \tau ) S_{j}^{x} ( 0 ) \rangle \equiv \frac{1} {4} \langle \theta ( ( i , \tau ) , ( j , 0 ) ) \rangle_{\mathrm{{M C}}} , $$

    其中 $\theta ( X , Y )$ 在时空点 $X$ 和 $Y$ 位于同一环上时为 1，否则为 0。通过对 $X$ 和 $Y$ 积分，我们得到均匀、静态的横向磁化率，

    $$ \begin{array} {l} {{\displaystyle \chi^{\perp} = \frac{1} {N \beta} \sum_{i , j} \int_{0}^{\beta} d \tau_{1} d \tau_{2} \langle T S_{i}^{x} ( \tau_{2} ) S_{i}^{x} ( \tau_{1} ) \rangle}} \\ {{\displaystyle \quad = \frac{1} {4} \frac{1} {N \beta} \sum_{i , j} \int_{0}^{\beta} d \tau_{1} d \tau_{2} \langle \theta ( ( i , \tau_{2} ) , ( j , \tau_{1} ) ) \rangle_{\mathrm{MC}}}} \\ {{\displaystyle \quad = \frac{1} {4} \frac{1} {N \beta} \left. \sum_{i} \Lambda_{l}^{2} \right._{\mathrm{MC}} ,}} \end{array} $$

    其中 $\tau$ 表示编时乘积，$\Lambda_{l}$ 是由环索引 l 指定的环的长度。类似地，在蠕虫算法中，我们通过简单测量头部相对于尾部访问特定位置的频率来估计关联函数。对角关联函数的简单高效测量是环/团簇算法和蠕虫算法的重要优势。我们将在第6章更详细地讨论这些要点。
