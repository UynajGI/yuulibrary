    第11章 · 费米子基态方法#

    在本章中，我们讨论用于计算相互作用电子系统基态性质的固定节点（fixed-node）方法和约束路径（constrained-path）蒙特卡洛方法。这两种方法可以说是目前进行此类计算可用的最强大的两种方法，但它们是近似的。通过牺牲精确性，它们避免了通常伴随相互作用电子系统模拟的蒙特卡洛误差随系统尺寸呈指数增长的问题。这种指数增长被称为费米子符号问题（Fermion sign problem）。在对符号问题进行一些一般性评论之后，我们概述这两种方法，指出它们的异同点以及优缺点。我们还将讨论约束相位方法（constrainedphase method），它是约束路径方法的扩展，用于控制在基态波函数不能为实数时产生的相位问题。

    11.1 11.1 符号问题#

    "符号问题"指的是蒙特卡洛误差随系统尺寸增大或温度降低而呈指数增长（例如，Loh et al., 1990, 2005; Gubernatis and Zhang, 1994），这通常伴随着极限分布并非处处为正的马尔可夫链模拟。这种情况通常出现在费米子系统及受挫量子自旋系统的模拟中。它在费米子系统蒙特卡洛模拟中似乎如此根深蒂固，以至于对许多人来说，"符号问题"这个短语几乎等同于"费米子符号问题"。

    对于符号问题成因的解释各不相同，并且仍在争论中。在本章中，我们选择总结两种似乎将连续介质和格点上基态费米子模拟中的成因联系起来的解释。当然，符号问题并不仅限于基态计算，甚至也不仅限于费米子模拟。虽然我们在§11.2和§11.3中讨论的成因集中于类似扩散算子的低位激发态，但人们也提出了几种拓扑图景（Muramatsu et al., 1992; Samson, 1993; Gubernatis and Zhang, 1994; Samson, 1995）。其中一些讨论是在零温和有限温行列式方法（determinant methods）的背景下进行的（Muramatsu et al., 1992; Gubernatis and Zhang, 1994）。另一些则更偏解析地从费曼路径积分（Feynman path-integral）的角度进行（Samson, 1993, 1995）。还有一些是针对非费米子统计的粒子。本章中对符号问题的阐述适用于本章讨论的蒙特卡洛方法。

    事先判断给定的费米子或量子自旋蒙特卡洛模拟是否存在符号问题通常很困难。然而，经验表明，某些类别的哈密顿量（Hamiltonians）的模拟很可能存在这样的问题。例如，具有反铁磁交换相互作用的量子自旋系统就是一个很好的例子，除非晶格是二分图（bipartite）。在后一种情况下，马歇尔变换（Marshall transformation）（Marshall, 1955）可能将具有最近邻交换相互作用的哈密顿量的模拟转换为一个无符号问题，其基态波函数必为正（§5.2.4）。对于"受挫"晶格（如三角形或Kagome晶格）上的相同哈密顿量，该变换不再能消除符号问题。对于具有相反符号的最近邻和次近邻交换相互作用的自旋模型的模拟，在任何晶格结构下通常都存在符号问题。

    对于格点上的相互作用电子，无符号问题的情况甚至更为罕见。存在证明表明，特殊的哈密顿量没有符号问题（例如，Wu and Zhang, 2005）。不幸的是，这些哈密顿量是非物理的。在某些电子填充下具有粒子-空穴对称性（particle-hole symmetry）的哈密顿量通常没有符号问题（Hirsch, 1985），大多数一维哈密顿量也是如此，如果跳跃项是最近邻的。在后一种情况下，沿直线的受限运动允许对电子进行明确的重新标记，从而消除了电子交换和符号问题的根源。某些类型的无自旋费米子模型可以通过切换到马约拉纳费米子表示（Majorana Fermion representation）后使用辅助场方法进行模拟（Li et al., 2014）。这里的思想是，用马约拉纳费米子表述会使行列式的数量加倍，这以类似于具有粒子-空穴对称性的自旋费米子模型的方式消除了符号问题。

    如果符号问题不能通过哈密顿量的选择或其替代基下的表达式来克服，那么处理符号问题的重任就落在了算法上。在连续介质中，有几种格林函数方法不存在费米子符号问题（Zhang and Kalos, 1991；Kalos and Pederiva, 2000），但遗憾的是，这些方法似乎并不适合通用高效使用。固定节点方法和约束路径方法填补了这一空白，它们是经过充分基准测试的近似方法，且不受符号问题的困扰。接受这种近似使我们能够对中等规模的系统进行模拟，而不是在精确模拟小系统时苦苦挣扎。

    原则上，从混合符号分布中采样的策略是直接了当的。在分布为正的区域，我们正常采样。在分布为负的区域，我们从分布的绝对值正常采样，并改变可观测量的符号。期望值变为符号加权平均。更明确地说，如果 $p ( x )$ 是目标分布，其符号函数为 $s ( x ) = p ( x ) / | p ( x ) |$，那么符号加权平均为

    $$ \langle A \rangle = \frac{\int d x s ( x ) A ( x ) \left| p ( x ) \right|} {\int d x s ( x ) \left| p ( x ) \right|} = \frac{\langle s A \rangle_{| p |}} {\langle s \rangle_{| p |}} . $$

    角括号的下标表示该平均是关于分布$| p ( x ) |$进行的。

    对于许多马尔可夫链蒙特卡洛模拟，尤其是蒙特卡洛幂方法，极限分布是未知的，且通常非非负。特别成问题的情况是，当我们不知道划分符号不同的相空间区域的节点曲面（nodal surface）的位置时，这限制了符号加权平均（sign-weighted average）的效用。缺乏这一知识，构型的权重便具有混合符号（mixed sign），最终约一半的构型群体具有正符号，另一半具有负符号。这种几乎平衡的混合导致$\langle s A \rangle \approx 0$和$\langle s \rangle \approx 0$，使得符号加权平均不确定（误差棒远大于可观测量的值）。各种启发式论证（第5.4节）以及无数模拟的经验表明，随着格点数或逆温增加，这些符号加权平均以指数速度趋近于零。相关的相对误差也以指数速度增长，导致算法迅速变得不可用。费米子符号问题的讽刺之处在于，原则上我们并不需要符号加权平均。我们只需要在对应一种符号的区域中求解波函数。从这个解出发，我们可以使用反对称化过程显式地生成解的其余部分。问题在于我们不知道这个区域的边界。

    如前一章所述，蒙特卡洛幂方法通过格林函数$\langle C^{\prime} | G | C \rangle = G ( C^{\prime} , C ) \equiv w ( C^{\prime} , C ) P ( C^{\prime} | C )$定义生成马尔可夫链的转移概率$P ( C^{\prime} | C )$。虽然我们总可以定义非负的转移概率，但我们不能同时总是定义非负的权重$w ( C^{\prime} , C )$。同样如前一章所述，这些幂方法通常通过导引函数（guiding function）$\psi_{G} ( C ) \equiv \langle C | \psi_{G} \rangle$进行重要性采样，因此实际使用的权重和转移概率是相对于$\bar{G} ( C^{\prime} , C ) \equiv \psi_{G} ( C^{\prime} ) G ( C^{\prime} , C ) / \psi_{G} ( C )$而定义的。对于受挫量子自旋系统，典型情况是导引函数为正，而格林函数的矩阵元素具有混合符号。对于费米子系统，情况通常相反，$G ( C^{\prime} , C )$非负，但$C^{\prime}$和$C$处的导引函数符号相反。在从$| C \rangle$到$| C^{\prime} \rangle$的转移（由正的$G ( C^{\prime} , C )$定义）上，重要性采样的格林函数$\bar{G} ( C^{\prime} , C )$可能改变符号，因为该转移从$\psi_{G} ( C )$到$\psi_{G} ( C^{\prime} )$产生了奇数次的电子交换。

    固定节点法（fixed-node method）和约束路径法（constrained-path method）的复杂度通常与系统大小的三次方成正比。它们通过阻止游走者越过采样分布的近似节点曲面，消除了指数级误差缩放。我们首先讨论固定节点法的基础，然后讨论约束路径法的基础。这些方法的细节对于期望值的估计具有重要影响。我们针对混合估计子同时讨论这两种方法，并重点突出固定节点法中的前向行走估计子与约束路径法中常替代混合估计子的反向传播估计子之间的差异。在讨论混合估计子之后，我们将讨论约束相位法（constrained-phase method），它是约束路径法的一个扩展，用于控制在基态波函数不能为实数时出现的相位问题。

    11.2 11.2 固定节点法#

    固定节点法（fixed-node method）几十年前就被用于连续介质中费米子的扩散蒙特卡洛模拟（第10.4.2节）（Anderson, 1975, 1976; Moskowitz et al., 1982; Reynolds et al., 1982）。与之相关的一个有趣物理图景，描述了当存在符号问题时波函数会发生什么。在这个幂方法中，蒙特卡洛算法演化一个由$N$个标记电子位置组成的构型$C =$ $( r_{1} , r_{2} , \ldots , r_{N} )$通过相空间。当使用重要性采样时（第10.4.3节和第11.1节），蒙特卡洛采样经验性地生成一个分布函数$f ( C , \tau ) = \psi ( C , \tau ) \psi_{G} ( C )$，其中$\psi_{G} ( C )$是代表基态的引导（重要性）函数。1 可以证明（Hammond et al., 1994），这个分布函数求解了一个偏微分方程，该方程定义了电子分布函数的确定性虚时动力学

    $$ \frac{\partial f} {\partial \tau} = {D \nabla^{2} f} - {D \nabla \cdot \left[ f F_{Q} \right]} + {[ E_{T} - E_{L} ] f} , $$

    其中“扩散常数”为$D = \hbar^{2} / 2 m$，局部能量为（第9.3.1节）

    $$ E_{L} = \langle{C | H | \psi} \rangle / \langle{C | \psi} \rangle \equiv H \psi ( C ) / \psi ( C ) , $$

    矢量场$F_{Q}$由$F_{Q} = \nabla \ln | \psi | ^{2}$给出。这个被称为量子力的场，是一个漂移项，出现在动力学进行重要性采样时，即权重$f ( C , \tau ) = \psi ( C , \tau ) \psi_{G} ( C )$的情况下。

    这个微分方程的重要性（注意我们并不旨在数值求解它）在于，众所周知，扩散型微分方程的空间极值本征函数是无节点的。无节点解最大化驱动扩散的拉普拉斯算子$\nabla^{2}$的效应。在连续介质扩散蒙特卡洛模拟中，我们将无节点解称为玻色子（Bosonic）解。第一激发态，如同所有其他本征态一样，必须有节点。2 这个第一激发态正是我们想要的费米子态。对于大系统，玻色子本征值和费米子本征值之间的能隙变得指数级小，最终淹没在模拟的噪声中。因此，在连续介质中，费米子解坍缩到玻色子解导致了符号问题。

    连续介质中的固定节点法通过在费米子节面处施加一个无限高的势垒来阻止玻色子坍缩，阻止任何随机游走者穿越它，从而产生混合符号的游走者。通过添加这些势能，原始哈密顿量变成某个$H_{\mathrm{eff}}$。该方法精确地求解了该节点内部区域内有效哈密顿量的基态$( E_{\mathrm{eff}} , \psi_{\mathrm{eff}} )$的薛定谔方程。由于节面的位置未知，该方法通过使用费米子基态的良好近似所对应的节面来逼近精确节面。如果该方法使用了精确节面，它将产生精确答案。用于重要性采样的费米子引导态$| \psi_{G} \rangle$希望能非常接近精确基态，并且通过同时固定节面而起到双重作用。蒙特卡洛解从该引导态继承其费米子特性。

    这种受到连续介质模拟支持的符号问题图景具有启发性，但在格点上并不如在连续介质中那么适用。在格点上，图景发生了变化，因为与类似扩散的运动方程的联系不那么清晰。毫不奇怪，固定节点法的实现方式在离散空间中定义节面以及当随机游走者试图穿越节面时有效哈密顿量作用的细节方面都有所不同。

    在格点固定节点法中（Bemmel et al., 1994; ten Haaf et al., 1995），我们使用了称为线性格林函数蒙特卡洛（Green's function Monte Carlo）方法（第10.4.1节）。有时也使用扩散蒙特卡洛方法（第10.4.2节）。

    回想一下，线性方法没有Trotter近似，并迭代

    $$ \left| \psi^{\prime} \right. = \left[ I - \tau \left( H - E_{T} \right) \right] \left| \psi \right. . $$

    在这一迭代过程中，我们将基态 $\left| {\psi_{0}} \right.$ 表示为一个完备正交归一化标记电子基中的形式，即

    $$ | \psi_{0} \rangle = \sum_{C} | C \rangle \langle C | \psi_{0} \rangle = \sum_{C} \psi_{0} ( C ) | C \rangle , \quad \psi_{0} ( C ) > 0 ,\tag{11.1} $$

    然后将该迭代过程投影到标记构型基上，我们得到

    $$ \psi^{\prime} ( C^{\prime} ) = \sum_{C} \langle C^{\prime} | I - \tau ( H - E_{T} ) | C \rangle \psi ( C ) = \sum_{C} G ( C^{\prime} , C ) \psi ( C ) . $$

    当迭代达到平稳状态时，即满足条件 $\psi^{\prime} = \psi_{0}$ 时，蒙特卡洛模拟从以下分布中进行采样

    $$ P ( C ) = \frac{\psi_{0} ( C )} {\sum_{C} \psi_{0} ( C )} . $$

    采用重要性采样（第10.4.3节）后，

    $$ \bar{\psi}^{\prime} ( C^{\prime} ) = \sum_{C} \bar{G} ( C^{\prime} , C ) \bar{\psi} ( C ) , $$

    其中 $\bar{\psi} ( C ) = \psi_{G} ( C ) \psi ( C )$ 。因此，重要性采样改变了我们从中抽样的转移概率。通常我们取 $\psi_{G} ( C ) = \psi_{T} ( C )$ 。仅在 $\psi_{T} ( C )$ 为正的区域进行采样可以解决符号问题。使 $\bar{G}$ 的矩阵元素为正，便得到了马尔可夫链的转移概率。在连续系统中，$G$ 通常很容易变为正的。3

    对于晶格问题，节面无法精确定位。它不像在连续系统中那样，可以通过逐步缩小空间步长来避免穿越该表面。在晶格上，符号问题转变为与符号 $\hbar i p s$ 相关的问题，因为电子在格点之间跳跃。符号翻转的原因有多种。与连续系统类似，引导态可能会改变符号。与连续系统中动能始终为负不同，许多晶格哈密顿量的跳跃部分具有混合符号的振幅。更一般地说，哈密顿量几乎总是连接不同节面区域的晶格位点。

    我们还回想起，在线性格林函数蒙特卡洛方法（第10.4节）中，无符号问题的条件是：对于所有 $C$ 和 $C^{\prime}$，满足 $\langle C | H - \sigma I | C^{\prime} \rangle \leq 0$。4 如果满足此条件，则仅当行走者从一个节面区域移动到另一个节面区域时才会发生符号变化。更一般地，当构型变化导致 $\langle C^{\prime} | H | C \rangle \psi_{T} ( C ) \psi_{T} ( C^{\prime} ) > 0$ 时，晶格固定节点法（Bemmel et al., 1994; ten Haaf et al., 1995）通过用原始哈密顿量截断非对角项并向对角项添加项而创建的有效哈密顿量来替换原始哈密顿量，从而防止符号翻转。5 该有效哈密顿量的非对角元素为

    $$ \langle C^{\prime} | H_{\mathrm{eff}} | C \rangle = \left\{\begin{array} {r l} {\langle C^{\prime} | H | C \rangle} & {\mathrm{if} \ \langle C^{\prime} | H | C \rangle \psi_{T} ( C^{\prime} ) \psi_{T} ( C ) < 0} \\ {0} & {\mathrm{otherwise} ,} \end{array} \right.\tag{11.2} $$

    而对角元素为

    $$ \langle C | H_{\mathrm{eff}} | C \rangle = \langle C | H | C \rangle + \langle C | V_{\mathrm{sf}} | C \rangle .\tag{11.3} $$

    该方程中的最后一项是自旋翻转势，部分修正了 $H_{\mathrm{eff}}$ 中缺失的跃迁贡献。该势仅具有对角元素，定义为

    $$ \langle C | V_{\mathrm{sf}} | C \rangle = \sum_{C^{\prime}} {\mathrm{sf}} \langle C | H | C^{\prime} \rangle \psi_{T} ( C^{\prime} ) / \psi_{T} ( C ) .\tag{11.4} $$

    对于对角元，求和符号上的上标 $\mathbf{\hat{s}} 6 \mathbf{\hat{s}}$ 表示对 $C^{\prime}$ 的求和仅限于那些满足 $\langle C^{\prime} | H | C \rangle \psi_{T} ( C )$ $\psi_{T} ( C^{\prime} ) ~ > ~ 0$ 的构型 $C^{\prime}$。在有效哈密顿量的对角线上，一个正势能替换了会导致符号翻转的跳跃（一个非对角元）。蒙特卡洛幂方法精确求解了该有效哈密顿量的基态 $( E_{\mathrm{eff}} , \psi_{\mathrm{eff}} )$。该方法的一个重要特点是，我们将证明，$H_{\mathrm{eff}}$ 的基态是精确基态能量的一个上界。

    晶格固定节点法的几个关键特征如下：（1）它能够处理费米子符号问题以外的符号问题；（2）仅当 $\psi_{T} ( C ) = \psi_{0} ( C )$ 精确成立时，才能得到精确的基态能量（第11.4节）；（3）有效基态能量是变分的，即 $E_{\mathrm{eff}} \geq E_{0}$。

    无论是否存在符号问题，有效使用格林函数方法都需要能够对不同构型轻松求值的试态。这对于许多有趣的多体波函数（如第9.2节讨论的那些）是可行的。能够使用多种试态是该方法的一个优势。如果先通过变分蒙特卡洛方法、一个良好的近似理论或其他波函数优化方法来优化这个试态，就能获得最佳结果。我们注意到，为了得到对基态能量非常优秀的估计而优化试态，可能会产生一个在估计其他可观测量方面效果不如预期的试态。

    11.3 11.3 约束路径方法#

    在约束路径方法 (Zhang et al., 1995, 1997; Carlson et al., 1999) 中，我们将基态 $\left| {\psi_{0}} \right.$ 表示为斯莱特行列式之和 $\begin{array} {r} {| \psi_{0} \rangle = \sum_{\phi} c_{\phi} | \phi \rangle} \end{array}$，其中 $c_{\phi} > 0$，而不是如式(11.1)那样表示为空间构型之和。这一差异及其影响将约束路径方法与扩散蒙特卡洛固定节点法（第10.4.2节）区分开来。在其他许多方面，它们在形式上是一致的。

    约束路径方法是针对投影问题而发展的，在该方法中，Trotter近似和Hubbard-Stratonovich变换被应用于哈密顿量的指数上，从而实现将一个斯莱特行列式投影到另一个斯莱特行列式（附录F）。由此产生的随机游走发生在斯莱特行列式 $| \phi \rangle$ 的空间中，通过对Hubbard-Stratonovich场的分布进行蒙特卡洛采样，生成一个分布 $f ( \phi , \tau )$，该分布估计了到基态的虚时投影。

    与连续问题类似，当Hubbard-Stratonovich场服从高斯分布时（Fahy and Hamann, 1991），存在一个关于权重 $f ( \phi , \tau )$ 的确定性类似扩散的偏微分方程：

    $$ \frac{\partial f} {\partial \tau} = \frac{1} {2} D ( \phi ) f + \left[ \nabla_{\phi} V_{1} ( \phi ) \right] \cdot \nabla_{\phi} f + V_{2} ( \phi ) f\tag{11.5} $$

    这里，扩散算符 $D$、漂移算符 $V_{1}$ 和分支算符 $V_{2}$ 都可以用哈密顿量的算符显式表示。

    这个方程的特殊之处在于它在斯莱特行列式宇称变换 $| \phi \rangle \to - | \phi \rangle$ 下具有不变性（Fahy and Hamann, 1990, 1991; Hamann and Fahy, 1990）。这一对称性使得我们可以将其斯莱特行列式本征函数分类为具有奇宇称或偶宇称。该偏微分方程的最低本征态具有偶宇称，即 $f_{0} ( \phi , \tau ) = f_{0} ( - \phi , \tau )$。由于它是斯莱特行列式之和，因此在构型空间中自然是费米子性的。

    用这个分布来表示，依赖于虚时的波函数为：

    $$ | \psi ( \tau ) \rangle = \sum_{\phi} f ( \phi , \tau ) | \phi \rangle ,\tag{11.6} $$

    其中求和遍及所有归一化的斯莱特行列式，期望值变为

    $$ \langle A ( \tau ) \rangle = \frac{\langle \psi ( \tau ) | A | \psi ( \tau ) \rangle} {\langle \psi ( \tau ) | \psi ( \tau ) \rangle} = \frac{\sum_{\phi} \sum_{\phi^{\prime}} f ( \phi , \tau ) f ( \phi^{\prime} , \tau ) \langle \phi | A | \phi^{\prime} \rangle} {\sum_{\phi} \sum_{\phi^{\prime}} f ( \phi , \tau ) f ( \phi^{\prime} , \tau ) \langle \phi | \phi^{\prime} \rangle} .\tag{11.7} $$

    Fahy 和 Hamann (1990, 1991) 指出，(11.5) 的最低本征态具有偶宇称。如同我们在投影方法的其他讨论中所做的那样，让我们进行本征函数展开

    $$ f ( \phi , \tau ) = \sum_{i} c_{i} f_{i} ( \phi ) \exp ( - E_{i} \tau ) . $$

    当 $\tau \to \infty$ 时，我们看到偶宇称的最低本征态主导此展开，同时也主导 (11.7) 中分子和分母的被积函数。在此极限下，由于 $| \psi \rangle$ 和 $- | \psi \rangle$ 是斯莱特行列式流形中的不同点，因此 (11.6) 中偶宇称的 $f ( \phi , \tau )$ 贡献必须相互抵消，以得到最低的奇宇称本征态。因此，(11.7) 中的积分由最低的奇宇称本征函数主导。对这些积分进行统计评估最终缺乏足够的精度来捕捉不同宇称贡献之间的差异。

    这种“斯莱特行列式空间中的扩散”方法是在网格的主要基态方法为附录I中讨论的确定性方法时发展起来的。该方法为符号问题提供了另一个有趣的视角。它也试图调和连续介质中扩散蒙特卡洛的符号问题与确定性方法中符号问题的相似性质。当时人们认为连续介质方法并未使用哈伯德-斯特拉托诺维奇场。我们现在知道，连续介质扩散蒙特卡洛方法是一种辅助场方法，将高斯哈伯德-斯特拉托诺维奇变换应用于动能指数项 (Zhang et al., 1997; 第10.7节)。在确定性方法中，该变换应用于势能指数项。这两种方法的主要区别在于，一种在构型空间中执行随机游走，另一种在斯莱特行列式空间中执行随机游走。

    导出扩散方程 (11.5) 的分析假设辅助场服从高斯分布。扩散方程的细节取决于哈伯德-斯特拉托诺维奇场的分布，通常我们使用离散场。无论如何，这种形式化分析的优势在于提供深入理解，而非高效的算法。然而，该分析确实在斯莱特行列式空间中引出了一个提议，称为正投影方法（Fahy and Hamann, 1990），作为构型空间中固定节点近似的类比。正投影方法成为约束路径方法的跳板。如果 $\psi \ < \ 0$，正投影方法假设 $f ( \psi ) = 0$，这等价于假设 (11.5) 中的 $V_{2} ( \psi )$ 为无穷大。在这些条件下，最低本征态具有奇宇称。

    其核心而言，约束路径方法是扩散蒙特卡洛类型的幂法。该方法中的关键算符是 $\exp ( - \Delta \tau H )$，这与零温和有限温确定性方法（第7章和附录I）中的关键算符相同。我们回顾一下，在这些章节中，经过特罗特近似和哈伯德-斯特拉托诺维奇变换后，算符 B 表示 $\exp \left( - \Delta \tau H \right)$，传播由依赖于时间依赖辅助场的非相互作用哈密顿量进行。我们可以将传播中的一步表示为

    $$ {\big |} \psi^{\prime} {\big \rangle} = \sum_{x} P ( x ) B ( x ) | \psi \rangle . $$

    这里，$B$ 是一个算符，而 $P(x)$ 是辅助场 $x$ 的分布。

    在约束路径方法中，每个随机游走者 $|\phi\rangle$ 是一个斯莱特行列式：$|\phi\rangle = a_{N_{\sigma}}^{\dagger} \cdots a_{2}^{\dagger} a_{1}^{\dagger} |0\rangle$，其中 $a_{i}^{\dagger} = \sum_{j} c_{j}^{\dagger} \Phi_{j i}$ 由 $N \times N^{\sigma}$ 矩阵 $\Phi$ 定义。根据 Thoules 定理（附录 F），该态经由一个无相互作用的费米子哈密顿量的传播 $e^{-\Delta \tau c^{\dagger} M c} |\phi\rangle$ 得到一个新的斯莱特行列式 $|\phi^{\prime}\rangle$，其中 $|\phi^{\prime}\rangle = b_{N_{\sigma}}^{\dagger} \cdots b_{2}^{\dagger} b_{1}^{\dagger} |0\rangle$，而 $b_{i}^{\dagger} = \sum_{j} c_{j}^{\dagger} \Phi_{j i}^{\prime}$ 由 $N \times N^{\sigma}$ 矩阵 $\Phi^{\prime} = \exp(-\Delta \tau M) \Phi = B \Phi$ 定义。这里 $B$ 是一个矩阵6。

    为了定义实现正投影方法（positive projection method）的方式，我们首先回忆在基态行列式方法（附录 I）中，我们的期望值形式为

    $$ \langle A \rangle = \frac{\sum_{x} P(x) \langle \psi_T | B_L(x_L) \cdots B_{L/2-1}(x_{L/2-1}) A B_{L/2}(x_{L/2}) \cdots B_1(x_1) | \psi_T \rangle}{\sum_{x} P(x) \langle \psi_T | B_L(x_L) B_{L-1}(x_{L-1}) \cdots B_1(x_1) | \psi_T \rangle}. $$

    这里 $\boldsymbol{x} = (x_1, x_2, \ldots, x_L)$ 表示 Hubbard-Stratonovich 场的一个构型，而 $P(x)$ 是与这些场相关联的联合分布函数（第 7.1.1 节和附录 G）。对分母中项的符号施加约束修正了分子的符号。这个约束创建了一个节面（nodal surface），该节面等于垂直于某个约束波函数 $|\psi_C\rangle$ 的超平面，该约束波函数可以是一个或少数几个斯莱特行列式的和。$|\psi_C\rangle$ 越接近精确基态，该超平面就越接近 $f$ 的精确节面。正投影方法施加了 $L$ 个约束

    $$ \begin{array}{rl} & \langle \psi_C | B_1 | \psi_T \rangle > 0, \\ & \langle \psi_C | B_2 B_1 | \psi_T \rangle > 0, \\ & \vdots \\ & \langle \psi_C | B_{L/2} B_{L/2-1} \cdots B_1 | \psi_T \rangle > 0, \\ & \langle \psi_T | B_L B_{L-1} \cdots B_{L/2+1} | \psi_C \rangle > 0, \\ & \vdots \\ & \langle \psi_T | B_L | \psi_C \rangle > 0. \end{array} $$

    仅使用满足所有 $L$ 个约束的 Hubbard-Stratonovich 场构型。对于高斯分布的 Hubbard-Stratonovich 场，这些条件为随机游走者创建了一个无法穿越的屏障。

    约束路径方法（Zhang et al., 1995, 1997；Carlson et al., 1999）通过两项修改使正投影建议更加实用。第一项修改使得投影是开放式的

    $$ \begin{array}{rl} & \langle \psi_C | B_1 | \psi_T \rangle > 0, \\ & \langle \psi_C | B_2 B_1 | \psi_T \rangle > 0, \\ & \langle \Delta \psi_C | B_1 | \psi_T \rangle = 0, \\ & \langle \Delta \psi_C | B_2 B_1 | \psi_T \rangle > 0, \end{array} $$

    也就是说，它变成了幂方法，而不是行列式方法中的固定长度投影。第二个修改消除了任何违反 $\langle \phi | \psi ( \tau ) \rangle > 0$ 的随机游走器；也就是说，它不关注未来可能发生的情况。在实践中，我们选择约束态 $| \psi_{C} \rangle$ 作为引导态 $| \psi_{G} \rangle$，而后者就是 $| \psi_{T} \rangle$。这样，约束就变得容易施加了。

    渐近地，约束路径方法从虚时无关分布 $\pi ( \phi ) = c_{\phi} / \sum_{\phi} c_{\phi}$ 中采样，其中 $c_{\phi} > 0$。将 $\left| {\psi_{0}} \right.$ 分解为一组斯莱特行列式并不是唯一的。我们同样可以写成 $\begin{array} {r} {| \psi_{0} \rangle = \sum_{\Omega^{\prime}} d_{\phi} | \phi \rangle} \end{array}$，其中 $\begin{array} {r} {d_{\phi} > 0 \mathrm{或} | \psi_{0} \rangle = \sum_{\Omega^{\prime \prime}} f_{\phi} | \phi \rangle} \end{array}$，且 $f_{\phi} > 0$，这里 $\Omega^{\prime}$ 和 $\Omega^{\prime \prime}$ 是斯莱特行列式的不同集合。这种灵活性源于斯莱特行列式集合既不正交，$\langle \phi^{\prime} | \phi \rangle \neq \delta_{\phi \phi^{\prime}}$，又过完备，$\textstyle \sum_{\phi} | \phi \rangle \langle \phi | \neq I$。

    由于约束路径方法中的随机游走发生在与固定节点方法不同的基中（图11.1），我们需要一种不同类型的节点约束。在这种不同的基中，从重叠积分 $\langle \psi_{T} | \phi ( \tau ) \rangle$ 为正的区域到为负的区域的跃迁会导致符号问题。重叠积分为不同符号的区域在物理上是无法区分的，因此传播过程会混合由正负对称性导致的简并基。我们需要一个程序来防止这种不加区分的混合。通过将每个游走器约束在 $\langle \psi_{T} | \phi ( \tau ) \rangle > 0$ 的区域来打破正负对称性，可以防止这种混合。

    我们不是同时对整个游走器群体施加约束，而是每次只对一个随机游走器施加约束：如果 $\begin{array} {r} {| \psi ( \tau ) \rangle = \sum_{\phi} c_{\phi} | \phi ( \tau ) \rangle} \end{array}$ 且 $c_{\phi} > 0$，那么对于所有 $| \phi ( \tau ) \rangle$，如果 $\langle \psi_{T} | \phi ( \tau ) \rangle \ > \ 0$，则 $\langle \psi_{T} | \psi ( \tau ) \rangle \ > \ 0$。对于给定步骤，$\begin{array} {r} {\left| {\psi^{\prime}} \right. = \sum_{x} P ( x ) B ( x ) | \psi \rangle} \end{array}$，所以我们从 $P ( x )$ 中采样一个 $x$，然后生成一个新的游走器 $| \phi^{\prime} \rangle = B ( x ) | \phi \rangle$。如果 $\langle \psi_{T} | \phi^{\prime} \rangle > 0$，我们保留该游走器。如果不是，则将其消除。

    约束路径近似并非固定节点近似。虽然选定的 $\vert \psi_{T} \rangle$ 可能将 $| \phi ( \tau ) \rangle$ 约束为“正”集合的一员，但这个 $| \phi ( \tau ) \rangle$ 可能与“负”集合的某个成员重叠。因此，与固定节点方法不同，约束路径方法不会将基分为正交集合。一个后果是，即使约束波函数是近似的，约束路径方法有时也能产生精确解。

    11.4 11.4 估计量 (Estimators)#

    11.4.1 混合估计量 (Mixed estimator)#

    在关于幂方法那一章（第10.5节）中，我们讨论了三种能量估计量：变分估计量 $E_{\nu}$、混合估计量 $E_{m}$ 和增长估计量 $E_{g}$。在固定节点方法中，三者等价，并且提供了基态能量 $E_{0}$ 的上界。在约束路径方法中，这三者之间的关系尚不清楚 (Carlson et al., 1999)。只有变分估计量始终是一个上界。

    在固定节点方法中，因为 $\psi \langle C , \tau \rangle = \bar{\psi} ( C , \tau ) / \psi_{T} ( C )$，我们显然有

    $$ E_{\nu} = \frac{\sum_{C} \psi ( C , \tau ) ( H \psi ( C , \tau ) )} {\sum_{C} \psi^{2} ( C , \tau )} \geq E_{0} . $$

    在大 $\tau$ 时，游走者以概率密度 $\psi_{T} ( C ) \psi ( C )$ 分布，并且 $\psi_{T} ( C )$ 和 $\psi ( C )$ 在节面附近都线性趋近于零。由于哈密顿量和约束都是局域算符，这意味着增长估计量和混合估计量相等：

    $$ E_{g} = E_{m} = \frac{\sum_{C} \psi_{T} ( C ) E_{L} ( C ) \psi ( C )} {\sum_{C} \psi_{T} ( C ) \psi ( C )} . $$

    约束传播子除了在节面附近外，与精确传播子相同，并且约束丢弃了任何与 $| \psi_{T} \rangle$ 和 $| \psi \rangle$ 正交的对基态的贡献。因此，节面对 $\langle \psi_{T} | H | \psi \rangle$ 或 $\langle \psi | H | \psi \rangle$ 都没有净贡献。所以，变分能量与增长估计量和混合估计量相同。混合能量是计算效率较高的那个。

    我们可以证明，由有效固定节点哈密顿量定义的变分能量是基态能量 $E_{0}$ 的一个上界 (ten Haaf et al., 1995)。为此，我们需要证明 $H_{\mathrm{eff}}$ 的基态是真实基态能量的上界。首先，我们将有效哈密顿量和实际哈密顿量写成

    $$ H_{\mathrm{eff}} = H_{\mathrm{tr}} + V_{\mathrm{sf}} , $$$$ H = H_{\mathrm{tr}} + H_{\mathrm{sf}} , $$

    然后定义 $H_{\mathrm{tr}}$ 和 $H_{\mathrm{sf}}$ 在构型空间中的矩阵元。

    为了定义这些矩阵元，我们首先使用上述第一个方程，并注意到 (11.3) 意味着 $H_{\mathrm{tr}}$ 的对角元等于 $H_{\mathrm{eff}}$ 的对角元。然后，从 (11.2) 我们看到 $H_{\mathrm{eff}}$ 的非对角元等于 H 的非对角元。因此，由于 $V_{\mathrm{sf}}$ 是对角的，$H_{\mathrm{tr}}$ 的非对角元等于 H 的非对角元。由于截断哈密顿量 $H_{\mathrm{tr}}$ 的对角元是 $\langle C | H | C \rangle$，从上述第二个方程可以得出 $H_{\mathrm{sf}}$ 的对角元必须为零。最后，因为 $H_{\mathrm{tr}}$ 的非对角元为零，同一个方程意味着 $H_{\mathrm{sf}}$ 的非对角元等于 H 的非对角元。

    现在，我们计算相对于任意态 $| \psi \rangle$ 的 H 与 $H_{\mathrm{eff}}$ 之间的能量差。我们发现

    $$ \Delta E = \left. \psi \right| ( H_{\mathrm{eff}} - H ) \left| \psi \right. = \left. \psi \right| ( V_{\mathrm{sf}} - H_{\mathrm{sf}} ) \left| \psi \right. , $$

    利用 $V_{\mathrm{sf}}$ 和 $H_{\mathrm{sf}}$ 的矩阵元，上式等于

    $$ \Delta E = \sum_{C} \psi ( C ) \left[ \langle C | V_{\mathrm{sf}} | C \rangle \psi ( C ) - \sum_{C^{\prime}} \langle C | H_{\mathrm{sf}} | C^{\prime} \rangle \psi ( C^{\prime} ) \right] . $$

    接下来，我们将这个表达式改写为 $H_{\cdot}$ 的矩阵元形式

    $$ \Delta E = \sum_{C} \psi ( C ) \left[ \sum_{C^{\prime}} \mathrm{s}^{\mathrm{f}} \langle C | H | C^{\prime} \rangle \frac{\psi_{T} ( C^{\prime} )} {\psi_{T} ( C )} \psi ( C ) - \sum_{C^{\prime}} \mathrm{s}^{\mathrm{f}} \langle C | H | C^{\prime} \rangle \psi ( C^{\prime} ) \right] .\tag{11.8} $$

    将 $s ( C , C^{\prime} )$ 记为矩阵元符号并注意到求和中的所有项均满足条件

    $$ \langle C | H | C^{\prime} \rangle \psi_{T} ( C ) \psi_{T} ( C^{\prime} ) < 0 , $$

    我们得到

    $$ \Delta E = \sum_{C , C^{\prime}}^{\mathrm{vf}} | \langle C | H | C^{\prime} \rangle | \left| \psi ( C ) \sqrt{\left| \frac{\psi_{T} ( C^{\prime} )} {\psi_{T} ( C )} \right|} - s ( C , C^{\prime} ) \psi ( C^{\prime} ) \sqrt{\left| \frac{\psi_{T} ( C )} {\psi_{T} ( C^{\prime} )} \right|} \right| ^{2} , $$

    对于任意态 $| \psi \rangle$，该表达式显然非负。因此 $H_{\mathrm{eff}}$ 的基态能量是原始哈密顿量 H 的基态能量的上界，即 $E_{\mathrm{eff}} \geq E_{0}$。

    晶格固定节点方法还有其他几个重要性质。首先，我们可以容易地证明 $H | \psi_{T} \rangle = H_{\mathrm{eff}} | \psi_{T} \rangle$。该等式确保了固定节点格林函数蒙特卡洛过程能够改善试探态的能量：

    $$ \frac{\langle \psi_{T} | H | \psi_{T} \rangle} {\langle \psi_{T} | \psi_{T} \rangle} = \frac{\langle \psi_{T} | H_{\mathrm{eff}} | \psi_{T} \rangle} {\langle \psi_{T} | \psi_{T} \rangle} \geq E_{\mathrm{eff}} = \frac{\langle \psi_{\mathrm{eff}} | H | \psi_{\mathrm{eff}} \rangle} {\langle \psi_{\mathrm{eff}} | \psi_{\mathrm{eff}} \rangle} \geq E_{0} . $$

    其次，我们可以证明，如果使用精确基态作为试探态，则有效哈密顿量的基态能量等于真实基态能量。因此，通过改变试探态，我们可以达到真实的基态能量。

    晶格和连续体系固定节点方法之间存在一个重要区别。在连续体系中，只有试探态的符号起作用。如果节点面（nodal surface）正确，无论试探态的幅度大小如何，我们都能得到精确能量。而在晶格上，由(11.8)可知，当 $\Delta E = 0$ 时，如果对于所有符号翻转构型对 $( C , C^{\prime} )$ 满足

    $$ \frac{\psi ( C^{\prime} )} {\psi ( C )} = \frac{\psi_{0} ( C^{\prime} )} {\psi_{0} ( C )} $$

    我们就能得到精确能量。因此，在由符号翻转连接的构型中，试探态的符号和相对幅度必须正确。

    在约束路径方法中，虽然由 $| \psi ( \tau ) \rangle$ 构造的变分能量满足

    $$ E_{\nu} = \frac{\langle \psi_{\mathrm{eff}} ( \tau ) | H | \psi_{\mathrm{eff}} ( \tau ) \rangle} {\langle \psi_{\mathrm{eff}} ( \tau ) | \psi_{\mathrm{eff}} ( \tau ) \rangle} , $$

    但 $E_{\nu}$、$E_{m}$ 和 $E_{g}$ 的相对大小关系并不明确，因为约束条件丢弃了与 $\vert \psi_{T} \rangle$ 正交的构型，但这些被丢弃的构型并不一定与 $| \psi \rangle$ 正交 (Carlson et al., 1999)。

    一般而言，如果约束路径方法定义了一个具有平稳分布的马尔可夫过程，使得

    $$ H_{\mathrm{eff}} | \psi ( \tau ) \rangle = E_{g} | \psi_{\mathrm{eff}} ( \tau ) \rangle , $$

    其中 $H_{\mathrm{eff}} = H + \delta H$，则

    $$ \frac{\langle \psi ( \tau ) | H | \psi ( \tau ) \rangle} {\langle \psi ( \tau ) | \psi ( \tau ) \rangle} = E_{m} + \delta E_{m} \geq E_{0} , $$

    其中

    $$ \delta E_{m} = \frac{\langle \psi_{T} | \delta H | \psi ( \tau ) \rangle} {\langle \psi_{T} | \psi ( \tau ) \rangle} - \frac{\langle \psi ( \tau ) | \delta H | \psi ( \tau ) \rangle} {\langle \psi ( \tau ) | \psi ( \tau ) \rangle} . $$

    只有当 $\delta E_{m} \leq 0$ 时，$E_{m}$ 才是基态能量的上界。能量的混合估计量通常不是基态能量的上界。一般而言，增长估计量和混合估计量并不相等，只有变分估计量是上界。7

    11.4.2 前向行走与反向传播（forward walking and back propagation）#

    混合估计量对与哈密顿量对易的可观测量能提供精确的估计。对于那些不对易的可观测量，它可能会给出较差的估计。例如，混合估计量能很好地预测总能量，但通常对动能和势能给出较差的估计。对于与哈密顿量不对易的可观测量，我们需要使用其他估计量。对于这些情况，最简单的估计量是外推估计量

    $$ \langle A \rangle \approx \langle A \rangle_{\mathrm{ex}} = 2 \langle A \rangle_{m} - \langle A \rangle_{\nu} .\tag{11.9} $$

    如果尝试态与真实基态偏差显著，那么这个估计量也会给出较差的估计。幸运的是，存在更可靠的估计量。我们在第10.5节中简要提及了这些方法。对于固定节点方法，我们只讨论

    所谓的前向（或未来）行走方法（前向行走方法，Hammond et al., 1994）。在讨论中，我们区分可观测量在模拟基中为对角的情况和不为对角的情况（Nightingale, 1999）。

    我们从对角情况开始。我们希望

    $$ \langle A \rangle = \frac{\langle \psi_{0} | A | \psi_{0} \rangle} {\langle \psi_{0} | \psi_{0} \rangle} = \frac{\sum_{C} A ( C ) \psi_{0}^{2} ( C )} {\sum_{C} \psi_{0}^{2} ( C )} . $$

    问题在于，当马尔可夫链平稳时，它从 $\psi_{T} ( C ) \psi_{0} ( C )$ 而非 $\psi_{0}^{2} ( C )$ 中采样。为了从基态分布中采样，策略是估计一个权重 $w_{0} ( C )$，使得

    $$ \langle A \rangle = \frac{\sum_{C} A ( C ) \psi_{0}^{2} ( C )} {\sum_{C} \psi_{0}^{2} ( C )} = \frac{\sum_{C} w_{0} ( C ) A ( C ) \psi_{T} ( C ) \psi_{0} ( C )} {\sum_{C} w_{0} ( C ) \psi_{T} ( C ) \psi_{0} ( C )} . $$

    显然，我们需要 $w_{0} ( C ) \propto \psi_{0} ( C ) / \psi_{T} ( C )$

    为了估计这个权重，我们首先注意到对于大的 L

    $$ \langle A \rangle = \frac{\langle \psi_{T} \vert G^{L} A \vert \psi_{0} \rangle} {\langle \psi_{T} \vert G^{L} \vert \psi_{0} \rangle} . $$

    通过多次插入单位算子，我们将这个表达式变换为

    $$ \langle A \rangle = \frac{\sum_{C_{0} , C_{1} , \ldots , C_{L}} \psi_{T} ( C_{L} ) G ( C_{L} , C_{L - 1} ) \cdot \cdot \cdot G ( C_{1} , C_{0} ) A ( C_{0} ) \psi_{0} ( C_{0} )} {\sum_{C_{0} , C_{1} , \ldots , C_{L}} \psi_{T} ( C_{L} ) G ( C_{L} , C_{L - 1} ) \cdot \cdot \cdot G ( C_{1} , C_{0} ) \psi_{0} ( C_{0} )} . $$

    下一步是用其重要性采样值的形式写出相关量。这样做我们得到

    $$ \langle A \rangle = \frac{\sum_{C_{0} , C_{1} , \dots , C_{L}} \bar{G} ( C_{L} , C_{L - 1} ) \cdot \cdot \cdot \bar{G} ( C_{1} , C_{0} ) A ( C_{0} ) \bar{\psi}_{0} ( C_{0} )} {\sum_{C_{0} , C_{1} , \dots , C_{L}} \bar{G} ( C_{L} , C_{L - 1} ) \cdot \cdot \cdot \bar{G} ( C_{1} , C_{0} ) \bar{\psi}_{0} ( C_{0} )} , $$

    其中

    $$ \bar{G} ( C^{\prime} , C ) = \frac{\langle{C^{\prime} | \psi_{T} \rangle \langle{C^{\prime} | G | C} \rangle}} {\langle{C | \psi_{T} \rangle}}\tag{11.10} $$

    （经重要性采样修正的转移矩阵）。现在我们有了一个有用的结果。我们知道模拟最终会到达一个步骤0，在此处它开始从平稳重要性采样分布 $\bar{\psi}_{0} ( C ) =$ $\psi_{T} ( C ) \psi_{0} ( C )$ 中采样。在这个步骤，我们记录 $A ( C_{0} )$ 的值以及每个行走器的权重和状态。我们还知道，对于步0处的每个行走器，长度为L的马尔可夫链会生成一个后代群体，该群体近似采样一个平稳分布 $\psi_{0} ( C )$。因此，后代的权重为我们提供了所需的 $w_{0} ( C )$ 估计值。

    然而，由于第10.3节中随机重组过程中的“杀死”操作，步0处的一些行走器在步L处没有后代。由于分支过程，其他行走器可能有多个后代。因此，为了估计 $\langle A \rangle$，我们需要沿着L段路径的每一步记录行走器亲本的标识。我们通过将行走器在步0处的权重、其在步0处的A值及其在步L处后代的权重相乘，来获得步0处行走器对分子的贡献。并非所有步0处的行走器都对分子值有贡献。我们以类似的方式积累分母的估计值。

    非对角算符的估计更为复杂，并且可能非常困难（Nightingale, 1999）。首先，我们现在有

    $$ \langle A \rangle = \frac{\sum_{C_{0} , C_{1} , \dots , C_{L + 1}} \bar{G} ( C_{L + 1} , C_{L} ) \cdot \cdot \cdot \bar{G} ( C_{2} , C_{1} ) A ( C_{1} , C_{0} ) \bar{\psi}_{0} ( C_{0} )} {\sum_{C_{0} , C_{1} , \dots , C_{L + 1}} \bar{G} ( C_{L + 1} , C_{L} ) \cdot \cdot \cdot \bar{G} ( C_{1} , C_{0} ) \bar{\psi}_{0} ( C_{0} )} ,\tag{11.11} $$

    我们可以立即将其更便捷地写为

    $$ \langle A \rangle = \frac{\sum_{C_{0} , C_{1} , \dots , C_{L + 1}} \bar{G} ( C_{L + 1} , C_{L} ) \cdot \cdot \cdot \bar{G} ( C_{2} , C_{1} ) \bar{G} ( C_{1} , C_{0} ) \frac{A ( C_{1} , C_{0} )} {\bar{G} ( C_{1} , C_{0} )} \bar{\psi}_{0} ( C_{0} )} {\sum_{C_{0} , C_{1} , \dots , C_{L + 1}} \bar{G} ( C_{L + 1} , C_{L} ) \cdot \cdot \cdot \bar{G} ( C_{1} , C_{0} ) \bar{\psi}_{0} ( C_{0} )} . $$

    最后一个方程分子中的分数替换了(11.11)中的 $A ( C_{0} )$，并且我们执行采样过程，就像在对角情况中那样，记住行走器的亲本。只要 $A ( C_{1} , C_{0} )$ 在 $\bar{G} ( C_{1} , C_{0} )$ 为零时也为零，且反之亦然，那么一切进展顺利。通常情况并非如此，因此需要其他我们没有描述的技术（Nightingale, 1999）。这些技术比之前提到的要复杂得多。

    对于约束路径方法，混合估计量和外推估计量的效用范围与它们在固定节点方法中的相同。我们使用与固定节点方法相同的策略来超越这些估计量，采用所谓的反向传播方法（back-propagation method）（Zhang et al., 1997）作为前向行走方法的类比。与前向行走方法相比，我们不需要为对角和非对角可观测量分别制定策略。在这里，我们以与行列式方法（第7.4节和附录I）相同的方式，计算作为格林函数乘积之和的期望值。因此，我们在实现测量时享有同样简便性和灵活性。这种简便性是约束路径方法的一个积极特点。

    在反向传播中，在步0处，我们保存行走器（权重和斯莱特行列式），然后通过B矩阵根据标准算法传播群体，记录亲本的标识，并额外保存每一步每个行走器的哈伯德-斯特拉托诺维奇场（Hubbard-Stratonovich fields）。对于每个到达步L的行走器，我们现在通过以下方式反向传播 $\langle \psi_{T} |$：

    $$ ( ( ( \langle \psi_{T} | B_{L} ) B_{L - 1} ) B_{L - 2} \cdot \cdot \cdot ) $$

    直到我们回到第 0 步，比如，得到游走者 $\langle \phi^{\prime} |$。在反向传播的每一步，我们不再采样新的哈伯德-斯特拉托诺维奇场，而是重用前向步骤中记录的场，但倒序使用。更明确地说，在第一个反向传播步骤中，我们使用最后一个前向传播步骤的场；在第二个反向传播步骤中，我们使用倒数第二个前向传播步骤的场；依此类推。在向后方向上，我们执行矩阵稳定化，但不应用约束路径条件或随机重构。每个到达 L 的游走者已经通过了约束路径测试，并且整个群体可能已被重构了数次。对于每个返回的游走者 $\langle \phi^{\prime} |$，我们计算它与其父游走者 $| \phi \rangle$ 的 $\langle \phi^{\prime} | A | \phi \rangle$，然后将这个值乘以父游走者在 0 步的权重和后裔在 L 步的权重的乘积。并非所有父游走者都起作用，只有那些有后裔的才起作用。蒙特卡洛测量通常是周期性地进行的（第 3.9 节）。我们可以方便地选择 L 为这个周期的倍数。完成反向传播测量后，我们返回到 L 步的群体，并用它来进行接下来的 L 步。

    如果$p_{1}$或$p_{2}$非正，我们将其设为零。如果两者均非正，我们将$\mathcal{O}_{T} < 0$并从该过程返回。否则，我们以概率$p_{1}$或$p_{2}$选择$x^{\prime}$等于$+1$或$-1$。使用这个$x^{\prime}$，我们随后更新游走者

    $$ w w N , O_{T} O_{T} ( x^{\prime} ) , \Phi b_{V} ( x^{\prime} ) \Phi . $$

    我们还需要更新 $[ \Psi_{T}^{T} \Phi ]^{- 1} [ \Psi_{T}^{T} \Phi^{\prime} ]^{- 1}$ 。

    我们可以简化 $p_{1}$ 和 $p_{2}$ 的表达式，但我们特意将它们写成显式的行列式比。回忆在行列式方法（第7章）中，当我们更新 Hubbard-Stratonovich 场 $x_{i}$ 的变化时，需要计算两个行列式的比值，如果知道零温格林函数 $G = [ I - \Phi \bigl ( \Psi_{T}^{T} \Phi \bigr )^{- 1} \Psi_{T} ]^{T}$ 的第 $i$ 个对角元素，这个计算可以大大简化。要知道这个元素，需要知道重叠矩阵 $\Psi_{T}^{T} \Phi$ 的逆。要点在于，在幂方法中，我们不需要知道格林函数的所有空间元素，只需要知道正在更新其辅助场的那个格点的元素。通过输入重叠矩阵的行列式和逆矩阵，再加上 walker 的状态，我们可以轻松地完成对该第一个场的更新：

    $$ G_{i i} = 1 - \sum_{j k} \Phi_{i j} \left[ \left( \Psi_{T}^{T} \Phi \right)^{- 1} \right]_{j k} [ \Psi_{T} ]_{j k} . $$

    要循环到下一个场，我们需要更新那些用于计算与该场对应的格林函数对角元素的量。这些量包括新行列式的值、walker 的新状态以及新重叠矩阵的逆。我们可以通过比例 ${\mathcal{O}}_{T}^{\prime} / {\mathcal{O}}_{T}$ 和旧重叠 $\mathcal{O}_{T}$ 的简单乘法来获得新的重叠行列式。为了得到新的重叠矩阵，我们首先需要更新 walker 的状态，这是通过将表示旧状态的矩阵与传播它的矩阵 $b_{V} ( x^{\prime} )$ 相乘来完成的。为了得到新的逆矩阵，我们并不构造 $\Psi_{T}^{T} \Phi^{\prime}$ 然后对其求逆，而是像在行列式方法和连续时间杂质算法（第7章和第8章）中所做的那样，利用 $\Phi$ 只改变了少数元素（此处仅一行）以及我们知道原始逆矩阵这一事实。如果我们称 $A^{- 1}$ 为已知的逆矩阵，$u$ 为 $\Psi_{T}$ 的第 $i$ 行，$v$ 为 $\Phi_{\mathrm{{;}}}$ 的第 $i$ 行，那么新的逆矩阵为（Sherman-Morrison 公式；Meyer, 2000; Press et al., 2007）

    $$ \big ( A + u \nu^{T} \big )^{- 1} = A^{- 1} - \frac{A^{- 1} u \nu^{T} A^{- 1}} {1 + u A^{- 1} \nu^{T}} . $$

    因此，我们可以在符号上将 $G_{i j}$ 替换为 $A_{i j} = [ \Psi_{T}^{T} \Phi^{\prime} ]_{i j}$ 后，利用 (7.45)（另见第8.4.2节）更新重叠矩阵的逆。这种更高效的计算方式正是我们在计算 $B_{K / 2}$ 和 $B_{K}$ 过程的末尾计算重叠矩阵的逆，以及在用 $B_{V}$ 传播时，每次选取新的 Hubbard-Stratonovich 场后更新该逆矩阵的原因。在一次扫描结束时，我们并不具备进行测量所需的格林函数矩阵元素。通常，我们会每隔几次扫描进行一次测量，并在需要进行测量时直接计算正确的格林函数。

    作为一种幂方法（power method），约束路径方法受益于随机重构（第10.3节）。其一优点在于标准的方差缩减。其二是对游走者种群的补充。由于约束会剔除负符号的游走者，若不补充，种群将衰减至零。此时，梳形法（comb）是一种有效的程序（算法38）。

    该方法固有的重复矩阵-矩阵乘法会积累数值误差。矩阵乘积稳定化程序比有限温行列式方法（第7章）更简单，但与零温方法（附录I）相同。稳定化归结为周期性地对传播矩阵的列进行正交归一化。同样，修正的 Gram-Schmidt 方法效果良好（第7.4节）。在附录I中，我们提供了执行此正交归一化的伪代码（算法50）。仅需此部分矩阵分解的理由与零温行列式方法中给出的论证相同。

    与有限温和零温行列式方法类似，利用约束路径方法，我们可以通过格林函数乘积的和来计算任意可观测量的期望值。几乎所有感兴趣的物理量都需要使用反向传播（back-propagation）。如前面几节所述，反向传播将混合估计中的 $\langle \psi_{T} |$ 替换为 $\langle \psi_{B} | = \langle \psi_{T} | B_{L} B_{L - 1} \cdot \cdot \cdot B_{1}$，其中 $B_{i} = B_{K / 2} B_{V} ( i ) B_{K / 2}$，而 $B_{V} ( i )$ 是乘积中第 i 项的 $B_{V}$ 矩阵。从左到右乘法中使用的 Hubbard-Stratonovich 场与从右到左乘法 $B_{L} B_{L - 1} \cdot \cdot \cdot B_{1} | \phi \rangle$ 中使用的相同。正如我们之前指出的，游走者现在必须在这 L 步中携带其父代的标识，并且为了执行反向传播，我们必须存储传播过程中为每个游走者生成的 Hubbard-Stratonovich 场 $B_{V} = \prod b_{V} ( x )$。这种簿记工作最方便的方式是在 $B_{V}$ 程序的最后一步完成。当进行随机重构时，我们需要维护系谱。

    最近，Shi 和 Zhang（2013）展示了如何利用对称性来提高使用辅助场的基态模拟的精度和效率。

    11.6 11.6 约束相位方法#

    约束相位方法（constrained-phase method），有时也称为无相位方法（phaseless method），它将约束路径方法扩展到具有复数权重的零温费米子模拟（Ortiz et al., 1997; Schmidt and Fantoni, 1999; Schmidt et al., 2001, 2003; Sarsa et al., 2003; Zhang and Krakauer, 2003; Zhang, 2013）。复数权重的产生有多个原因。最主要的原因是所模拟的系统破坏了时间反演对称性（time-reversal symmetry）。通常，这种对称性由外磁场破坏，使得波函数变为复数。在格点模型中，磁场通过跃迁矩阵元 $t_{i j}$ 进入，由于沿连接格点位置 $i$ 和 $j$ 的路径 $\ell$ 上作用的矢势 A，该矩阵元变为 $t_{i j}$ exp[i $\textstyle \int A \cdot d \ell ]$ 。通过闭合路径的净相位是穿过该回路的磁通量的度量。哈密顿量矩阵现在是厄米的（Hermitian），而不是简单的对称矩阵。严格的厄米哈密顿量也出现在包含自旋-轨道相互作用（spin-orbit interaction）的情况下，这在多电子物理中是一个当前备受关注的问题。如果 Hubbard-Stratonovich 变换未被选择为实数，则会产生复数权重（附录 G）。这种选择对于形式比简单的在位 Hubbard 相互作用更一般的电子-电子相互作用是方便的。

    获得复数波函数的一种人为但有用的方法是给周期性边界条件添加一个相移，例如，在一维情况下使用

    $$ \psi ( x + L ) = \exp ( i \varphi ) \psi ( x ) .\tag{11.17} $$

    这种边界条件打破了先前导致特征值简并的对称性，并且是扭曲相位边界条件方法（twisted-phase boundary condition method）的基础（Gammel et al., 1993; Lin et al., 2001）。该方法将系统的性质在区间 [0, 2π] 内均匀随机选择的许多不同 $\varphi$ 值上取平均。简并分裂和平均消除了总能作为电子密度函数的闭壳层效应（closed-shell effects），并产生了更接近无限系统的密度依赖性。对于小的有限系统，闭壳层效应表现为在某一密度处总能斜率的间断性，在该密度下，上下自旋电子的最高占据态能级与下一个未占据态能级之间各有一个能隙。通常，由于非相互作用电子的能带布里渊区的点群对称性（point-group symmetry），每个自旋的最高占据能级是简并的。小系统中的相互作用通常只能稀疏地填补非相互作用问题中存在的能隙。

    约束相位方法是固定相位方法（fixed-phase method）（Ortiz et al., 1993）的类似物，后者是为存在磁场的连续介质问题而发展起来的。后一种方法将波函数表示为

    $$ \psi ( C ) = | \psi ( C ) | \exp ( i \varphi ( C ) ) $$

    代入薛定谔方程，生成两个耦合的偏微分方程，一个用于振幅 $| \psi ( C ) |$ ，另一个用于相位 $\varphi ( C )$ 。然后，在 $| \psi ( C ) |$ 的方程中，用试探态的相位 $\varphi_{T} ( C )$ 来近似 $\varphi ( C )$ 。扩散蒙特卡洛方法现在求解 $| \psi ( C ) |$ 的方程。由于该函数通过构造处处非负，因此不存在符号问题。

    对于格点费米子问题，其中我们从一个 Slater 行列式传播到另一个（附录 F），我们以类似于生成约束路径近似的方式来生成约束相位近似。在那里，对于实数的 $\langle \psi_{T} | \psi ( \tau ) \rangle$ ，我们通过要求 $\langle \psi_{T} | \psi ( \tau ) \rangle > 0$ 并在虚时传播的每一步对每个游走者施加约束来实现正性。在这里，对于复数的 $\langle \psi_{T} | \psi ( \tau ) \rangle$ ，我们通过要求 $\langle \psi_{T} | \psi ( \tau ) \rangle = \mathrm{Re} \{\langle \psi_{T} | \psi ( \tau ) \rangle \} > 0$ 或 $\langle \psi_{T} | \psi ( \tau ) \rangle = | \langle \psi_{T} | \psi ( \tau ) \rangle | > 0$ 来实现实数和正性。和之前一样，我们一次对一个游走者施加约束。对于任一选项，我们定义

    $$ \langle \phi | \psi ( \tau ) \rangle = e^{i \varphi_{\tau}} | \langle \phi | \psi ( \tau ) \rangle | $$

    并取 $\varphi_{\tau} ( \phi ) = \varphi_{T} ( \phi ) = \mathrm{Im} \{\ln \langle \phi | \psi_{T} \rangle \}$ 固定相位。如果试探态恰好就是精确基态，那么这两种选择都会得到精确结果。

    基本步骤是将 $| \phi \rangle$ 传播到 $| \phi^{\prime} \rangle$，即 $\mathrm{xp} [ - \Delta \tau H ] | \phi \rangle = | \phi^{\prime} \rangle$，并用以下两种方式之一将 $| \phi^{\prime} \rangle$ 替换为 $| \tilde{\phi} \rangle$。对于 $\mathrm{Re} \{\langle \psi_{T} | \psi ( \tau ) \rangle \} > 0$ 的情况，

    $$ | \tilde{\phi} \rangle = \cos \varphi_{T} ( \phi^{\prime} ) e^{- i \varphi_{T} ( \phi^{\prime} )} | \phi^{\prime} \rangle , $$

    而对于 $| \langle \psi_{T} | \psi ( \tau ) \rangle | > 0$ 的情况，

    $$ | \tilde{\phi} \rangle = e^{- i \varphi_{T} ( \phi^{\prime} )} | \phi^{\prime} \rangle . $$

    这些替换仅在细微之处有差别，但第一种替换有时能给出方差更小的能量估计。然而，至今为止对固定相位方法的测试仍很有限。

    无论选择哪种方式，${\mathcal{O}}_{T} ( \phi )$ 都是实数，但 $\mathcal{O}_{T} ( \phi^{\prime} )$ 不是。将 $| \phi^{\prime} \rangle$ 替换为 $| \tilde{\phi} \rangle$ 得到了一个实数的 $\mathcal{O}_{T} ( \tilde{\phi} )$。如上定义，$| \tilde{\phi} \rangle$ 形如 det[-] 乘以 a，这等于 det $[ a^{1 / N_{\mathrm{elec}}} \Phi^{\prime} ]$，进而定义 $\tilde{\Phi}$ 为 $a^{1 / N_{\mathrm{elec}}} \Phi^{\prime}$。

    将约束路径算法（算法 39 和 40）转换为约束相位算法只需几个步骤。第一步是在必要时切换到复数运算和变量。接下来的步骤是：在通过 $B_{K / 2}$ 和 $B_{V}$ 传播的过程中，将测试 $\mathcal{O}_{T} ( \phi^{\prime} ) ~ < ~ 0$ 和用 $w w \vert \mathcal{O}_{T} ( \phi^{\prime} ) \vert$ 更新游走者权重的操作进行替换。然后，在执行 $B_{V}$ 的过程中，我们将计算 $p_{i}$ 时用到的 $\mathcal{O}_{T} ( i )$ 替换为 $| \mathcal{O}_{T} ( i ) | / \mathcal{O}_{T} ( \phi )$。最后，在执行完 $B_{K / 2} B_{V} B_{K / 2}$ 之后，限制相位。这里的第一步是计算 $\varphi_{T} ( \phi^{\prime} )$ 并测试其绝对值是否小于 $\pi / 2$。如果是，则用 $| \tilde{\phi} \rangle$ 更新 $| \phi \rangle$，并用 exp $[ - i \varphi_{T} ( \phi^{\prime} ) ] \mathcal{O}_{T} ( \tilde{\phi} )$ 更新 $\mathcal{O}_{T} ( \phi )$。如果不是，则将该游走者的权重设为零。
