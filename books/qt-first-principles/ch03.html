    新原理#

    本章概述了用于重构量子理论的新原理。我们在第2章已经看到这些原理，在那里它们是作为量子理论的定理推导出来的。在本书第四部分，我们将实际进行反向过程，即从这些原理推导出量子理论。量子理论的所有特征——从叠加原理、纠缠、不可克隆、隐形传态、贝尔不等式违反、量子密码学——都可以仅使用这些原理来理解和证明，而无需使用希尔伯特空间，这确实是本书第三部分的目标，而第四部分则包含从原理出发推导理论的过程。

    所有六条原理都是操作性（operational）的，因为它们规定了某些任务是否能够完成：它们为所有实验和所有可以在该理论中执行的协议设定了游戏规则。它们还为我们深入理解量子理论所暗示的世界观提供了巨大洞见。

    我们回顾一下这些原理的列表：

    1. 组成原子性（Atomicity of Composition）
    2. 完美可区分性（Perfect Discriminability）
    3. 理想压缩（Ideal Compression）
    4. 因果性（Causality）
    5. 局域可区分性（Local Discriminability）
    6. 纯化（Purification）

    除纯化原理外，所有六条原理都表达了经典理论和量子理论共有的标准特征。纯化原理在前五条原理所允许的理论中唯一地挑选出量子理论，部分解释了量子信息的魔力。

    因果性、局域可区分性和纯化原理将分别在第五章、第六章和第七章中深入分析。其余三条原理——组成原子性、完美可区分性、理想压缩——将在本章剩余部分进行考察。

    4.1 组成原子性#

    在一般框架中，我们遇到过粗粒化操作（coarse-grained operation）和原子操作（atomic operation）的概念。粗粒化操作是通过合并一次检验（test）中的结果而得到的，对应于忽略某些信息。粗粒化的逆过程是精炼（refining）。原子操作是没有忽略任何信息的操作，即无法再精炼的操作。当操作是原子性时，实验者对实验室中发生的事拥有最大程度的知识。由原子操作组成的检验代表了根据我们的理论所能达到的最高控制水平。但是，是否有可能在整序列实验中都保持这样的控制水平？

    组成原子性原理给出了肯定回答，精确地表述如下：

    公理1（组成原子性）两个原子操作的序列是一个原子操作。

    组成原子性是非常基本的规则，涉及最大信息如何随时间传播。设想一个该原理不成立的世界。在那个世界中，Alice执行一个操作 $\mathcal{A}_{x}$，其控制精度之高使她不可能更好地了解她的系统发生了什么。紧接着，Bob对Alice的系统执行另一个操作 $B_{y}$，他也对自己所做的事拥有最大知识。得到的操作 $B_{y} A_{x}$ 是粗粒化的，即这是一个可以由第三方Charlie模拟的操作：他执行一个检验 $\{{\mathcal{C}}_{z} \}_{z \in Z}$，然后将结果合并到合适的子集 $\mathsf{S}_{x y} \subset Z$ 中

    $$ \frac{\textsc{A} \boxed{\mathcal{A}_{x}} \textsc{A} \boxed{\mathcal{B}_{y}} \textsc{A}} {z \in \mathbb{S}_{x y}} = \sum_{z \in \mathbb{S}_{x y}} \frac{\textsc{A} \boxed{\mathcal{C}_{z}} \textsc{A}} {z} . $$

    尽管这一场景在逻辑上是可以想象的，但它引发了一些问题：所谓“额外信息”是关于什么的？哪些物理参数对应于结果 (z)？显然，它不是关于第一步发生的事件，因为爱丽丝已经掌握了关于这一步的最大知识。也不是关于第二步发生的事件，因为鲍勃掌握了关于那一步的最大信息。结果 (z) 必须指定两个时间步骤如何相互作用的一个特征——从某种意义上说，这是一种在时间上非局域的信息。量子理论是非局域的，但并不是以这种极端的方式！事实上，量子理论中的原子操作由具有单个Kraus算子的完全正映射描述，即形式为 (\mathcal{A}{x} ( \cdot ) = A{x} \cdot A_{x}^{\dagger}) 和 (\mathcal{B}{y} ( \cdot ) = B{y} \cdot B_{y}^{\dagger})，并且显然两个原子操作的复合仍然是原子的：(B_{y} A_{x} ( \cdot ) = ( B_{y} A_{x} ) {\cdot} ( B_{y} A_{x} )^{\dagger})。复合的原子性保证了这一性质在基本原理层面得以成立。

    在一个违反复合原子性的世界中，可能会产生另一个奇怪的特征。假设爱丽丝将系统A制备在纯态 (\alpha) 中，然后鲍勃将系统B制备在纯态 (\beta) 中。如果原子性不成立，那么结果态 (\alpha \otimes \beta) 可能是混合的：对单个系统A和B如何制备拥有最大知识，并不足以保证对复合系统AB的制备拥有最大知识。同样，这在量子理论中并非如此，其中两个波矢量为 (|\alpha\rangle) 和 (|\beta\rangle) 的纯态的张量积是波矢量为 (|\alpha\rangle \otimes |\beta\rangle) 的纯态。在接下来的几段中，我们将预览局部可区分性（local discriminability）原理，它与复合原子性一样，导致两个纯态的乘积必须是纯态这一结果。然而，值得强调的是，这两个原理的内容是不同的且在逻辑上相互独立。它们之所以都导致关于纯态乘积的相同结果，是因为对于制备过程而言，顺序复合（复合原子性所指）退化为了并行复合（局部可区分性所指）。

    4.2 完美可区分性（Perfect Discriminability）#

    无论是在经典理论还是量子理论中，每一个非平凡系统都至少有两个完美可区分的状态。例如，经典比特的状态0和1，或者量子比特的状态 (|0\rangle) 和 (|1\rangle)。在我们的通用设定中，两个确定性状态 (\rho_{0}) 和 (\rho_{1}) 是完美可区分的，如果存在一个测量 ({m_{y}}_{y \in {0, 1}})，使得

    $$ ( m_{y} \mid \rho_{x} ) = \delta_{x y} \qquad \forall x , y \in \{0 , 1 \}. $$

    完美可区分状态的存在非常重要，因为这些状态可以用于无误差地传输经典信息。在通信协议中，发送方可以将比特x的值编码为状态$\rho_{x}$，然后将系统传输给接收方，接收方可以使用测量$\{m_{\mathrm{y}} \}_{\mathrm{y \in \{0 , 1 \}}}$来解码该比特的值。

    原则上，我们可以设想一些理论，其中根本不存在完美可区分状态。例如，我们可以考虑经典理论的一个带噪声版本，其中比特的可能状态由概率分布$p ( x )$描述，对于每个$x \in \{0 , 1 \}$，满足$p ( x ) ~ \geq ~ \epsilon$。该噪声比特有两个（且只有两个）纯态，分别对应于概率分布$p_{0} ( x ) = ( 1 - \epsilon ) \delta_{x , 0} + \epsilon \delta_{x , 1}$和$p_{1} ( x ) = ( 1 - \epsilon ) \delta_{x , 1} + \epsilon \delta_{x , 0}$。所有其他状态都是这两个状态的混合。容易看出，噪声比特没有任何状态可以被完美区分（参见练习4.2）。像我们刚刚描述的这种噪声理论，对于现实实验可能是有用的模型，因为在实践中总是存在一些不完美，阻止我们完美区分状态。然而，如果我们希望在最基础的层面描述自然，噪声理论就显得相当不吸引人。

    完美可区分性公理确保了我们区分状态的能力尽可能地精确：除了一些平凡情况外，每个状态都能与某个其他状态完美区分。“平凡情况”指的是那些无法与任何其他状态区分的状态，因为它们在自身的凸分解中包含了所有其他状态。从几何上看，这些状态位于状态凸集的内部。我们可以称它们为内部状态，或完全混合状态。

    练习4.1 假设系统A的确定性状态构成一个凸集。证明状态$\rho \in \mathsf{St}_{1} ( \mathbf{A} )$是完全混合状态的充要条件是它位于该凸集的内部。

    公理2（完美可区分性）每个不是完全混合的确定性状态都可以与某个其他状态完美区分。

    注意，经典理论和量子理论都具有这一特征，如练习4.3和4.4所强调的。

    正如预期的，完美可区分性公理保证了每个非平凡系统至少有兩個完美可区分状态：

    命题4.1 在满足完美可区分性的理论中，每个物理系统至少有两个完美可区分状态，除非该系统是平凡的（即它只有一个确定性状态）。

    证明 选取一个纯态$\alpha \ \in \ \mathsf{St} ( \mathbf{A} )$。如果$\alpha$不是内部状态，那么完美可区分性保证α可以与某个其他状态$\alpha^{\prime}$完美区分，因此A有两个完美可区分状态。如果α是内部状态，那么每个純態都属于它的精炼集。此外，由于α也是纯态，即极值点，则每个其他确定性状态$\rho_{1} \in \mathsf{St}_{1} ( \mathrm{A} )$必须等于α，即A只有一个确定性状态。-

    这个结果的一个直接推论是，该理论可以描述无噪声的经典通信：如果Alice想向Bob传输一个比特串$\textbf{x} = ~ ( x_{1} , \ldots , x_{k} )$，她可以将每个比特编码为她系统的一对完美可区分状态——例如$\rho_{0}$和$\rho_{1}$——然后向Bob发送一系列处于状态$\rho_{x_{1}} \otimes \rho_{x_{2}} \otimes \cdot \cdot \cdot \otimes \rho_{x_{k}}$的相同系统。

    练习4.2 设$q_{0} ( x )$和$q_{1} ( x )$是$\{0 , 1 \}$上的两个概率分布，描述噪声经典比特的两个状态。证明这两个概率分布之间的全变差距离，定义为

    $$ \| q_{0} - q_{1} \| _{1} : = \sum_{x} | q_{0} ( x ) - q_{1} ( x ) | , $$

    的上界为

    $$ \| q_{0} - q_{1} \| _{1} \leq 2 ( 1 - 2 \epsilon ) . $$

    习题4.3 设 $p ( x )$ 和 $q ( x )$ 是有限集合 $\mathsf{X}$ 上的两个概率分布。

    1. 证明 $p ( x )$ 与 $q ( x )$ 兼容当且仅当 Supp $( p ) \supseteq \mathsf{Supp} ( q )$，其中 Supp $( f ) = \{x \in \mathsf{X} | f ( x ) \neq 0 \}$ 是函数 $f$ 的支持集。
    2. 证明 p 是完全混合的当且仅当它具有全支持集。
    3. 证明经典概率论满足完美可区分性公理（perfect discriminability axiom）。

    习题4.4 设 $\rho$ 和 $\sigma$ 是有限维希尔伯特空间 $\mathcal{H}$ 上的两个密度矩阵。

    1. 证明 $\rho$ 与 σ 兼容当且仅当 Supp $( \rho ) \supseteq \mathsf{Supp} ( \sigma )$，其中 Supp $( A ) = \mathsf{Ker} ( A )^{\perp}$ 是矩阵 A 的支持集。
    2. 证明 $\rho$ 是完全混合的当且仅当它具有满秩。
    3. 证明量子论满足完美可区分性公理。

    4.3 理想压缩#

    在平凡意义上，每个物理理论都是信息理论——物理系统在某时刻的状态只是描述了该时刻关于未来实验结果的可获取信息。使“信息理论”这一表述不那么平凡的是，在某些理论中，人们可以独立于承载信息的特定物理载体来谈论“信息”。经典理论就是如此，信息可以忠实地从报纸或DNA链转移到笔记本电脑或U盘上；量子论也是如此，至少在原则上，信息可以忠实地转移，例如从光子的偏振转移到原子的电子能级。

    并非所有物理理论都具有信息能从一个系统忠实地转移到另一个系统的性质。原则上，可以设想一些理论，其中从输入系统A到不同输出系统B的每次变换都会导致不可逆的信息丢失。让我们更精确地阐述这个想法：假设Alice有一个制备装置，该装置以某种状态 $\alpha$ 制备系统A。Alice不知道状态α，但她知道平均而言该装置制备确定性状态 $\rho \in \mathsf{St}_{1} ( \mathrm{A} )$。现在，假设Alice想将她系统的状态转移到Bob的实验室，但不幸的是她不能直接发送系统A。相反，她必须通过应用一个合适的确定性操作 $\mathcal{E}$（编码）将状态α编码成另一个系统B的状态，该操作将状态α变换成状态

    $$ \beta : = \mathcal{E} \alpha . $$

    我们称该编码对于状态 $\rho$ 是无损的，当且仅当存在另一个确定性操作（解码）使得

    $$ {\mathcal{DE}} \alpha = \alpha \qquad \forall \alpha \in F_{\rho} $$

    其中 $F_{\rho}$ 是 $\rho$ 的细化集（refinement set），由所有与 $\rho$ 兼容的状态 $\alpha$ 构成（在状态的凸集上，这相当于 $\rho$ 所属的面）。

    我们的第四个公理确立了一类特定无损编码（lossless encoding）的存在性，称为理想压缩（ideal compression）。直观上，信息压缩是将信息从较大系统编码到较小系统的任务，例如从笔记本电脑的内存到U盘。给定状态 $\rho$ 的无损压缩的终极极限，当编码系统 B 的每个状态都是 $F_{\rho}$ 中某状态的码字时达到，即当每个状态 $\beta \in{\mathsf{St}} ( {\mathbf{B}} )$ 都具有 $\mathcal{E} \alpha$ 的形式（其中 $\alpha \in F_{\rho}$）时。此时，我们称该压缩是高效的，并称三元组 (B, , ) 为理想压缩协议。

    我们的第四个公理确保此类协议始终存在：

    公理3（理想压缩） 每个状态都可以被无损且高效地压缩。

    不难看出，经典理论和量子理论都满足理想压缩公理（参见练习4.6和4.7）。此外，容易看出，除非状态 $\rho$ 是完全混合状态（completely mixed state），否则用于理想压缩的系统 B 必须严格小于系统 A：

    命题4.2 设 $( \mathbf{B} , {\mathcal{E}} , {\mathcal{D}} )$ 是状态 $\rho \in \mathsf{St}_{1} ( \mathbf{A} )$ 的理想压缩协议。若 $\rho$ 不是完全混合状态，则 $\mathrm{\Delta} \mathrm{\neq \ A}$。

    证明 容易证明其逆否命题：若 $\mathrm{B} = \mathrm{A}$，则 $\rho$ 必为完全混合状态。事实上，当 $\mathrm{B} = \mathrm{A}$ 时，映射 $\mathcal{E} : F_{\rho} \mathsf{St}_{1} ( \mathrm{A} )$ 既是单射（因为压缩是无损的），又是满射（因为压缩是高效的）。因此，我们有 $F_{\rho} \equiv \mathsf{St}_{1} ( \mathrm{A} )$，这意味着 $\rho$ 与系统 A 的所有其他状态兼容。∎

    练习4.5 设 $\mathcal{E} \in \mathsf{Transf} ( \mathrm{A {} B )}$ 是状态 $\rho \in \mathsf{St}_{1} ( \mathbf{A} )$ 的无损编码。

    1. 证明向量空间 $\mathsf{Span}_{\mathbb{R}} \{F_{\rho} \}$ 的维数必须小于或等于向量空间 $\mathsf{St}_{\mathbb{R}} ( \mathsf{B} )$ 的维数 $D_{\mathrm{B}}$。
    2. 证明当为理想压缩时，这两个维数相等。

    练习4.6 设 $p ( x )$ 是有限集合 X 上的概率分布，且 Supp $( p )$ 是 $\dot{\boldsymbol{p}}$ 的支持集。证明存在一个 $p ( x )$ 的理想压缩协议，其中编码系统的状态是集合 $\textsf{Y}$ 上的概率分布，且 $| \mathsf{Y} | = | \mathsf{Supp} ( p ) |$。

    练习4.7 设 $\rho$ 是有限维希尔伯特空间上的密度矩阵，且 $r$ 是 $\rho$ 的秩。证明存在 $\rho$ 的理想压缩协议，其中编码系统是 $r$ 维量子系统。

    4.4 三个主要原理预览#

    在本节中，我们简要预览另外三个原理——因果性（causality）、局部可区分性（local discriminability）和纯化（purification）——它们将在本书最后部分用于推导量子理论，并将在第5、6和7章中详细分析。

    4.4.1 因果性#

    在第3章中，我们描述了一个能够容纳大量物理理论的框架。这个框架如此之大，以至于我们可以找到具有相当奇特特征的理论，例如来自未来的信号或跨空间的瞬时信号。量子理论不允许此类信号传递，这是以下原理的结果。

    公理4（因果性） 制备测试结果的概率独立于在其输出端连接的观测测试的选择。

    因果性原理将电路的输入-输出顺序与信息流动的方向联系起来，将此顺序视为一个本征时间箭头，对应于未来选择不能影响现在的要求。

    因果性公理有许多后果，将在第5章中详细分析。它以一种非常特殊的方式塑造了理论的凸结构。

    因果性假设隐含地嵌入在大多数广义概率论传统工作的框架中。然而，将其识别为一个独立原则在概念上非常重要，因为有可能构建一种不遵循因果性的量子理论版本。我们将在第5章中提供一些思路；但在本书中，我们将满足于重构普通量子理论。

    图4.1 局部可区分性原则（也称局部层析成像）。爱丽丝仅通过对组分进行局部测量就能重构复合系统的状态。一个不满足该性质的世界将包含无法通过局部实验获取的全局信息。

    4.4.2 局部可区分性（Local Discriminability）#

    公理5（局部可区分性） 复合系统的任意一对状态都可以仅通过局部测量来区分。

    正如我们将在专门讨论该公理的第6章中看到的，这一陈述也等价于：复合系统的每一个状态都可以根据对组分进行局部测量的统计结果来重构，这也被称为局部层析成像公理（local tomography axiom）（参见图4.1）。如第6章所述，局部可区分性公理对于通过仅在某给定输入系统上实验来检验物理定律具有关键意义，而无需被迫考虑所有可能的系统——例如，检验宇宙中所有可能辐射模式上的麦克斯韦方程组。经典理论显然满足该原则，其中每个复合系统的状态都是两个局部测量结果的联合概率分布。在第2章中，我们已经看到该原则适用于量子理论，但它在实希尔伯特空间上的量子理论中并不成立，因为那里存在无法通过局部测量来区分的二分态。

    读者可能已经注意到，局部可区分性原则与复合的原子性（atomicity of composition）有相似之处（两者都排除了存在某些不可访问的全局信息的可能性），但这两个原则实际上截然不同。二者的逻辑独立性通过以下简单事实得到证明：实希尔伯特空间上的量子理论满足复合的原子性公理，但不满足局部可区分性公理。

    4.4.3 纯化（Purification）#

    迄今提出的五个原则定义了一系列可被视为标准的信息理论家族。如果仅凭这五个原则，世界很可能完全是经典式的。量子理论相对于所有其他理论的特殊之处在于，它始终允许一种具有最大知识的描述，并且这种描述与任何其他通常具有有限知识的描述都是兼容的。关于物理系统的知识由系统的状态提供：最大知识对应于状态为纯态，部分知识对应于状态为混合态。另一方面，部分知识也意味着我们忽略了一些系统，包括它们与被观测系统之间的关联。这意味着当部分知识通过忽略完美知识中所涉及的一些系统来获得时，它与完美知识是兼容的。换句话说，混合态描述必须是纯态描述的边际（marginal）。我们得出结论：两种知识的兼容性要求相当于假设存在一个纯态 $\Psi_{\rho}$，使得 $\rho$ 是其边际。状态 $\Psi_{\rho}$ 就是我们所说的状态 $\rho$ 的纯化（purification）。

    现在，一个关键问题是完美知识描述是否唯一：假设两位理论家被要求解释为什么系统A处于混合态 $\rho$。他们会给出相同的解释吗？在一个好的理论中，答案应该是肯定的，只是存在一些细微差别。所谓“细微差别”指的是：

    1. $\Psi_{\rho}^{\prime} : = \Psi_{\rho} \otimes \phi \in \mathsf{PurSt} ( \mathrm{AEF} )$ 是 $\rho$ 的一个纯化，其中 $\phi$ 是任意系统F的任意纯态。

    2. $\Psi_{\rho}^{\prime} : = ( \mathbb{Z}_{\mathrm{A}} \otimes \mathcal{U}_{\mathrm{E}} ) \Psi_{\rho}$ 是 $\dot{} \rho$ 的一个纯化（purification），其中 $\mathcal{U}_{\mathrm{E}}$ 是作用于环境的任意可逆变换。

    如果同一状态的任意两个纯化都只有这些无关紧要的差异，则称该纯化本质上是唯一的。纯化公理表述如下：

    公理6（纯化）每个状态都有一个本质唯一的纯化。

    纯化排除了经典概率论，因为不可能将变量 X 的任意概率分布表示为两个变量 X 和 Y 的概率分布的边缘分布，且该分布集中在单一值 $( x_{0} , y_{0} ) \in \mathrm{X} \times \mathrm{Y}$ 上。事实上，纯化不仅排除了经典理论，还为大多数非经典特征和量子理论的优势提供了种子。第7章将推导出大量量子特征，我们将在那里深入阐述纯化原理的意义。

    4.5 小结#

    本章我们鸟瞰了用于重构量子理论的六条原理。五条原理——因果性、局部可区分性、复合的原子性、完美可区分性和理想压缩——定义了信息理论的标准类，其中既包含量子理论也包含经典概率论。第六条原理——纯化——通过要求对物理系统制备的无知总是可以通过将环境纳入描述而以一种本质唯一的方式得到解释，从而引入了信息的量子本质。对于有限系统（其状态由有限个结果概率决定的系统），上述六条原理等价于量子理论。在第2章中，我们将这些原理作为定理从理论中推导出来；在本书其余部分，我们将从这六条原理推导出量子理论。复希尔伯特空间、叠加原理、海森堡不确定关系、纠缠、不可克隆、隐形传态、贝尔不等式的违反、量子密码学——每一个量子特征都已经压缩在这六条原理之中；本书的其余部分将致力于将其解压缩。

    笔记#

    关于复合的原子性 在我们的公理体系中，复合的原子性等价于 Wilce (2010) 提出的纯条件化公理（pure conditioning）。纯条件化要求：当一个两体系统处于纯态时，对其中一部分进行的原子测量会在另一部分上诱导出纯态。

    关于理想压缩与信息论中的通常压缩 值得注意的是，第4.3节中理想压缩的概念不同于经典信息中香农压缩和量子信息中舒马赫压缩所使用的概念。区别有二：首先，在我们的情况中，我们考虑信源的单次使用（即 Alice 制备设备的单次使用），而香农和舒马赫定理指的是同一信源被大量使用的渐近场景。其次，我们要求压缩协议完全无损耗（即零误差），而香农和舒马赫压缩的框架允许存在误差的压缩协议，只要当信源使用次数趋于无穷时，误差趋近于零。
