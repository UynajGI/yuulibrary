    纯化原理#

    是什么让量子理论如此特别？每个人对此都有自己的看法：有人会提到纠缠、非局域性、互补性、不确定性,以及许多其他特征。从表面意义上说,每种回答都是合理的：量子理论与我们所熟悉的其他理论不同的每一个特征都可以被视为其独特特征,因为这让你能够将量子理论与已知理论区分开来。但这并不会让你走得很远。真正的问题不在于如何将量子理论与你已经知道的理论区分开来,而在于如何将其与你可能想象的所有理论区分开来。大多数特征在这方面并不擅长——例如,物理学家已经发明了许多展示纠缠、非局域性和互补性但仍然不是量子理论的玩具理论。在本章中,我们将提出关于什么使量子理论变得特别这个问题的答案：纯化。接下来,我们将探讨量子理论的一些最基本事实,如纠缠和隐形传态,将纯化与可追溯至薛定谔和冯·诺依曼的深层思想脉络联系起来。本章呈现的结果也是如何从基本原理出发推理量子理论的第一个例子。

    7.1 一个独特且基本的特征#

    纯化原理规定,当你对系统A的状态无知时,你总可以声称你的无知源于这样一个事实：A是更大系统AB的一部分,而你对这个更大系统有完全的知识。当你这样做时,你必须赋予复合系统AB的纯态由A的状态以本质上唯一的方式决定。

    混合态的纯化是一个独特的特征——当然,不是我们在日常生活中体验到的。如果你连A本身都不知道,怎么能声称你知道A和B呢？这一反直觉的特征在量子理论的早期就已经被注意到,当时埃尔温·薛定谔曾著名地写道："表达这种特殊情况的另一种方式是：对整体的最佳可能认知并不必然包括对其所有部分的最佳可能认知。"1 在同一篇论文中,他还写道："我不会称之为一个特征,而宁可是量子力学的典型特征,是迫使其完全背离经典思维路线的特征。"

    这是一个大胆的说法,如果你想到这是在1935年提出的！然而在今天,有大量证据支持这一观点。这里有三个很好的论据：首先,我们知道纯化与五个相当基本的原则相结合,在所有可以想象的理论中筛选出了量子理论。其次,物理学家一直试图制造出展示类量子特征但又不是量子理论的玩具理论。虽然他们在许多特征上取得了成功,但迄今为止纯化仍未被攻克：如果你想纯化混合态,那么量子理论似乎是唯一选择,最多只允许最小程度的变体。2 第三,量子理论的许多重要特征可以直接从纯化推导出来,而无需先推导出量子理论。这一事实强烈表明,在隔离量子理论特有内容方面,纯化正好切中要害。

    混合态的纯化是量子理论特有的现象。但我们为何要将其视为自然的基本原理？初看之下，这似乎是个古怪的特性——它必须如此，因为量子理论本身就很奇特，若将其浓缩成一条原理，这条原理自然也会显得怪异。然而细想之下，我们会意识到纯化是一个基本要求：本质上它是连接物理学与信息论的桥梁。若没有概率与混合态的概念，信息论便毫无意义——信息的全部意义就在于存在我们预先未知的事物。但在牛顿与拉普拉斯的经典物理学世界中，一切事件都是确定的，根本层面上没有信息存在的空间。原则上，抛硬币或玩概率游戏毫无意义，因为结果早已注定，只要拥有足够的技术和计算能力，总能被预测。反观纯化原理告诉我们，"无知是物理的"。每个混合态都可以通过一个可靠流程一次性生成：将两个系统制备成纯态，然后丢弃其中一个。这个流程的结果是，剩余系统成为我们无知的物理化身。这一讨论表明，只有当纯化原理成立时，信息才可能成为物理学中的基本要素。

    在此我们强调了“信息”和“无知”的作用。但如果你对这些主观概念侵入物理学殿堂感到困扰，完全可以将其替换为“随机性”，并认为这里讨论的核心在于随机性是否存在于基础层面。事实上，一个存在随机性的世界是一个混合态的世界（如果你有一个随机数生成器，总能用它来制备混合态）。在这样的世界里，自然会提出疑问：“为什么我们会遇到混合态？”从实在论者的角度看，如果一个混合态无法纯化，那么实际上系统完全有可能处于纯态——只不过我们恰好不知道这个纯态而已。因此，若要确保在最基础层面上某些态确实是混合的，纯化就成为必要条件。

    在纯化与混合性的关系中，存在一种有趣的立场反转：纯化使我们能够从描述中消除混合性，从而用扩大系统的纯态取代混合态。不仅如此，纯化还允许我们将所有物理过程替换为作用于更大系统上的可逆过程。让我们明确阐述其含义。假设系统 (A) 经历了一个不可逆过程，转变为系统 (A')。那么纯化告诉我们，可以通过以下步骤模拟该过程：

    纯化与可逆模拟#

    1. 制备处于纯态 (\beta) 的辅助系统 (B)。

    2. 通过可逆变换共同演化系统 (A) 和 (B)，使之转变为系统 (A') 和 (B')。

    3. 丢弃系统 (B')。

    纯化与可逆模拟蕴含着深远的意义：

    1. 信息的守恒。在这个模拟中，丢弃系统 B 是整个过程里唯一的不可逆操作——如果我们保留系统 B，整个演化就是可逆的。如果你相信在基础层面上信息不能被摧毁，那这是个好消息。在专家们看来，这种信念被称为信息的守恒，其有效性通常被视为物理学最基础的定律之一。

    2. 可逆计算。信息的守恒不仅对理论物理学家很重要，对计算机科学家也是如此。总之，计算是一个物理过程，它将输入数据转化为输出数据。一般来说，这个过程不一定是可逆的——例如，考虑函数 (f(x) = x^{2}) 的计算。但是，是否可能仅使用可逆门来实现所有计算？贝内特（Bennett）在20世纪70年代提出了这个问题，并给出了肯定的回答：通过在过程的描述中添加一个系统，经典理论中可实现的所有确定性计算都可以仅使用可逆门来完成。现在，我们知道这在量子理论中也成立，但很容易构造出一些玩具理论，其中某些计算无法以可逆方式进行3。

    3. 随机算法的模拟。纯粹且可逆的模拟确保了概率计算也可以在不从外部泵入熵的情况下实现——也就是说，无需将系统 B 制备在混合态中。这意味着，即使是一个随机算法或蒙特卡洛模拟，也可以在不需要外部随机数生成器的情况下运行，仅从纯态开始。

    以上三点提供了充分的理由，要求将纯粹且可逆的模拟能力作为物理过程的一个基本性质。既然纯化（purification）作为副产品提供了这一点，我们至少有三个充分的理由为此感到高兴。

    但是，为了获得纯粹且可逆的模拟，我们是否必须要有纯化？答案是“是”，因为状态的制备是物理过程的一个特例——一个没有输入的过程。因此，如果你希望纯粹且可逆的模拟能力对所有过程都成立，那么你就需要将纯化作为一个特例包含在内。

    接下来，我们将更深入地探讨纯化的后果，初步说明如何基于第一原理的高层推理重建关键的量子特征。

    7.2 纯化原理#

    以下是纯化原理的精确陈述：

    纯化公理 对于每个系统 A 和每个状态 (\rho \in{\mathsf{St}} ( {\mathsf{A}} ))，都存在一个系统 B 和一个纯态 (\Psi \in \mathsf{PurSt} ( \mathrm{AB} ))，使得

    [ \underbrace{\langle \rho \ \rangle^{\mathrm{~ \tiny ~ A }}}_{\mathrm{{ \tiny ~ \left( ~ \frac ~ {~ \scriptstyle ~ \mathfrak ~ {~ B } } ~ \right)} { \scriptstyle ~ \mathfrak ~ { C ~} } }} = \underbrace{\langle \Psi \ \rangle_{\mathrm{{ \tiny ~ B } }}^{\mathrm{ \tiny ~ A }}}_{\mathrm{{ \tiny ~ \left( ~ \frac ~ { \scriptstyle ~ B } ~ \right)} { \scriptstyle ~ \mathfrak ~ { C ~} ~} ~ ~}} . ]

    如果两个纯态 (\Psi) 和 (\Psi^{\prime}) 满足

    [ \begin{array}{r} {\overbrace{\vphantom{\biggl |} \Psi^{\prime} \begin{array}{l} {\mathrm{~ {\cal{B}} }} \ {\mathrm{ {\cal{B}} }} \end{array} \biggr [ \ e )}^{\mathrm{ {\cal{A}} }} = \overbrace{\vphantom{\biggl |} \Psi \begin{array}{l} {\mathrm{ {\cal{B}} }} \ {\mathrm{ {\cal{B}} }} \ {\mathrm{ {\cal{B}} }} \end{array} \biggr [ \ e )}^{\mathrm{ {\cal{A}} ~}} \quad ,} \end{array} ]

    则存在一个仅作用于系统 B 的可逆变换 (u)，使得

    [ \begin{array}{l} \overbrace{{\underbrace{\Psi^{\prime}} \mathbf{\Psi} \underbrace{\mathbf{B}}}}^{\mathrm{\textbf{A}}} = \overbrace{{\underbrace{\Psi \mathbf{\Psi} \underbrace{\mathbf{B}}_{\mathrm{\textbf{\textbf{B}}} \overbrace{\mathcal{U}} \mathbf{\Psi} \mathbf{B}}}}^{\mathrm{\textbf{A}}}} \end{array} ]

    这里我们说 $\Psi$ 是 $\dot{} \rho$ 的一个纯化，B 是纯化系统。非正式地，式 (7.1) 保证你总能找到 AB 的一个纯态，它与仅对 A 的有限知识兼容。在此基础上，式 (7.2) 指出，与对 A 的知识兼容的 AB 的所有状态本质上相同，最多相差 B 上的一个可逆变换。我们将此性质称为纯化的唯一性。注意式 (7.2) 中的两个纯化具有相同的纯化系统。很容易将该陈述推广到纯化系统不同的情况：

    命题 7.1 如果两个纯态 $\Psi \in \mathsf{PurSt} ( \mathrm{AB} )$ 和 $\Psi^{\prime} \in \mathsf{PurSt} ( \mathrm{AB}^{\prime} )$ 是同一混合态的纯化，那么

    $$ {\widehat{\underbrace{\left( \Psi^{\prime} \right) \mathbf{B^{\prime}}}_{\mathbf{B^{\prime}}}}} = {\widehat{\underbrace{\left( \Psi \mathbf{\otimes} \right) \mathbf{B} \mathbf{\otimes} \left[ {\mathcal{C}} \right] \mathbf{B^{\prime}}}_{\mathbf{B} \mathbf{\widetilde{C}} \mathbf{\Big ]} {\widehat{\mathbf{B^{\prime}}}}}}} $$

    对于某个将系统 B 变换为系统 $\mathbf{B}^{\prime}$ 的确定性变换成立。

    证明 选取两个纯态 $\beta \in \mathsf{PurSt} ( \mathrm{B} )$ 和 $\beta^{\prime} \in \mathsf{PurSt} ( \mathrm{B}^{\prime} )$。由于 $\Psi \otimes \beta^{\prime}$ 和 $\Psi^{\prime} \otimes \beta$ 是 A 上同一状态的纯化，由纯化的唯一性可得

    $$ {\begin{array}{r} {\underbrace{{\Biggl (} \Psi^{\prime} {\Biggr )} {\frac{\mathbf{A}} {\mathbf{B}^{\prime}}}}_{\begin{array}{l} {\mathbf{(} \mathbf{{\boldsymbol{\beta}}}^{\prime} ]^{\begin{array}{l} {\mathbf{B}} \end{array}} \end{array} = \underbrace{\overbrace{\boldsymbol{\Psi} \left[ {\begin{array}{l} {\mathbf{\sigma}} \\ {\mathbf{B}} \end{array} \right]^{\mathbf{B}}}_{\begin{array}{l} {\left(} \\ {{\boldsymbol{\beta}}^{\prime} \right)^{\mathbf{\beta}^{\prime}}} \end{array} {\Biggl |} {\mathcal{U}} \underbrace{{\mathbf{B}}^{\prime}}_{\begin{array}{l} {\mathbf{B}} \end{array}} \end{array}} $$

    对于某个可逆变换成立（我们在此也假设了局部可区分性）。两侧丢弃系统 B，我们得到 $\Psi^{\prime} = ( \mathbb{Z}_{\mathrm{A}} \otimes \mathcal{C} ) \Psi$，其中 C 是由下式定义的确定性变换

    $$ \begin{array}{r} \frac{\textbf{\textsf{B}} \displaystyle \prod \mathcal{C} \Big ]^{\textbf{\textsf{B^{\prime}}}} : = \frac{\textbf{\textsf{B}}} {\displaystyle \left( \beta^{\prime} \right)^{\textbf{\textsf{B^{\prime}}}}} \Bigg \downarrow \mathcal{U} \Bigg ]_{\textbf{\textsf{B}} \displaystyle \left[ \textbf{\textsf{e}} \right)}^{\textbf{\textsf{B^{\prime}}}} .} \end{array} $$

    这就是纯化所需了解的全部内容——现在让我们开始推导其推论。

    7.3 纠缠#

    纯化最明显的推论就是纠缠。

    命题 7.2（纠缠态的存在性）混合态的纯化是一个纠缠态。

    证明 设 $\Psi$ 是 $\rho$ 的一个纯化。如果 $\Psi$ 不是纠缠的，根据定义意味着 $\Psi$ 是两个纯态的直积态。因此，$\rho$ 必须是纯态。由逆否命题，如果 $\rho$ 是混合态，那么唯一的可能性就是 $\Psi$ 是纠缠态。∎

    每当我们纯化一个混合态时，我们就会得到一个纠缠态。有趣的是，其逆命题也成立：

    命题 7.3 纯纠缠态的约化态是混合态。

    证明 设 $\Psi$ 是 AB 的一个纯态，并且其在系统 A 上的约化态是纯的——称其为 α（约化态的存在性隐含了因果性）。那么，对于每一个纯态 $\beta$，直积态 $\Psi' = \alpha \otimes \beta$ 将是 α 的一个纯化。纯化的唯一性，如式 (7.2) 所述，意味着存在一个仅作用于 B 的可逆变换，使得 $\Psi = \alpha \otimes \mathcal{U} \beta$。这意味着 $\Psi$ 是一个直积态。因此，如果一个纯态是纠缠的，那么它的约化态必定是混合态。∎

    总结一下，我们证明了：在一个满足我们原理的理论中，一个态是混合态的充要条件是其纯化是纠缠态。根据这一观察，唯一满足纯化且没有纠缠的理论是那些根本不存在混合态的理论。在这些理论中，没有事件可以是随机的，因为随机事件可以用来生成混合态。换句话说，对于一个因果理论，我们证明了以下蕴含关系：“纯化 + 无纠缠 ⇒ 决定论”。这里这主要是一个趣闻，因为本书将重点关注并非所有结果都预先确定的概率论理论。在这些理论中，纯化意味着存在纠缠。

    7.4 可逆变换与旋转#

    纯化不仅意味着我们的理论中存在纠缠态，而且还意味着存在“足够多的可逆变换”。例如，有以下结果。

    命题 7.4 对于一般系统 B 的每一对归一化纯态 $\psi$ 和 $\psi'$，必定存在一个可逆变换 $\mathcal{U}$，使得

    $$ \begin{array}{r} {\boxed{\psi'}^{\mathrm{~ \textsc ~ {~ B ~} ~}} = \boxed{\psi}^{\mathrm{~ \textsc ~ {~ B ~} ~}} \boxed{\mathcal{U}}^{\mathrm{~ \textsc ~ {~ B ~} ~}} .} \end{array} $$

    证明 这是由方程(7.2)表述的纯化唯一性的简单推论：如果我们从图中擦除系统A（数学上，将其设为平凡系统I），则唯一性条件给出 $\mathfrak{e} \mathfrak{e}_{\mathrm{i} \mathrm{f}} \mathfrak{\mathrm{\Pi}}_{\mathrm{I}} ( e | \psi^{\prime} ) = \mathfrak{\mathrm{\Pi}}_{\mathrm{I}} ( e | \psi )$ ，那么存在一个可逆变换使得 $\psi^{\prime} = \mathcal{U} \psi$ 。

    通过可逆变换将任意纯态转变为另一个纯态的能力被称为传递性（transitivity），这意味着可逆变换集合的作用在纯态集合上是传递的。

    传递性与纠缠的存在性相结合，直接引出了纠缠门的存在性，即能够将乘积态转变为纠缠态的可逆门。传递性的另一个结果是，每个物理系统都有一个唯一的最大混合态，即唯一在所有可逆变换作用下保持不变的态。为了理解这一点，我们需要对作用于给定物理系统上的可逆变换的结构稍作探讨。首先，注意到作用于给定系统上的可逆变换构成一个群，即：

    1. 恒等变换是一个可逆变换；
    2. 可逆变换的逆变换是一个可逆变换；
    3. 两个可逆变换的复合是一个可逆变换。

    对于系统A，我们将作用于A上的可逆变换群记为 $\mathbf{G}_{\mathrm{A}}$ 。当A是有限维时，可以证明 $\mathbf{G}_{\mathrm{A}}$ 是一个紧致群。我们将这一事实的证明推迟到第7.12节，因为一旦掌握了适当的方法，证明就变得很容易。目前，我们仅用 $\mathbf{G}_{\mathrm{A}}$ 的紧致性来构造一个特殊的信道，称为旋转（twirling）信道。旋转信道 $\mathcal{T}$ 定义为

    $$ \mathcal{T} : = \int_{\mathbf{G}_{\mathrm{A}}} \mathrm{d} \mathcal{U} \mathcal{U} , $$

    其中 $\mathrm{d}$ 是 $\mathbf{G}_{\mathrm{A}}$ 上的归一化哈耳测度（Haar measure），积分遍及整个群。假设局部可区分性，很容易看出这里定义的线性映射是一个信道：实际上，由于A上的变换形成一个有限维空间， $\mathbf{G}_{\mathrm{A}}$ 中可逆变换生成的凸集是一个有限维凸集，并且由于映射 $\mathcal{T}$ 在这个凸集内部，由卡拉泰奥多里（Carathéodory）定理可知，可以将其写为有限凸组合的形式

    $$ \mathcal{T} : = \sum_{x \in \mathsf{X}} p_{x} \mathcal{U}_{x} . $$

    显然，信道的凸组合本身也是一个信道。

    注意，旋转信道的作用在与其复合可逆变换后保持不变：根据 $\tau$ 的定义以及哈耳测度的不变性，我们有

    $$ \mathcal{WTV} = \mathcal{T} \qquad \forall \mathcal{V} , \mathcal{W} \in \mathbf{G}_{\mathrm{A}} . $$

    这一事实立即引导我们得出结论：存在唯一的不变态。一方面，旋转的输出必须是不变态，因为

    $$ \left( \mathcal T \rho \right) = \mathcal W \left( \mathcal T \rho \right) , \qquad \forall \mathcal W \in{\bf G}_{\mathrm{A}} , \forall \rho \in{\bf S t} ( \mathrm{A} ) . $$

    这意味着至少存在一个不变态，比如说对于固定态 $\rho_{0}$ ，

    $$ \chi : = \mathcal{T} \rho_{0} $$

    另一方面，很容易看出上述定义与 $\rho_{0}$ 的选择无关，而且事实上，只能存在一个不变态。证明是一个简单的练习：

    练习 7.1 证明：

    1. 假设因果性，对于每一对态 $\rho$ 和 $\rho^{\prime}$ ，有 $\tau_{\rho} \propto \tau_{\rho^{\prime} ;}$
    2. 如果 $\chi^{\prime}$ 是一个不变态，它必须满足 $\tau \chi^{\prime} = \chi^{\prime}$

    利用以上两点，得出结论：存在唯一的不变态，因此

    $$ {\frac{\operatorname{A}} {\hbar}} \left[ {\mathcal{T}} \right]^{\operatorname{A}} = {\frac{\operatorname{A}} {\hbar}} {\overline{{\big [}}} {\mathcal{C}} {\big )} {\overline{{\big (}}} {\big )}^{\operatorname{A}} . $$

    7.5 导引#

    假设你有一个制备设备，它以概率$p_{x}$制备系统A处于态$\alpha_{x}$。如果你忽略x的值，这个过程会导致混合态的制备

    $$ \rho = \sum_{x \in \mathsf{X}} p_{x} \alpha_{x} , $$

    其中X是x可能取值的集合。现在，假设你并非忽略x，而是将其编码在另一个物理系统（称之为C）的态中，使得任何测量这个系统的人都能找出x的值。这可以通过将系统C制备在态$\gamma_{x}$来实现，且这些态$\{\gamma_{x} \mid x \in \mathsf{X} \}$是完美可区分的。这个操作的结果是，复合系统AC处于态

    $$ \sigma = \sum_{x \in \mathsf{X}} p_{x} \alpha_{x} \otimes \gamma_{x} , $$

    （这可能是由完美可区分性公理保证的）。态$\sigma$有两个重要性质：首先，它是$\rho$的扩展，即

    $$ \sqrt{\sigma \frac{\mathrm{~ A ~}} {\mathrm{~ c ~}}} = \frac{\widehat{\rho \mathrm{~ \biggr ~ \biggr ~ \biggr ~ \wedge ~}}} {\mathrm{~ \cdot ~}} . $$

    其次，如果用区分态$\{\gamma_{x} \mid x \in \mathsf{X} \}$的测试来测量系统C，就可以模拟系统A的原始制备设备：实际上，我们有

    $$ \begin{array}{r} \boxed{\sigma \underbrace{\vphantom{\sigma \smash{\sigma \smash{\sigma}}} \mathbf{\check{A}}}_{\mathrm{~ \tiny ~ \left( ~ C ~ \right)}_{x}} = p_{x} \underbrace{( \alpha_{x} \mathbf{\Theta} )^{\mathrm{~ A ~}}}_{\mathrm{~ \tiny ~ \left( ~ C ~ \right)}_{x}} \qquad \forall x \in \mathsf{X} ,} \end{array} $$

    其中$\mathbf{c} : = \{c_{x} \}$是区分态$\{\gamma_{x} \mid x \in \mathsf{X} \}$的观察测试。这是一个有趣的技巧，因为它允许我们将随机纯态的制备替换为更大系统$\mathrm{AC}$的一个单一态的制备，随后对C进行测量。

    显然，这里展示的技巧对$\rho$的每个系综分解都有效：给定一个系综分解，我们总能找到合适的系统C、一个AC的态以及C上的一个测量，使得式(7.11)成立。但我们能否找到一个对所有系综都有效的扩展？感谢纯化，答案是肯定的：

    证明 对于每个系综$\{p_{x} \alpha_{x} \}$，按式(7.9)构造一个扩展$\sigma \in{\mathsf{St}} ( {\mathsf{AC}} )$，并取其纯化，记为$\Psi^{\prime} \in \mathsf{PurSt} ( \mathrm{ACD} )$。由于$\Psi^{\prime}$和$\Psi$是$\rho$的两个纯化，

    7.6 过程层析#

    纯化与局域可区分性一起，建立了变换与状态之间的一种有趣对应关系。这一点很容易看出：取一组张满系统A整个状态空间的状态$\{\alpha_{x} | x \in \mathsf{X} \}$和一组正概率$\{p_{x} \}_{x \in \mathsf{X}}$。然后，取混合态$\begin{array}{r} {\rho = \sum_{x} p_{x} \alpha_{x} - \mathrm{say} \Psi \in \mathsf{PurSt} ( \mathrm{AB} )} \end$的一个纯化。现在，如果两个变换$\mathcal{A}$和$\mathcal{A}^{\prime}$满足

    $$ \left( \Psi \ {\frac{\mathrm{\bf ~ A}} {\mathrm{\bf ~ B}}}^{\mathrm{\bf ~ A}} \overbrace{\mathrm{\bf ~ A}}^{\mathrm{\bf ~ A}^{\prime}} - \right) = \left( \Psi \ {\frac{\mathrm{\bf ~ A}} {\mathrm{\bf ~ B}}}^{\mathrm{\bf ~ A}} {\frac{\mathrm{\bf ~ A}^{\prime}} {\mathrm{\bf ~ A}^{\prime}}}^{\mathrm{\bf ~ A}^{\prime}} \right. , $$

    显然$\mathcal{A}$必须等于$\mathcal{A}^{\prime}$，即对应关系${\mathcal{A}} \mapsto ( {\mathcal{A}} \otimes{\mathcal{T}}_{\mathrm{B}} ) \Psi$是单射的。

    实际上，利用式(7.12)的操纵性质，我们得到

    $$ ( \alpha_{x} )^{\mathrm{~ \tiny ~ A ~}} \boxed{\mathcal{A}}^{\mathrm{~ \tiny ~ A ~ ^{\prime} ~}} = ( \alpha_{x} )^{\mathrm{~ \tiny ~ A ~}} \boxed{\mathcal{A}^{\prime}}^{\mathrm{~ \tiny ~ A^{\prime} ~}} \qquad \forall x \in \mathsf{X} , $$

    $$ \begin{array}{r} {\frac{\mathrm{~ {\cal ~ A} ~}} {\mathrm{~ {\cal ~ A} ~}} \frac{\mathrm{~ {\cal ~ A} ~}^{\prime}} {\mathrm{~ {\cal ~ A} ~}^{\prime}} = \frac{\mathrm{~ {\cal ~ A} ~}} {\mathrm{~ {\cal ~ A} ~}^{\prime}} \Bigl [ \frac{\mathrm{~ {\cal ~ A} ~}^{\prime}} {\mathrm{~ {\cal ~ A} ~}^{\prime}} .} \end{array} $$

    这一结果为我们提供了一种识别未知变换的步骤：

    1. 准备纠缠态 $\Psi$；
    2. 对系统 A 施加未知变换；
    3. 对输出态 $( \mathcal{A} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi$ 进行完整测量。

    过程层析本身并不神奇：如果你只需要一个单射对应关系，并不需要纯化——相反，你可以直接取一个关联态，比如方程(7.9)中的态 $\sigma$。这里的关键在于，态 $\Psi$ 是纯的，这一特征直接导致了惊人的结果，比如无干扰即无信息原理。

    7.7 无干扰即无信息原理#

    假设你想提取关于系统 A 的某些信息，同时又想“不可见”，即不留下任何操作痕迹。这类似于你想窃听 Alice 和 Bob 之间的秘密通信，而不让他们察觉你在偷听。从数学上讲，你的检验 $\{\mathcal{A}_{x} \}$ 不引入任何干扰的条件是：

    $$ \sum_{x} {\mathcal{A}}_{x} = {\mathcal{T}}_{\mathrm{A}} , $$

    这意味着，对于那些忽略检验结果的人来说，你的检验与系统 A 上的恒等操作是无法区分的。

    在熟悉的经典世界中，满足无干扰条件很容易。例如，我们可以在不引入任何可探测变化的情况下，找出硬币落地时是正面还是反面。但在满足纯化原理的世界中，情况截然不同。如果我们取一个可用于过程层析成像的纯态 $\Psi ~ \in ~ \mathsf{PurSt} ( \mathrm{AB} )$，那么无干扰条件意味着 $\begin{array}{r} {\sum_{x} ( \mathcal{A}_{x} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi = \Psi} \end$。但由于 $\Psi$ 是纯态：因此每个未归一化态 $( {\mathcal{A}}_{x} \otimes{\mathcal{T}}_{\mathrm{B}} ) \Psi$ 必须与 $\Psi$ 成正比。精确地说，必然存在一组概率 $\{p_{x} \}$，使得 $( \mathcal{A}_{x} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi \ : = \ : p_{x} \Psi$。由于映射 ${\mathcal{A}} \mapsto ( {\mathcal{A}} \otimes{\mathcal{T}}_{\mathrm{B}} ) \Psi$ 是单射（见第7.6节），我们得出结论：$\mathcal{A}_{x} = p_{x} \mathcal{T}_{A}$。换句话说，我们的检验等价于以下过程：

    1. 在系统 A 上不做任何操作；#

    1. 以概率 $p_{x}$ 随机输出一个结果 $x$

    显然，这意味着我们的检验根本没有提取任何信息。总之，我们证明了“无干扰 ⇒ 无信息”这一推论：在满足纯化原理的世界中，不可能在不留下痕迹的情况下提取信息。这一事实对密码学具有重大意义：当两方通信时，每一次获取他们消息信息的尝试都会导致干扰，而原则上他们可以探测到这种干扰。

    7.8 量子隐形传态#

    纯化原理的另一个重要结果是量子隐形传态。量子隐形传态的任务本质上类似于传真：利用经典传输线路以及发送方和接收方之间预先建立的一些关联，将数据从发送方传输到接收方。然而，传输一个与另一系统纠缠的系统中携带的数据，似乎并非易事。

    假设我们想要传输系统 A 携带的数据，并且接收方拥有一份系统 A 的副本，该副本与发送方实验室中的另一个系统 B 处于纠缠态。量子隐形传态协议具有以下一般结构：

    1. 发送方测量输入系统 A 和系统 B，得到一个结果 x。
    2. 结果 x 通过经典传输线路传送给接收方。
    3. 根据结果 x，接收方在输出系统上执行一个操作。

    如果在完成这三个步骤后，输入系统的状态被传输到了输出系统，则该协议有效。让我们看看如何仅从第一原理出发构建一个可行的量子隐形传态协议。

    对于发送方和接收方共享的纠缠态，我们选择系统 A 的稳态 $\chi$ 的一个纯化——称之为 $\Phi \in \mathsf{PurSt} ( \mathrm{AB} )$。我们还选择一个可逆变换的系综 $\{p_{x} \mathcal{U}_{x} \}$，其平均值等于双向绕道信道，即 $\begin{array}{r} {\sum_{x} p_{x} \mathcal{U}_{x} = \mathcal{T}} \end$。然后，在假设局域可区分性的条件下，由式 (7.8) 我们得到

    $$ \begin{array}{r l r} {{\sum_{x \in{\mathsf{X}}} p_{x} ( \mathcal{U}_{x} \otimes \mathcal{T}_{\mathrm{B}} ) \Phi = ( \mathcal{T} \otimes \mathcal{T}_{\mathrm{B}} ) \Phi}} \\ & {} & {= \chi \otimes \beta ,} \end{array} $$

    其中 $\beta$ 是 $\Phi$ 在系统 B 上的边际态。现在，注意，根据定义

    $$ \frac{\left( \Phi \right)_{{\bf{\bar{\Phi}}}}^{{\bf{A}}} ( {\bf{\bar{\Phi}}} )} {{\bf{\bar{\Phi}}}^{\prime}} = \underbrace{\langle{{\bf{\bar{\Phi}}}} \rangle_{{\bf{\bar{\Phi}}}}^{{\bf{A}}}}_{{\bf{\Phi}} ( {\bf{\bar{\Phi}}} ) {\bf{\Phi}}^{{\bf{B}}}} , $$

    由于在假定局部可区分性（见第 7.6 节）下，映射 ${\mathcal{A}} \mapsto ( {\mathcal{A}} \otimes{\mathcal{T}}_{\mathrm{B}} ) \Phi$ 是单射，我们得出结论

    $$ \begin{array}{l} \begin{array}{r l} & {\underbrace{\vphantom{\biggl (} \Phi \qquad \frac{\mathrm{~ \scriptstyle ~ A ~}} {\mathrm{~ \scriptstyle ~ A ~}}}_{\mathrm{~ \scriptstyle ~ A ~}}} \\ & {\qquad \frac{\mathrm{~ \scriptstyle ~ A ~}} {\mathrm{~ \scriptstyle ~ A ~}}} \end{array} \biggr [ \begin{array}{l} {B_{x}} \\ {B_{x}} \\ {B_{x}} \end{array} \biggr ] = p_{x} \ \frac{\mathrm{~ \scriptstyle ~ A ~}} {\mathrm{~ \scriptstyle ~ A ~}} \biggl [ \mathcal{U}_{x} \biggr ] \biggr array \qquad \forall x \in \mathsf{X} . \end{array} $$

    $$ \frac{_{\mathrm{~ {\scriptstyle ~ A} ~}}} {\mathrm{~ {\scriptstyle ~ A} ~}} [ \mathcal{C} ]^{\mathrm{~ {\scriptstyle ~ A} ~}^{\prime}} = \frac{( \eta ]^{\mathrm{~ {\scriptstyle ~ E} ~}}} {\mathrm{~ {\scriptstyle ~ A} ~}} \sqrt{\mathcal{U}} \frac{_{\mathrm{~ {\scriptstyle ~ E} ~}^{\prime}} [ \ e )} {_{\mathrm{~ {\scriptstyle ~ A} ~}^{\prime}}} , $$

    其中 η 是 E 的一个纯态，而 是一个将系统 AE 转换为 $\mathrm{A^{\prime} E^{\prime}}$ 的可逆变换。

    7.10 移动冯·诺依曼截断#

    在前一节中，我们考虑了确定性过程。现在我们转而关注可能产生随机结果的检验，证明一般检验的引导（steering）的类似结论。

    假设一个实验者对系统 A 执行一个检验，记为 $\{\mathcal{A}_{x} \mid x \in \mathsf{X} \}$。当结果 x 发生时，实验者会知道它，例如因为设备在显示器上打印了它。一个自然的问题是："显示器也是一个物理系统，为什么它不像理论中其他系统那样被描述？" 显而易见的答案是，如果我们想，我们总是可以把它包括在描述中：我们可以通过确定性过程描述提供关于结果 x 信息的设备

    $$ \mathcal{C} : = \sum_{x \in \mathsf{X}} \mathcal{A}_{x} \otimes \beta_{x} , $$

    其中 $\{\beta_{x} \}$ 是一组可完美区分的状态，对应一个携带结果信息的合适系统 B。非正式地，我们在此将系统 B 称为“显示器”。那么，“查看显示器”意味着执行区分状态 $\{\beta_{x} \}$ 的检验。当实验者执行这样一个检验——记作 $\{b_{x} \}$——她将恢复原始的检验：

    $$ \begin{array}{r} {\frac{\mathrm{~ ~ {~ A ~} ~} [ \begin{array}{l} {\boldsymbol{A}_{x}} \end{array} ]^{\mathrm{~ \tiny ~ {~ A ~} ~}}} {\mathrm{~ ~ {~ ( ~ \frac ~ {~ \xi ~} {~ B ~} ~ [ ~ \boldsymbol{~ b}_{x} ~} \end{array} ) ~}} , \quad \forall x \in \mathsf{X} .} \end{array} $$

    这个方程并无神秘之处，它只是将检验 $\{\mathcal{A}_{x} \}$ 表示为实验者对显示器进行测量的结果的等价方式。然而，有趣的是，我们可以对该过程进行纯化且可逆的模拟，并将我们的检验视为被测系统 A、显示器 B 以及可能的环境 E 之间可逆相互作用的结果。用公式表示：

    $$ \begin{array}{r l} {\frac{\textsc{a}} {\textsc{a}} \boxed{A_{x}} \frac{\textsc{a}} {\textsc{a}} =} & {\sqrt{\eta \frac{\textsc{a}} {\textsc{\textbf{E}}}} \mathcal{U} \frac{\textsc{a}} {\textsc{b} \textsc{\textbf{E}}} \boxed{\begin{array}{l} {\textbf{\textsc{b}}} \\ {\textbf{\textsc{b}}} \end{array} \quad \forall x \in \mathsf{X} ,} \end{array} $$

    其中 E 是一个合适的系统，η 是一个纯态，而 是一个可逆变换。这一事实的证明留给你作为练习：

    练习 7.6 证明式 (7.18) 并将其推广到输入和输出系统不同的检验。[提示：使用练习 7.5 的结果。]

    描述中包含的物理系统与省略的系统之间的分界被称为冯·诺伊曼分界。一般来说，分界可以有不同的方式：我们可以想象有光子从显示器进入实验者的眼睛，并且我们可以再次将它们包含在描述中，在引发检验的相互作用中添加一个系统。当然，这个游戏可以永远继续下去：我们可以将实验者本身也纳入描述，甚至可以包括一个无限长的实验者链，每个实验者对前一个进行检验。得益于式 (7.18)，我们总可以在可逆演化的系统与经历最终测量的系统之间移动分界。由于纯化的存在，每位实验者都可以声称她正在进行测量，而所有其他系统则根据某个根本上的可逆动力学确定性演化。5

    7.11 态-变换同构#

    在满足纯化的理论中，态与变换之间存在一种特殊对应关系，其基本思想源于过程层析。建立对应关系的步骤如下——对于给定系统 $\operatorname{A} {\mathrm{:}}$

    1. 取一组跨越整个态空间的纯态 $\{\alpha_{x} \}$；
    2. 取一个混合态 $\begin{array}{r} {\rho = \sum_{x} p_{x} \alpha_{x}} \end$，其中所有概率 $\{p_{x} \}$ 为正；
    3. 取 $\rho$ 的一个纯化，记作 $\Psi \in \mathsf{PurSt} ( \mathrm{AB} )$，其中 B 为某个纯化系统。

    一旦做出这些选择，我们就可以将态-变换同构定义为线性映射

    $$ \begin{array}{r} {\mathcal{A} \longmapsto \Psi_{\mathcal{A}} : = ( \mathcal{A} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi ,} \end{array} $$

    它将 $\mathsf{Transf} ( \mathrm{A} \to \mathrm{A}^{\prime} )$ 中的变换映射为 $\mathsf{St} ( \mathrm{A^{\prime} B} )$ 中的（未归一化）态。由于态 $\Psi$ 可用于过程层析，映射 $\mathcal{A} \to \Psi_{\mathcal{A}}$ 是单射的。除了单射性之外，式 (7.19) 的对应关系还具有更多独特特征，如下面段落所述。

    7.11.1 确定性变换#

    这里我们根据对应的态来刻画理论中的确定性变换。根据定义，确定性变换 $\mathcal{A} \in \mathsf{Transf} ( \mathrm{A} \to \mathrm{A}^{\prime} )$ 满足条件

    $$ e_{\mathrm{A^{\prime}}} \circ{\mathcal{A}} = e_{\mathrm{A}} . $$

    7.11.1 确定性变换#

    根据这一条件，可以立即看出态 $\Psi_{A}$ 满足关系式

    $$ ( e_{\mathrm{A}} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi_{\mathcal{A}} = \beta , $$

    其中 $\beta$ 是 $\Psi$ 在系统 B 上的边缘态。换言之，$\Psi_{A}$ 是 $\beta$ 的一个扩展。更有趣的是，其逆命题也成立：所有作为 $\beta$ 扩展的两体态，对于某个确定性变换，都具有 $\Psi_{A}$ 的形式。

    命题 7.6 若态 $\Sigma \in \mathsf{St} ( \mathrm{A}^{\prime} \mathrm{B} )$ 是 $\beta$ 的一个扩展，则存在一个确定性变换 $\mathcal{A} \in \mathsf{Transf} ( \mathrm{A} \to \mathrm{A^{\prime}} )$，使得 $\Sigma = ( \mathcal{A} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi$。

    证明 取 $\Sigma$ 的一个纯化，设为 $\Psi^{\prime} \in \mathsf{PurSt} ( \mathrm{C A^{\prime} B} )$。由纯化的唯一性，存在一个确定性变换 $\mathcal{C} \in \mathsf{Transf} ( \mathrm{A} \to \mathrm{CA}^{\prime} )$，使得 $\Psi^{\prime} = ( \mathcal{C} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi$（参见命题 7.1）。于是，我们得到

    $$ \begin{array}{r l} & \underbrace{\overset{\mathrm{C}} {\sum} \bigg | _{\mathbf{B}}^{\mathrm{A}^{\prime}}}_{= \displaystyle ( \Psi \bigg | _{\mathbf{B}}^{\prime} \bigg | _{\mathbf{B}}^{\prime} = ( \Psi^{\prime} \bigg | _{\mathbf{B}}^{\prime} ) )}^{\mathbf{A}^{\prime}} = ( \Psi^{\prime} \bigg | _{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^{\prime} , \mathbf{\Lambda}_{\mathbf{B}}^ \end{array} $$

    其中定义了 $\mathcal{A} : = ( e_{\mathrm{C}} \otimes \mathcal{T}_{\mathrm{A}}^{\prime} ) \mathcal{C}$。根据定义，该变换是确定性变换。∎

    7.11.2 一般检验#

    现在考虑一般检验 $\{{\mathcal{A}}_{x} \}_{x \in \mathsf{X}}$ 的情况。根据定义，如果对所有结果进行粗粒化，我们得到一个确定性变换 $\textstyle A : = \sum_{x \in{\mathsf{X}}} A_{x}$。因此，相应的制备检验 $\{\Psi_{\mathcal{A}_{x}} \}$ 必须满足条件

    $$ \sum_{x \in \mathsf{X}} ( e_{\mathrm{A}} \otimes \mathcal{T}_{B} ) \Psi_{\mathcal{A}_{x}} = \beta , $$

    这直接由方程 (7.20) 得出。同样，其逆结果也成立。由于一般检验的情况包含了确定性变换的情况，这便是态与变换之间同构关系的普遍表述：

    定理 7.7（态-变换同构）若系统 $\mathrm{A^{\prime} B}$ 的一个制备检验 $\{\Sigma_{x} \}_{x \in \mathsf{X}}$ 满足条件

    $$ \sum_{x \in{\mathsf{X}}} ( e_{\mathsf{A}} \otimes{\mathcal{T}}_{B} ) \Sigma_{x} = \beta \qquad \forall x \in{\mathsf{X}} , $$

    则存在一个输入为 A 输出为 A' 的检验 $\{{\mathcal{A}}_{x} \}_{x \in \mathsf{X}}$，使得

    $$ \Sigma_{x} = ( A_{x} \otimes{\mathcal{T}}_{\mathrm{B}} ) \Psi \qquad \forall x \in \mathsf{X} . $$

    证明 证明这一事实需要用到我们在本章中已数次使用的一个技巧：将结果 x 编码为来自某系统 C 的一组完全可区分态中的态 $\gamma_{x}$。定义态

    $$ \Gamma : = \sum_{x \in{\mathsf{X}}} \gamma_{x} \otimes \Sigma_{x} \in{\mathsf{St}} ( {\mathrm{CA}}^{\prime} {\mathrm{B}} ) , $$

    根据构造，$\Gamma$ 是 $\Psi$ 在系统 B 上边缘态 $\beta$ 的一个扩展。利用命题 7.6 的结果，我们得到

    $$ \Gamma = ( {\mathcal A} \otimes{\mathcal T}_{\mathrm{B}} ) \Psi , $$

    其中 $\mathcal{A}$ 是从 A 到 $\mathrm{CA}^{\prime}$ 的一个确定性变换。此外，根据定义，我们有

    $$ ( c_{x} \otimes \mathcal{T}_{\mathrm{A^{\prime} B}} ) \Gamma = \Sigma_{x} \qquad \forall x \in \mathsf{X} , $$

    其中 $\{c_{x} \}_{x \in \mathsf{X}}$ 是区分状态集 $\{\gamma_{x} \}$ 的测量。结合上述两个方程，我们最终得到，对于每个 $x$，

    $$ \begin{array}{r l} & {\Sigma_{x} = ( c_{x} \otimes \mathcal{T}_{\mathrm{A^{\prime} B}} ) \Gamma} \\ & {\quad = ( c_{x} \otimes \mathcal{T}_{\mathrm{A^{\prime} B}} ) ( \mathcal{A} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi} \\ & {\quad = ( \mathcal{A}_{x} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi ,} \end{array} $$

    其中定义了 $\mathcal{A}_{x} : = ( c_{x} \otimes \mathcal{T}_{\mathrm{A}^{\prime}} ) \mathcal{A}$

    总之，状态-变换同构告诉我们，检验与用于建立同构的状态的边缘态 $\beta$ 的扩展的凸分解之间存在一一对应关系。

    7.12 一切未被禁止的都是允许的#

    状态-变换同构有一个惊人的结论：在我们的理论中，任何与状态集相容的假设变换都必须是理论所允许的物理变换。

    假设出于某种原因，你只对一个物理理论有部分了解：你知道归一化状态集，但对变换和测量一无所知。你对此能说些什么呢？

    你可以做的一件事是利用从你对状态的了解中获得的约束来猜测变换和测量。例如，你可以尝试以下列方式猜测从 A 到 $\mathrm{A}^{\prime}$ 的确定性变换。首先，我们知道物理变换对状态空间是线性作用的。因此，你的候选变换 $\mathcal{A}$ 必须是从 $\mathsf{St}_{\mathbb{R}} ( \mathrm{A} )$ 到 $\mathrm{S t_{\mathbb{R}}} ( \mathrm{A}^{\prime} )$ 的一个线性映射。^6 其次，确定性变换应将归一化状态映射为归一化状态。这一条件不仅对系统 A 的状态成立，对任意复合系统 AB 的状态也必须成立，因为你的候选变换在应用于复合系统的一部分时也应表现良好。换句话说，候选映射 $\mathcal{A}$ 必须满足

    $$ \rho \in \mathsf{St}_{1} ( \mathrm{AB} ) \quad \Longrightarrow \quad ( \boldsymbol{A} \otimes \mathbb{Z}_{B} ) \rho \in \mathsf{St}_{1} ( \mathrm{A^{\prime} B} ) . $$

    线性和归一化状态的保持是一个映射成为确定性变换的两个必要条件。但它们是否充分呢？相当惊人地，答案是“是”，这要归功于状态-变换同构：

    命题 7.8 如果一个线性映射 $\mathcal{A} : \mathsf{St}_{\mathbb{R}} ( \mathsf{A} ) \to \mathsf{St}_{\mathbb{R}} ( \mathsf{A}^{\prime} )$ 将归一化状态变换为归一化状态，那么它是一个确定性变换。

    证明 由于 $\mathcal{A}$ 将确定性状态变换为确定性状态，我们必须有 $( e | \mathcal{A} \rho ) = ( e | \rho )$ 对于每个状态 $\rho \in{\mathsf{St}} ( {\mathsf{A}} )$ 成立，或者等价地

    $$ e_{\mathrm{A^{\prime}}} \circ \mathcal{A} = e_{\mathrm{A}} . $$

    现在，定义状态 $\Psi_{\mathcal{A}} : = ( \mathcal{A} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi$。根据构造，我们有

    $$ \begin{array}{r l} & {( e_{\mathrm{A^{\prime}}} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi_{\mathcal{A}} = ( e_{\mathrm{A^{\prime}}} \otimes \mathcal{T}_{\mathrm{B}} ) ( \mathcal{A} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi} \\ & {\quad \quad \quad = [ ( e_{\mathrm{A^{\prime}}} \circ \mathcal{A} ) \otimes \mathcal{T}_{\mathrm{B}} ] \Psi} \\ & {\quad \quad \quad = ( e_{\mathrm{A}} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi} \\ & {\quad \quad \quad \equiv{\beta} .} \end{array} $$

    换句话说，$\Psi_{A}$ 是 $\Psi$ 在系统 B 上的边际的扩展。根据命题 $^{7.6}$，我们有 $\Psi_{\mathcal{A}} = \Psi_{\mathcal{A^{\prime}}}$，其中 $\mathcal{A}^{\prime}$ 是理论允许的一个确定性变换。但对应关系 ${\mathcal{A}} \mapsto \Psi_{\mathcal{A}}$ 是单射的，因此 $\mathcal{A}$ 与 $\mathcal{A}^{\prime}$ 重合。总之，$\mathcal{A}$ 是理论允许的一个确定性变换。

    得益于这一结果，我们终于能够证明作用在给定有限系统上的可逆变换群是紧致的。由于根据局域可分辨性，作用在 A 上的变换构成一个有限维向量空间，只需证明群 $\mathbf{G}_{\mathrm{A}}$ 是闭的即可：

    命题 7.9 对于每个系统 A，可逆变换群 $\mathbf{G}_{\mathrm{A}}$ 是闭的。

    证明 设 $\{\mathcal{U}_{n} \}_{n \in \mathbb{N}} \subset \mathbf{G}_{\mathrm{A}}$ 是一个收敛到某个线性映射 $\mathcal{M} . ^{7}$ 的序列。收敛意味着对于每个系统 B 和每个状态 $\rho \in{\mathsf{St}} ( {\mathsf{AB}} )$，序列 $\{( {\mathcal{U}}_{n} \otimes{\mathcal{T}}_{\mathrm{B}} ) \rho \}$ 收敛到 $( \mathcal{M} \otimes \mathcal{T}_{\mathrm{B}} ) \rho$。由于状态空间是闭的，$( \mathcal{M} \otimes \mathcal{T}_{\mathrm{B}} ) \rho$ 必须是一个状态。此外，当 $\rho$ 是归一化状态时，$( \mathcal{M} \otimes \mathcal{T}_{\mathrm{B}} ) \rho$ 也是归一化的。因此，命题 7.8 蕴含 $\mathcal{M}$ 是一个确定性变换。现在，考虑序列 $\{\mathcal{U}_{n}^{- 1} \}_{n \in \mathbb{N}}$。由于物理变换构成一个有限维向量空间的紧致子集，该序列必有一个收敛子序列，设为 $\{\mathcal{U}_{n_{k}}^{- 1} \}$。记 $\mathcal{N}$ 为该子序列的极限，我们可以对 $\mathcal{M}$ 重复之前的论证，从而证明 $\mathcal{N}$ 是一个确定性变换。此外，显然有 $\mathcal{MN} = \mathcal{NM} = \mathcal{T}_{\mathrm{A}}$。因此，$\mathcal{M}$ 是可逆变换，而 $\mathcal{N}$ 是其逆变换。

    确定性变换方面的成功表明，从状态集出发推断物理变换的方法可能同样适用于任意变换和任意测量。确实如此。

    设 $\{\mathcal{A}_{x} \}_{x \in \mathsf{X}} \subset \mathsf{Transf} ( \mathsf{A} \to \mathsf{A}^{\prime} )$ 是一个候选测试。与之前一样，我们知道物理变换线性地作用于状态空间，因此每个 $\mathcal{A}_{x}$ 必须是从 $\mathsf{St}_{\mathbb{R}} ( \mathbf{A} )$ 到 $\mathsf{St}_{\mathbb{R}} ( \mathrm{A}^{\prime} )$ 的线性映射。此外，每个变换 $\mathcal{A}_{x}$ 应将归一化状态映射到（可能子归一化的）状态，即

    $$ \begin{array}{r} {\rho \in \mathsf{St}_{1} ( \mathrm{AB} ) \quad \Longrightarrow \quad ( \boldsymbol{A}_{x} \otimes \mathcal{T}_{B} ) \rho \in \mathsf{St} ( \mathrm{A}^{\prime} \mathrm{B} ) ,} \end{array} $$

    并且对所有可能结果求和应得到一个归一化状态，即

    $$ \rho \in{\bf S t}_{1} ( {\mathrm{AB}} ) \quad \Longrightarrow \quad \sum_{x \in{\mathsf{X}}} ( A_{x} \otimes{\bar{\mathcal{Z}}}_{B} ) \rho \in{\bf S t}_{1} ( {\mathrm{A}}^{\prime} {\mathrm{B}} ) . $$

    同样地，式(7.21)和式(7.22)的条件是将一组映射称为检验的必要要求。由于状态-变换同构，这些条件也是充分的：每一组满足式(7.21)和式(7.22)的线性映射都是该理论允许的检验。

    命题7.10 设 $\{{\mathcal{A}}_{x} \}_{x \in \mathsf{X}}$ 是从 $\mathsf{St}_{\mathbb{R}} ( \mathbf{A} )$ 到 $\mathsf{St}_{\mathbb{R}} ( \mathsf{A}^{\prime} )$ 的一组线性映射。如果 $\{{\mathcal{A}}_{x} \}_{x \in \mathsf{X}}$ 满足式(7.21)和式(7.22)，那么它就是该理论允许的检验。

    证明 定义线性映射 $\begin{array}{r} {A : = \sum_{x \in \mathsf{X}} \mathcal{A}_{x}} \end$ 。由式(7.22)可知，它将归一化态变换为归一化态，因此 $e_{\mathrm{A^{\prime}}} \circ{\mathcal{A}} = e_{\mathrm{A}}$ 。现在，定义状态 $\Psi_{\mathcal{A}_{x}} : = ( \mathcal{A}_{x} \otimes \mathcal{T}_{\mathrm{B}} ) \Psi$ 。显然，它们满足条件

    $$ \begin{array}{r l} {\displaystyle \sum_{x \in \mathsf X} ( e_{\mathrm A^{\prime}} \otimes \mathcal T_{\mathrm B} ) \Psi_{\mathcal A_{x}} = ( e_{\mathrm A^{\prime}} \otimes \mathcal T_{\mathrm B} ) \Psi_{\mathcal A}} & {} \\ & {\quad \quad = ( e_{\mathrm A^{\prime}} \otimes \mathcal T_{\mathrm B} ) ( \mathcal A \otimes \mathcal T_{\mathrm B} ) \Psi} \\ & {\quad = [ ( e_{\mathrm A^{\prime}} \circ \mathcal A ) \otimes \mathcal T_{\mathrm B} ] \Psi} \\ & {\quad = ( e_{\mathrm A} \otimes \mathcal T_{\mathrm B} ) \Psi} \\ & {\quad \quad \equiv \beta .} \end{array} $$

    因此，由定理7.7可知，存在一个该理论允许的检验({\mathcal{A}{x}^{\prime}})，使得对于每个(x)，有(\Psi{\mathbf{\mathcal{A}}{x}} = ( \mathcal{A}{x}^{\prime} \otimes \mathcal{T}{\mathrm{B}} ) \Psi)。由于映射({\mathcal{A}} \mapsto \Psi{\mathcal{A}})是单射，我们得出结论：对于每个(x)，(\mathcal{A}{x} = \mathcal{A}{x}^{\prime})。因此，({\mathcal{A}_{x} })是该理论允许的一个检验。

    总之，未被禁止的一切都必须被允许。这意味着归一化态集唯一地确定了该理论所允许的所有变换、测量和一般检验：整个理论都编码在归一化态集中。我们对量子理论的推导将利用这一事实：首先，我们将证明该理论允许的归一化态是希尔伯特空间上的密度矩阵。然后，我们将利用该理论由态集唯一确定这一事实。换句话说，我们确定地知道，满足纯化且以密度矩阵为可能态的理论有且只有一个：那就是量子理论。

    7.13 纯化概要#

    在前几节中，我们带你领略了纯化原则最重要的一些结论。由于这一旅程思想密集，在此我们将对所探索的概念图景做一个鸟瞰。我们从一个纯化原则的陈述出发：每一个混合态都可以通过丢弃一个复合系统纯态的一部分来制备，且这种制备方式本质上是唯一的。混合态的纯化首先引导我们进入纠缠和导引性质，后者使得我们能够通过对纯化系统进行测量来生成混合态任意系综分解。导引是使我们能够对过程进行层析成像的特征；得益于此特征，当我们纯化一个完全混合态时，我们得到一个可用于忠实编码物理变换的双粒子态。这个双粒子态的关键特征是它是纯的：正因如此，我们证明了"无干扰则无信息"的性质，这对密码学有着根本性的影响。

    纯化的后果不仅涉及态，还涉及变换。首先，纯化意味着每个纯态都可以通过可逆变换转化为任意其他纯态，这一性质称为传递性。传递性意味着，对于每个有限维系统，存在唯一的最大混合态，即唯一在任意可逆变换作用下保持不变的态。这一事实与导引性质的一个相当引人注目的应用是量子隐形传态。从第一性原理出发进行推理，无需希尔伯特空间形式体系，就能推导出量子信息的经典结果，而量子隐形传态的重构或许是这方面最好的例子。

    除了量子隐形传态，纯化还意味着（事实上是等价于）物理变换的纯态可逆模拟。每一个确定性过程都可以模拟为一个可逆变换，该变换共同作用于系统及其初始处于纯态的环境。在这种图景中，不可逆性的唯一来源在于环境被丢弃。这一事实是信息守恒的表述，根据这一表述，在基本层面上信息不能被摧毁，只能被丢弃。纯态可逆模拟可以推广到一般检验的情况，包括随机结果。在这种情况下，结果是每个检验都可以建模为输入系统与检验设备之间的可逆相互作用，随后对检验设备进行测量以读出结果。原则上，可逆相互作用的系统与被测量的系统之间的分界线可以无限移动，正如冯·诺依曼对量子测量过程的处理那样。8

    最后，纯化的一个显著结果是，一旦我们指定了归一化态集，理论就完全确定了。换句话说，无需再指定测量、变换，或更一般地，检验：与归一化态集相容的任何假设变换必然包含在理论中。这一结果保证了，如果我们想要推导出量子理论，只需证明可能的归一化态可以表示为复希尔伯特空间上的密度矩阵即可。满足纯化的理论由归一化态集唯一确定，这是过程层析成像所建立的态与变换之间的同构性的一个高层次推论。

    7.14 总结#

    在本章中，我们介绍了纯化原理，展示了它如何导致纠缠、导引、变换层析成像、无干扰则无信息以及隐形传态。在满足纯化的世界中，给定系统的纯态可以通过可逆变换相互转换。此外，不可逆性可以归因于环境中信息的丢失，并且测量系统与测量装置之间的界限可以任意移动，同时仍将系统与装置之间的相互作用建模为可逆变换。最后，我们证明了纯化建立了变换与二分态之间的同构性，并利用这一同构性证明了每个保持态空间结构的线性映射都必须是理论所允许的物理变换。

    注释#

    纯化与冯·诺依曼要求 像无知这样的主观元素必须允许客观的物理描述，这一要求立即让人联想到冯·诺依曼的观察，他写道（von Neumann, 1996）：“科学视角的一个基本要求——所谓的心身平行论原理——是必须可能如此描述主观感知的额外物理过程，仿佛它实际上存在于物理世界中——即，为其各部分在客观环境中，在普通空间中，分配等价的物理过程。”这里冯·诺依曼指的是通过测量收集信息的任务，但将其观察扩展到其他主观经验（如无知的经验）是相当自然的。如果这样做，那么纯化无非就是冯·诺依曼的要求：纯化系统B是“客观环境”，用于解释我们的无知之源，作为“等价的物理过程”的结果——即在纯态下制备AB。

    上帝掷骰子吗？ 众所周知，爱因斯坦抱怨量子理论的随机性，体现在流行的口头禅“上帝不掷骰子”中。然而，纯态和可逆模拟提供了一个有趣的反驳：上帝不需要掷骰子。相反，祂可以将世界制备在纯态中，并让它可逆地演化。如果是这样，我们在实验中观察到的随机性和不可逆性只是我们观察世界的有限角度以及我们对环境交互控制的有限程度的结果。

    纯化与因果性 本章中提出的纯化原理要求因果性的成立。事实上，式(7.1)明确包含了唯一的决定论效应。然而，也可以在没有因果性的情况下表述纯化。这种更一般的表述在 Chiribella et al. (2014) 中讨论。

    附录 7.1 Carathéodory 定理#

    定理 7.11 (Carathéodory) 如果点 $\boldsymbol{x} \in \mathbb{R}^{d}$ 属于集合 $P \subseteq \mathbb{R}^{d}$ 的凸包 Conv(P)，则存在一个由至多 $d + 1$ 个点组成的子集 $P^{\prime} \subseteq P$，使得 $x \in{\mathsf{Conv}} ( P^{\prime} )$。

    证明 设 $x \in{\mathsf{Conv}} ( P ) \subseteq \mathbb{R}^{d}$。那么，x 是 P 中有限个点的凸组合：

    $$ \mathbf{x} = \sum_{j = 1}^{k} \lambda_{j} \mathbf{x}_{j} , \qquad x_{j} \in P , \lambda_{j} \geq 0 , \quad \forall j = 1 , \ldots , k $$

    对于 $k \leq d$，该陈述平凡成立。假设 $k > d + 1$。那么，点 $x_{2} \textrm{--}$ $x_{1} , \ldots , x_{k} - x_{1}$ 线性相关，因此存在不全为零的实标量 $\mu_{2} , \ldots , \mu_{k}$，使得

    $$ \sum_{j = 2}^{k} \mu_{j} ( \mathbf{x}_{j} - \mathbf{x}_{1} ) = \mathbf{0} {\mathrm{.}} $$

    通过定义 $\textstyle \mu_{1} : = - \sum_{j = 2}^{k} \mu_{j}$，可得

    $$ \sum_{j = 1}^{k} \mu_{j} \mathbf{x}_{j} = \pmb{0} , \quad \sum_{j = 1}^{k} \mu_{j} = 0 , $$

    并且并非所有 $\mu_{j}$ 都为零。因此，至少有一个 $\mu_{j} > 0$。于是，对于任意实数 α，

    $$ \mathbf{x} = \sum_{j = 1}^{k} \lambda_{j} \mathbf{x}_{j} - \alpha \sum_{j = 1}^{k} \mu_{j} \mathbf{x}_{j} = \sum_{j = 1}^{k} ( \lambda_{j} - \alpha \mu_{j} ) \mathbf{x}_{j} $$

    成立。特别地，当 α 由下式给出时，该等式仍然成立：

    $$ \alpha : = \operatorname*{min}_{1 \le j \le k} \left\{\frac{\lambda_{j}} {\mu_{j}} : \mu_{j} > 0 \right\} = \frac{\lambda_{i}} {\mu_{i}} . $$

    注意 $\alpha > 0$，并且对于每个 $j = 1 , \dotsc , k$，有

    $$ \lambda_{j} - \alpha \mu_{j} \geq 0 . $$

    特别地，根据 α 的定义，$\lambda_{i} - \alpha \mu_{i} = 0$。因此，

    $$ \mathbf{x} = \sum_{j = 1}^{k} ( \lambda_{j} - \alpha \mu_{j} ) \mathbf{x}_{j} , $$

    其中每个 $\lambda_{j} - \alpha \mu_{j}$ 非负，它们的和为 1，并且此外，$\lambda_{i} - \alpha \mu_{i} = 0$。换句话说，x 被表示为 P 中至多 $k - 1$ 个点的凸组合。这个过程可以重复，直到 x 被表示为 P 中至多 $d + 1$ 个点的凸组合。
